ComObjError() [AHK_L 53+]

Aktiviert oder deaktiviert die Meldungen von COM-Fehlern.

Aktiviert := ComObjError(Aktivieren)

Parameter

Aktivieren Ein boolescher Wert (True oder False). Optional.
Aktiviert [v1.0.91+]: Empfängt die Einstellung, die vor dem Funktionsaufruf aktiv war.

Allgemeine Bemerkungen

Nachdem auf einem COM-Objekt zugegriffen wurde, enthält A_LastError den HRESULT-Code, der von der IDispatch::Invoke-Funktion des COM-Objekts zurückgegeben wird. Scripts können eine eigene Fehlerbehandlung implementieren, wenn ComObjError(false) aufgerufen wird, um Fehlermeldungen zu deaktivieren und auf A_LastError zuzugreifen.

Verwandte Befehle

ComObjCreate, ComObjGet, ComObjActive, ComObjConnect
