FileGetShortcut

Ermittelt Informationen von einer Verknüpfungsdatei (.lnk), sowie von ihrer Zieldatei. 

FileGetShortcut, Verknüpfungsdatei [, Ziel, Verzeichnis, Parameter, Beschreibung, Icon, Iconnummer, Ausführungsstatus]

Parameter

Verknüpfungsdatei Name der zu überprüfenden Verknüpfung, die sich im A_WorkingDir-Verzeichnis befindet, solange kein absoluter Pfad angegeben ist. Die .lnk-Erweiterung muss mit angegeben werden.
Ziel Der Name der Ausgabevariable, in der das Ziel der Verknüpfung gespeichert wird (ohne irgendwelche vorhandenen Argumente). Zum Beispiel: C:\WINDOWS\system32\notepad.exe
Verzeichnis Der Name der Ausgabevariable, in der das Arbeitsverzeichnis der Verknüpfung gespeichert wird. Zum Beispiel: C:\Eigene Dokumente. Wenn Umgebungsvariablen wie %WinDir% in einem String vorhanden sind, dann können sie mittels StringReplace aufgelöst werden. Zum Beispiel: StringReplace, Verzeichnis, Verzeichnis, `%WinDir`%, %A_WinDir%
Parameter Der Name der Ausgabevariable, in der die Parameter der Verknüpfung gespeichert werden (leer, wenn nichts vorhanden).
Beschreibung Der Name der Ausgabevariable, in der die Kommentare der Verknüpfung gespeichert werden (leer, wenn nichts vorhanden).
Icon Der Name der Ausgabevariable, in der der Dateiname des Verknüpfungsicons gespeichert wird (leer, wenn nichts vorhanden).
Iconnummer Der Name der Ausgabevariable, in der die Iconnummer des Verknüpfungsicons gespeichert wird (leer, wenn nichts vorhanden). Dieser Wert ist meistens 1, d. h. das erste Icon.
Ausführungsstatus

Der Name der Ausgabevariable, in der der Anfangsstatus der Verknüpfung gespeichert wird:

1: Normal
3: Maximiert
7: Minimiert

ErrorLevel

Bei Problemen (z. B. Verknüpfungsdatei existiert nicht) werden alle Ausgabevariablen leer gemacht und ErrorLevel auf 1 gesetzt. Ansonsten enthält ErrorLevel eine 0.

Bemerkungen

Jede Ausgabevariable kann weggelassen werden, wenn sie nicht gebraucht wird.

Verwandte Befehle

FileCreateShortcut, SplitPath

Beispiel

FileSelectFile, Datei, 32,, Verknüpfung auswählen, um sie zu überprüfen., Verknüpfungen (*.lnk)
If Datei =
    Return
FileGetShortcut, %Datei%, Ziel, Verzeichnis, Parameter, Beschreibung, Icon, Iconnummer, Ausführungsstatus
MsgBox %Ziel%`n%Verzeichnis%`n%Parameter%`n%Beschreibung%`n%Icon%`n%Iconnummer%`n%Ausführungsstatus%
