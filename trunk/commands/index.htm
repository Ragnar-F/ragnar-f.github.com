Alphabetischer Befehls- und Funktionsindex

Klickt für mehr Details auf einem Befehls- oder Funktionsnamen. Häufig verwendete Einträge sind groß geschrieben.

Für eine Liste von AutoHotkey_L-spezifischen Befehlen und Funktionen, siehe AutoHotkey_L-Funktionen.

Eintrag Beschreibung
{ ... } (Block) Zwei geschweifte Klammern kennzeichnen einen Block. Blöcke werden für gewöhnlich bei Funktionen, Else, Loop, While-Loop und If-Befehle benutzt.
{ ... } / Object() Erstellt ein scriptfähiges assoziatives Array.
[ ... ] / Array() Erstellt ein skriptfähiges assoziatives Array mit Integer-Schlüssel.
AutoTrim Bei Anweisungen wie "Var1 = %Var2%" werden Leerzeichen und Tabulatoren am Beginn und am Ende von Var2 entfernt.
BlockInput Deaktiviert oder aktiviert die Fähigkeit des Benutzers, mit dem Computer mittels Tastatur und Maus zu interagieren.
Break Beendet bzw. verlässt eine Schleife. Nur in einer Schleife gültig.
Click Klickt mit einer Maustaste auf bestimmte Koordinaten. Kann auch eine Maustaste gedrückt halten, das Mausrad drehen oder die Maus bewegen.
ClipWait Wartet, bis die Zwischenablage Text oder Dateien enthält.

If (Ausdruck) Ein oder mehrere Befehle werden ausgeführt, wenn ein Ausdruck als WAHR gewertet wird.

If var [not] in/contains Übereinstimmungsliste Überprüft, ob der Inhalt der Variable mit einem Element aus der Liste übereinstimmt.
If Variable is [not] Typ
 Überprüft, ob der Inhalt der Variable numerisch, groß geschrieben, etc.
