If (Ausdruck)

Ein oder mehrere Befehle werden ausgeführt, wenn ein Ausdruck als WAHR gewertet wird. 

If (Ausdruck)

Bemerkungen

Eine If-Anweisung mit einem Ausdruck unterscheidet sich von einem traditionellen If wie If Fundfarbe <> Blau, wenn das Zeichen nach dem Wort "If" eine offene Klammer ist. Obwohl dies in der Regel dadurch erreicht wird, den kompletten Ausdruck in Klammern zu setzen, ist dies beispielsweise auch mit If (x > 0) and (y > 0) möglich. Darüber hinaus kann die offene Klammer komplett weggelassen werden, wenn das erste Element nach dem Wort "If" ein Funktionsaufruf oder ein Operator wie "not" oder "!" ist.

Wenn der Ausdruck der If-Anweisung als wahr gewertet wird (d. h. das Ergebnis darf nicht leer oder die Zahl 0 sein), dann wird die Zeile oder Block darunter ausgeführt. Ansonsten wird die Zeile oder Block bei einem nachfolgenden ELSE ausgeführt.

Wenn IF oder ELSE mehrere Zeilen enthält, dann müssen diese Zeilen mit Klammern umschlossen werden. Wenn allerdings nur eine Zeile vorhanden ist, dann sind die Klammern optional. Siehe die Beispiele im untersten Abschnitt.

Der One True Brace (OTB)-Style kann mit If-Anweisungen optional verwendet werden, die Ausdrücke besitzen (aber keine traditionellen If-Anweisungen). Zum Beispiel:

If (x < y) {
    ...
}
If WinExist("Unbenannt - Editor") {
    WinActivate
}
If IsDone {
    ...
} Else {
    ...
}

Nicht wie bei einer Else-Anweisung, die jegliche Arten von Anweisungen rechts von ihr unterstützt, unterstützen If-Anweisungen nur einen "{" danach.

Ein weiterer Befehl If Variable [not] between Untergrenze and Obergrenze überprüft, ob eine Variable zwischen zwei Werten liegt. If Variable [not] in Wert1,Wert2 überprüft, ob der Inhalt einer Variable in einer Liste von Werten vorhanden ist.

Verwandte Befehle

Ausdrücke, Zuweisungsausdruck (:=), If Variable in/contains Übereinstimmungsliste, If Variable between, IfInString, Blocks, Else, While-Loop

Beispiel

If (A_Index > 100 or Fertig)
    Return

If (A_TickCount - Startzeit > 2*Maximalzeit + 100)
{
    MsgBox Zu viel Zeit ist vergangen.
    ExitApp
}

If (Farbe = "Blau" or Farbe = "Weiß")
{
    MsgBox Die Farbe ist einer der erlaubten Werte.
    ExitApp
}
Else If (Farbe= "Silber")
{
    MsgBox Silber ist keine erlaubte Farbe.
    Return
}
Else
{
    MsgBox Diese Farbe wurde nicht erkannt.
    ExitApp
}
