Sprachsyntax

Wann werden im Zusammenhang mit Befehlen und Parametern Anführungszeichen verwendet?

Anführungszeichen (") haben nur innerhalb von Ausdrücken eine besondere Bedeutung. Ansonsten werden sie wie ganz normale direkt vorhandene Zeichen behandelt. Wenn allerdings ein Script ein Programm oder Dokument ausführt, dann erwartet das Betriebssystem für gewöhnlich Anführungszeichen um Kommandozeilenparameter mit Leerzeichen, wie in diesem Beispiel: Run, Notepad.exe "C:\Eigene Dateien\Adressenliste.txt"

Wann genau werden Variablen mit Prozentzeichen umschlossen (%)?

Variablennamen werden immer mit Prozentzeichen umschlossen, bis auf die unten fett dargestellten Ausnahmen:

1) In Parametern, die Eingabe- oder Ausgabevariablen sind: StringLen, Ausgabevariable, Eingabevariable
2) Auf der linken Seite einer Zuweisung: Var = 123abc
3) Auf der linken Seite einer traditionellen If-Anweisung (kein Ausdruck): If Var1 < %Var2%
4) Überall in Ausdrücken. Zum Beispiel:

If (Var1 <> Var2)
    Var1 := Var2 + 100

Wann sollten Prozentzeichen und Kommas geschützt werden?

Direkt vorhandene Prozentzeichen müssen mit einem Akzent geschützt werden. Zum Beispiel: MsgBox Der aktuelle Prozentwert liegt bei 25`%. Direkt vorhandene Kommas müssen auch geschützt werden (`,), wenn sie in einer MsgBox oder im letzten Parameter eines Befehls verwenden werden (in diesem Fall ist der Akzent zulässig, aber nicht notwendig).

Wenn Kommas oder Prozentzeichen innerhalb eines Ausdrucks von Anführungszeichen umschlossen werden, dann ist der Akzent zulässig, aber nicht notwendig. Zum Beispiel: Var := "15%"

Allgemeine Aufgaben

Warum werden einige Zeilen in meinem Script nie ausgeführt?

Jede Zeile, die unmittelbar beim Script-Start ausgeführt werden soll, sollte am Anfang des Scripts und vor dem ersten Hotkey, Hotstring oder Return angegeben werden. Für weitere Informationen, siehe automatischer Ausführungsbereich.

Außerdem muss ein Hotkey, der mehr als eine Zeile ausführt, seine erste Zeile unter dem Hotkey auflisten, aber nicht auf derselben Zeile. Zum Beispiel:

#space::  ; Win+Leerzeichen
Run Notepad
WinWaitActive Unbenannt - Editor
WinMaximize
Return

Warum ist der Run-Befehl nicht in der Lage, mein Spiel oder Programm auszuführen?

Einige Programme müssen in ihren eigenen Verzeichnissen gestartet werden (im Zweifelsfall der sicherste Weg). Zum Beispiel:

Run, %ProgramFiles%\Beliebige Anwendung\Anwendung.exe, %ProgramFiles%\Beliebige Anwendung

Wie kann die Ausgabe einer Befehlszeilenoperation abgefragt werden?

Tests zeigen, dass eine temporäre Datei aufgrund von Datei-Caching sehr schnell bei kleinen Ausgaben sein kann. Tatsächlich wird sie gar nicht erst auf das Laufwerk geschrieben, wenn die Datei unmittelbar danach gelöscht wird. Zum Beispiel:

RunWait %comspec% /c dir > C:\Eigene Temp-Datei.txt
FileRead, Ausgabevariable, C:\Eigene Temp-Datei.txt
FileDelete, C:\Eigene Temp-Datei.txt

Um das Verwenden von temporären Dateien zu vermeiden (besonders wenn die Ausgabe groß ist), zieht CmdRet oder StdoutToVar in Betracht.

Wie kann ein Script andere Scripts schließen, pausieren oder sperren?

Zunächst ein Beispiel, dass ein anderes Script schließt:

DetectHiddenWindows On  ; Damit kann das versteckte Hauptfenster eines Scripts erkannt werden.
SetTitleMatchMode 2  ; Verhindert, dass unten der vollständige Pfad der Datei angegeben werden muss.
WinClose Script-Dateiname.ahk - AutoHotkey  ; Das hier auf den gewünschten Dateinamen aktualisieren (von der Groß- und Kleinschreibung abhängig).

Um ein anderes Script zu sperren oder zu pausieren, ersetzt die letzte Zeile von oben jeweils mit dieser Zeile:

PostMessage, 0x111, 65305,,, Script-Dateiname.ahk - AutoHotkey  ; Suspend.
PostMessage, 0x111, 65306,,, Script-Dateiname.ahk - AutoHotkey  ; Pause.

Wie kann eine sich wiederholende Aktion gestoppt werden, ohne das Script beenden zu müssen?

Um das ganze Script beim Drücken einer Taste zu pausieren oder fortzusetzen, kann dem Pause-Befehl ein Hotkey wie im folgenden Beispiel zugewiesen werden:

^!p::Pause  ; Drückt Strg+Alt+P, um das Script zu pausieren. Drückt ihn nochmal, um es fortzusetzen.

Um eine Aktion zu stoppen, die sich innerhalb einer Schleife wiederholt, zieht den folgenden funktionierenden Hotkey in Betracht, um die wiederholende Aktion sowohl zu starten als auch zu stoppen. Mit anderen Worten wird beim einmaligen Drücken des Hotkeys die Schleife gestartet. Wenn der gleiche Hotkey nochmals gedrückt wird, dann wird sie gestoppt.

#MaxThreadsPerHotkey 3
#z::  ; Hotkey: Win+Z (passt diesen Hotkey je nach Bedarf an).
#MaxThreadsPerHotkey 1
If WinZLaufenLassen  ; Bedeutet, dass ein darunterliegender Thread bereits die untere Schleife ausführt.
{
    WinZLaufenLassen := false  ; Weist den Thread darauf hin, die Schleife zu stoppen.
    Return  ; Beendet diesen Thread, damit der darunterliegende Thread fortgesetzt und die Veränderung von der oberen Zeile bemerken wird.
}
; Ansonsten:
WinZLaufenLassen := true
Loop
{
    ; Die nächsten vier Zeilen sind die gewünschten Aktionen, die wiederholt werden sollen (passt sie je nach Bedarf an):
    ToolTip, Drückt nochmals Win+Z, um das hier zu stoppen.
    Sleep 1000
    ToolTip
    Sleep 1000
    ; Lasst den Rest aber unverändert.
    If not WinZLaufenLassen ; Der Benutzer hat die Schleife durch erneutes Drücken von Win+Z gestoppt.
        break  ; Schleife unterbrechen.
}
WinZLaufenLassen := false  ; Zurücksetzen, damit der Hotkey wieder verwendet werden kann.
Return

Wie kann die Performance für Spiele oder zum anderen Zeitpunkt verbessert werden, wenn die CPU stark belastet wird?

Wenn Hotkeys, Clicks oder Sends im Script deutlich langsamer als normal sind, während die CPU stark belastet wird, dann könnte das Erhöhen der Script-Prozesspriorität helfen. Fügt dafür die folgende Zeile am Anfang des Scripts hinzu:
Process, Priority, , High

Wie kann man feststellen, dass eine Webseite fertig geladen ist?

Beim Internet Explorer ist die zuverlässigste Methode das Verwenden von DllCall und COM, wie auf www.autohotkey.com/forum/topic19256.html demonstriert. Zusätzlich kann der Inhalt der Adress- und Statusleiste abgerufen werden, wie auf www.autohotkey.com/forum/topic19255.html demonstriert.

Ältere, weniger zuverlässige Methode: Das folgende Beispiel funktioniert fast mit allen Webseiten im MS Internet Explorer. Eine ähnliche Methode funktioniert möglicherweise auch in anderen Browsern:

Run, www.yahoo.com
MouseMove, 0, 0  ; Hindert die Statusleiste daran, den Link unter dem Mauszeiger anstelle von "Fertig" anzuzeigen.
WinWait, Yahoo! -
WinActivate
StatusBarWait, Done, 30
If ErrorLevel
   MsgBox Die Wartezeit ist abgelaufen oder das Fenster wurde geschlossen.
Else
   MsgBox Die Webseite wurde geladen.

Wie können Datums- oder Uhrzeitwerte verglichen oder geändert werden?

Der EnvAdd-Befehl kann eine bestimmte Menge an Tagen, Stunden, Minuten oder Sekunden im YYYYMMDDHH24MISS-Format addieren oder subtrahieren. Das folgende Beispiel subtrahiert 7 Tage von der angegebenen Zeit:
EnvAdd, Zeitstempel, -7, days

Um den Zeitraum zwischen zwei Daten oder Zeiten zu ermitteln, siehe EnvSub für ein Beispiel. Außerdem enthält die interne Variable A_Now die aktuelle Ortszeit. Zudem gibt es noch mehr interne Datum/Zeit-Variablen, sowie der FormatTime-Befehl, um einen benutzerdefinierten Datum/Zeit-String zu erstellen.

Warum funktionieren Hotstrings, Sends und Clicks in einigen Spielen nicht?

Einige Spiele verwenden ausschließlich DirectInput. Als Nebeneffekt werden alle simulierten Tastatureingaben und Mausklicks möglicherweise ignoriert. Als Übergangslösung kann eine folgende Angabe (oder in Kombination) ausprobiert werden:

  • Verwendet SendPlay mittels: 1) dem SendPlay-Befehl; 2) SendMode Play; und/oder 3) der Hotstring-Option SP.
  • Erhöht SetKeyDelay. Zum Beispiel:
    SetKeyDelay, 0, 50
    SetKeyDelay, 0, 50, Play
  • Versucht den ControlSend-Befehl, der möglicherweise funktioniert, während jeder andere Send-Modus versagt.

Hotkeys, Hotstrings und Neubelegung

Wie können meine Hotkeys und Hotstrings automatisch bei jedem PC-Start ausgeführt werden?

Es gibt einen Ordner namens "Autostart" im Startmenü.  Wenn eine Verknüpfung des Scripts in diesem Ordner eingefügt wird, dann wird das Script jedesmal automatisch ausgeführt, sobald Windows hochgefahren wird. Um eine Verknüpfung zu erstellen:

  1. Findet die Script-Datei, wählt sie aus und drückt Steuerung+C.
  2. Klickt mit der rechten Maustaste auf die Start-Schaltfläche bzw. auf "Alle Programme" (normalerweise in der linken unteren Ecke des Bildschirms) und wählt "Öffnen - Alle Benutzer" aus.
  3. Navigiert zum Autostart-Ordner innerhalb von "Programme".
  4. Wählt in der Menüleiste Bearbeiten -> Verknüpfung einfügen aus.  Die Verknüpfung des Scripts sollte sich nun im Autostart-Ordner befinden.

Ich habe Schwierigkeiten, dass meine Maustasten wie Hotkeys funktionieren. Irgendwelche Hinweise?

Beachtet, dass Maus-Hotkeys zurzeit nicht in Win95/98/Me möglich sind. In anderen Betriebssystemen sollte die linke und die rechte Maustaste ganz normal zuweisbar sein (zum Beispiel ist #LButton:: der Hotkey Win+LeftButton). Ebenso sollte die mittlere Maustaste und das Drehen des Mausrades normal zuweisbar sein, außer bei Mäusen, deren Treiber direkt solche Tasten kontrolliert.

Die vierte (XButton1) und fünfte Maustaste (XButton2) sind möglicherweise auch zuweisbar, wenn der Maustreiber seine Klicks für das System sichtbar macht. Wenn sie nicht sichtbar sind (oder wenn die Maus mehr als 5 Tasten hat, die verwendet werden sollen), dann versucht, die dazugehörige Software der Maus zu konfigurieren (meistens in der Systemsteuerung oder im Startmenü), um eine Tastatureingabe zu senden, sobald eine dieser Tasten gedrückt wird. Solch eine Tastatureingabe kann dann in einem Script als Hotkey definiert werden. Wenn zum Beispiel die vierte Maustaste konfiguriert wird, um Steuerung+F1 zu senden, dann kann diese Taste mithilfe von ^F1:: im Script indirekt zu einem Hotkey gemacht werden.

Wenn bei einer Maus mit 5 Tasten die vierte und fünfte Maustaste nicht sichtbar ist, dann versucht, den aktuellen Maustreiber auf den Standardtreiber des Betriebssystems zu setzen. Das setzt voraus, dass solch ein Treiber für die spezielle Maus vorhanden ist und dass der Benutzer ohne den mitgelieferten Funktionen der Software leben kann.

Wie können Tabulatoren und Leerzeichen als Hotkeys definiert werden?

Verwendet die Namen der Tasten (Tab und Space) anstelle der Zeichen. Zum Beispiel ist #Space die Tastenkombination Win+Space und ^!Tab die Tastenkombination Steuerung+Alt+Tab.

Wie kann ein Hotkey oder Hotstring exklusiv für ein bestimmtes Programm bereitgestellt werden? Mit anderen Worten möchte ich eine bestimmte Taste dazu bringen, dass sie sich anders als normal verhält, falls ein bestimmtes Fenster aktiv ist.

Die bevorzugte Methode ist #IfWinActive. Zum Beispiel:

#IfWinActive, ahk_class Notepad
^a::MsgBox Steuerung+A wurde gedrückt, während der Editor aktiv ist.

Windows 95/98/Me: Obwohl die obige Methode funktioniert, passiert durch das Drücken von Steuerung+A in einem Fenster außer dem Editor überhaupt nichts (selbst deren ursprüngliche Funktion nicht). Verwendet als Übergangslösung:.

$^a::Send ^a  ; Dieser Hotkey muss zuerst in Windows 9x aufgelistet werden. Das $-Präfix erlaubt dem Hotkey, "sich selbst zu senden".
#IfWinActive, ahk_class Notepad
^a::MsgBox Steuerung+A wurde gedrückt, während der Editor aktiv ist.

Wie kann bei einer Präfix-Taste erreicht werden, dass sie ihre ursprüngliche Funktion durchführt, anstatt nichts zu tun?

Zieht das folgende Beispiel in Betracht, dass Numpad0 zum Bestandteil einer Tastenkombination macht:

Numpad0 & Numpad1::MsgBox, Numpad1 wurde gedrückt, während Numpad0 gedrückt gehalten wird.

Damit Numpad0 nun einen echte Numpad0-Tastatureingabe sendet, sobald sie nicht dazu verwendet wurde, einen Hotkey wie oben auszuführen, fügt den folgenden Hotkey hinzu:

 $Numpad0::Send, {Numpad0}

Der $-Präfix ist notwendig, um eine Warnmeldung über eine Endlosschleife zu unterdrücken (da der Hotkey "sich selbst sendet"). Darüber hinaus wird die obere Aktion in dem Moment ausgelöst, wenn die Taste losgelassen wird.

Meine Tastatur hat eine besondere 000-Taste. Kann ich damit einen Hotkey erzeugen?

Es ist möglich, aber nur wenn das Betriebssystem Windows NT, 2000, XP oder höher ist. Dieses Beispiel-Script wandelt die "000"-Taste in eine Gleichheitstaste um. Diese Aktion kann geändert werden, wenn die Zeile "Send, =" je nach Bedarf mit einer anderen Zeile ersetzt wird.
