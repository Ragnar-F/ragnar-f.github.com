ClipWait

Wartet, bis die Zwischenablage Text oder Dateien enthält.

ClipWait [, Wartezeit, 1]

Parameter

Wartezeit Wenn nicht vorhanden, wartet der Befehl unendlich lang. Andernfalls wartet der Befehl die angegebenen Sekunden (kann einen Dezimalpunkt enthalten oder ein Ausdruck sein). Der Wert 0 entspricht 0.5.
1
 Wenn dieser Parameter nicht vorhanden ist, dann wartet der Befehl gezielt auf Texte oder Dateien, die in der Zwischenablage erscheinen. Falls dieser Parameter eine 1 enthält (kann auch ein Ausdruck sein), wartet der Befehl auf Daten beliebiger Art, die in der Zwischenablage erscheinen.

ErrorLevel

[v1.1.04+] Dieser Befehl ist in der Lage, bei Misserfolg eine Ausnahme auszulösen. Für mehr Informationen, siehe Laufzeitfehler.

Wenn die Wartezeit abläuft, wird ErrorLevel auf 1 gesetzt. Andernfalls wird ErrorLevel auf 0 gesetzt, wenn die Zwischenablage Daten enthält.

Bemerkungen

Dieser Befehl ist besser als eine Schleife, die überprüft, ob die Zwischenablage leer ist. Denn die Zwischenablage wird nie mit diesem Befehl geöffnet, deshalb wird jegliches Risiko einer Störung mit anderen Anwendungen vermieden, die vielleicht auch die Zwischenablage verwenden.

Dieser Befehl berücksichtigt alle Daten, die sich in Text umwandeln lassen können (z. B. HTML). Es werden auch Dateien berücksichtigt, die mit Steuerung+C in einem Explorer-Fenster kopiert werden. Solche Dateien werden automatisch in ihre Dateinamen umgewandelt (mit vollständigem Pfad), sobald die Zwischenablage-Variable (%clipboard%) im Script verwendet wird. Siehe weitere Details unter Clipboard.

Wenn der letzte Parameter eine 1 enthält, akzeptiert der Befehl alle möglichen Daten, die in der Zwischenablage erscheinen. Diese Option kann in Verbindung mit ClipboardAll benutzt werden, um Nicht-Text-Elemente wie Bilder zu speichern.

Während der Befehl in einem Wartezustand ist, können neue Threads mittels Hotkey, benutzerdefinierte Menüeinträge oder Timer gestartet werden.

Verwandte Befehle

Clipboard, WinWait, KeyWait

Beispiel

clipboard = ; Zwischenablage leeren
Send, ^c
ClipWait, 2
If ErrorLevel
{
    MsgBox, Kopieren des Textes in die Zwischenablage fehlgeschlagen.
    Return
}
MsgBox, clipboard = %clipboard%
Return
