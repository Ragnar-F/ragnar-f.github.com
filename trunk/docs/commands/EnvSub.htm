EnvSub

Legt eine Variable an, die sich selbst mit einem vorgegebenen Wert subtrahiert (kann auch Datum-Zeit-Werte vergleichen). Gleichbedeutend mit: Variable -= Wert

EnvSub, Variable, Wert [, Zeiteinheit]
Variable -= Wert [, Zeiteinheit]
Variable--

Parameter

Variable Der Name der Variable, die bearbeitet wird.
Wert Beliebiger Integer-Wert, Fließkommazahl oder Ausdruck (Ausdrücke werden nicht unterstützt, wenn Zeiteinheit vorhanden ist).
Zeiteinheit

Der Wert wird von der Variable subtrahiert, wobei beide als Datum-Zeit-Stempel im YYYYMMDDHH24MISS-Format behandelt werden. Die Zeiteinheit enthält entweder Seconds, Minutes, Hours oder Days (oder jeweils nur den ersten Buchstaben). Wenn die Variable leer ist, dann wird die aktuelle Uhrzeit verwendet. Ist der Wert leer, dann wird ebenfalls die aktuelle Uhrzeit verwendet.

Das Ergebnis wird immer auf die ganze Zahl abgerundet. Wenn beispielsweise die tatsächliche Differenz zwischen zwei Zeitstempeln 1.999 Tage ist, dann wird die Ausgabe auf 1 Tag gesetzt. Für eine höhere Genauigkeit muss Seconds in der Zeiteinheit verwendet und das Ergebnis durch 60.0, 3600.0 oder 86400.0 geteilt werden.

Wenn die Variable oder der Wert einen ungültigen Zeitstempel oder ein Jahr vor 1601 enthält, dann wird die Variable leer gemacht, um auf ein Problem hinzuweisen.

Die interne Variable A_Now enthält die aktuelle Ortszeit im YYYYMMDDHH24MISS-Format.

Zur genauen Bestimmung der verstrichenen Zeit zwischen zwei Ereignissen wird die A_TickCount-Methode verwendet, weil A_TickCount eine Präzision im Millisekundenbereich unterstützt.

Für das Addieren oder Subtrahieren von Sekunden, Minuten, Stunden oder Tagen einer bestimmten Zahl mit einem Zeitstempel wird EnvAdd verwendet (mit einer negativen Zahl wird eine Subtraktion erreicht).

Bemerkungen

Dieser Befehl ist gleichbedeutend mit der Kurzfassung: Variable -= Wert

Der Wert einer Variable kann um 1 erhöht oder verringert werden, wenn Variable++, Variable--, ++Variable oder --Variable verwendet wird.

Wenn die Variable oder der Wert leer ist oder nicht mit einer Zahl beginnt, dann wird eine 0 als Berechnungsgrundlage verwendet (außer bei interner Verwendung in Ausdrücken oder bei Verwendung des Zeiteinheit-Parameters).

Wenn die Variable oder der Wert einen Dezimalpunkt enthält, dann ist das Endergebnis eine Fließkommazahl, abhängig vom eingestellten Format (SetFormat).

Verwandte Befehle

EnvAdd, EnvMult, EnvDiv, SetFormat, Ausdrücke, If Variable is [not] Typ, SetEnv, FileGetTime

Beispiel

EnvSub, EigeneZählung, 2
EigeneZählung -= 2  ; das Gleiche wie oben

Variable1 = 20050126
Variable2 = 20040126
EnvSub, Variable1, %Variable2%, days
MsgBox, %Variable1%  ; Das Ergebnis ist 366, da 2004 ein Schaltjahr ist.
