Break

Beendet bzw. verlässt eine Schleife. Nur in einer Schleife gültig.

Break [, LoopLabel]

[AHK_L 59+]: Wenn angegeben, kennzeichnet LoopLabel, für welche Schleife diese Anweisung gilt. Entweder mit dem Labelnamen oder der numerischen Verschachtelungsebene. Wenn nicht vorhanden oder 1, dann gilt diese Anweisung für die innerste Schleife, in dem sie eingeschlossen ist. LoopLabel muss ein konstanter Wert sein, d. h. Variablen und Ausdrücke werden nicht unterstützt.

Break und Continue sollten gegenüber Goto immer vorgezogen werden, da sich Scripts damit einfacher lesen und pflegen lassen.

Verwandte Befehle

Continue, Loop, While-Loop, For-Loop, Block

Beispiel

Loop
{
    ...
    If var > 25
        break
    ...
    If var <= 5
        continue
}
; Unterbricht die äußere Schleife.
outer:
Loop 3
{
    x := A_Index
    Loop 3
    {
        If (x*A_Index = 6)
            break outer  ; Gleichwertig mit break 2 oder goto break_outer.
        MsgBox %x%,%A_Index%
    }
}
break_outer: ; Für goto.
