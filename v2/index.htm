AutoHotkey v2

AutoHotkey v2 soll die Benutzerfreundlichkeit und Einfachheit der Sprache und des Befehlssatzes verbessern, ohne die Abwärtskompatibilität zu berücksichtigen. Da Funktionsumfang, Verhalten und Syntax jederzeit noch geändert werden können, ist diese Version nicht für den allgemeinen Gebrauch bestimmt.

Downloads

Diese Downloads beinhalten AutoHotkey.exe und einige verschiedene Funktions-Scripts.

  • Unicode 32-Bit - für 32-Bit- oder 64-Bit-Systeme.
  • Unicode 64-Bit - nur für 64-Bit-Systeme.
  • Source-Code - über git oder per Klick auf den ZIP-Button auf GitHub herunterladen.

Die Binärdateien wurden mit Microsoft Visual C++ 2010 erstellt. Windows XP und höher werden unterstützt.

Source-Code ist auf GitHub verfügbar.
