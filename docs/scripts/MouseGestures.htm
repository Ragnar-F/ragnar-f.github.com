Mausgesten -- von deguix

Dieses Script überwacht die Mausbewegungen des Benutzers, sobald die rechte Maustaste gedrückt gehalten wird. Wenn der Benutzer eine erkennbare Form "zeichnet", dann wird ein Programm oder eine andere benutzerdefinierte Aktion ausgeführt (wie Hotkeys). Siehe beigefügte README-Datei, wie Gesten definiert werden können.
