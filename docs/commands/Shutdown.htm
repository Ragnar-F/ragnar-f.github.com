Shutdown

Das System kann entweder heruntergefahren, neu gestartet oder abgemeldet werden.

Shutdown, Code

Parameter

Code Eine Kombination von Codes aus der unteren Liste.

Bemerkungen

Der Code ist eine Kombination der folgenden Werte:

Abmelden 0
Herunterfahren 1
Neustarten 2
Erzwingen 4
Ausschalten 8
Sperren/Ruhezustand Siehe DllCall-Beispiel im untersten Abschnitt.

Addiert die gewünschten Werte. Zum Beispiel ist der Code eine 9 (Herunterfahren + Ausschalten = 1 + 8 = 9), um das System herunterzufahren und auszuschalten. Alternativ kann ein Ausdruck wie 1+8 angegeben werden.

Der "Erzwingen"-Wert (4) zwingt das System, alle offenen Anwendungen zu schließen. Sollte nur im Notfall verwendet werden, weil jede offene Anwendung dadurch Daten verlieren kann.

Der "Ausschalten"-Wert fährt das System herunter und schaltet den Strom aus.

Beachtet zudem, dass ein Script mit OnExit erkennen kann, wann das System heruntergefahren oder der Benutzer abgemeldet wird.

Verwandte Befehle

Run, ExitApp, OnExit

Beispiel

; Erzwingt einen Neustart (Neustart + Erzwingen = 2 + 4 = 6):
Shutdown, 6

; Ruft die API-Funktion "SetSuspendState" von Windows auf, um das System zu sperren oder im Ruhemodus zu versetzen.
; Windows 95/NT4: Da diese Funktion nicht vorhanden ist, hat der folgende Aufruf keine Wirkung.
; Parameter #1: Übergibt 1 anstelle von 0, um den Ruhemodus zu aktivieren, anstatt zu sperren.
; Parameter #2: Übergibt 1 anstelle von 0, um das System sofort ohne Nachfrage zu sperren.
; Parameter #3: Übergibt 1 anstelle von 0, um alle Weck-Ereignisse zu deaktivieren.
DllCall("PowrProf\SetSuspendState", "int", 0, "int", 0, "int", 0)
