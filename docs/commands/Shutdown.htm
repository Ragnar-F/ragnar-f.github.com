Shutdown

Bewirkt, dass das System entweder heruntergefahren, neugestartet oder abgemeldet wird.

Shutdown, Code

Parameter

Code Eine Kombination von den unten genannten Codes.

Bemerkungen

Der Code ist eine Kombination von folgenden Werten:

Abmelden 0
Herunterfahren 1
Neustarten 2
Erzwingen 4
Ausschalten 8
Anhalten/Ruhezustand Siehe DllCall-Beispiel im untersten Abschnitt.

Addiere die gewünschten Werte. Zum Beispiel ist zum Herunterfahren und Ausschalten des Systems eine 9 notwendig (Herunterfahren + Ausschalten = 1 + 8 = 9). Alternativ kann ein Ausdruck wie 1+8 angegeben werden.

Der "Erzwingen"-Wert (4) erzwingt das Schließen aller offenen Anwendungen. Dieser sollte nur im Notfall verwendet werden, weil offene Anwendungen dadurch Daten verlieren könnten.

Der "Ausschalten"-Wert fährt das System herunter und schaltet den Strom aus.

Desweiteren kann ein Script mit OnExit erkennen, wann das System heruntergefahren oder der Benutzer abgemeldet wird.

Verwandte Befehle

Run, ExitApp, OnExit

Beispiel

; Erzwingt einen Neustart (Neustart + Erzwingen = 2 + 4 = 6):
Shutdown, 6

; Ruft die API-Funktion "SetSuspendState" von Windows auf, um das System zu sperren oder in den Ruhemodus zu versetzen.
; Windows 95/NT4: Da diese Funktion nicht vorhanden ist, hat der folgende Aufruf keine Wirkung.
; Parameter #1: Übergebe 1 anstatt 0, um den Ruhemodus zu aktivieren, anstatt anzuhalten.
; Parameter #2: Übergebe 1 anstatt 0, um das System ohne Nachfrage sofort anzuhalten.
; Parameter #3: Übergebe 1 anstatt 0, um alle Aufwachereignisse zu deaktivieren.
DllCall("PowrProf\SetSuspendState", "int", 0, "int", 0, "int", 0)
