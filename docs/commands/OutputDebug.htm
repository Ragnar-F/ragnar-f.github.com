OutputDebug

Sendet einen String an einem Debugger (falls vorhanden), um ihn anzeigen zu lassen.

OutputDebug, Text

Parameter

Text Der Text, der an einem Debugger gesendet wird, damit dieser angezeigt wird. Mit LF-Zeichen (`n) können neue Zeilen begonnen werden. Außerdem kann eine lange Zeile in mehreren kurzen Zeilen mit einem Fortsetzungsbereich aufgetrennt werden.

Bemerkungen

Wenn dem Prozess des Scripts kein Debugger zugeordnet ist, dann zeigt stattdessen der Systemdebugger die den String. Wenn der Systemdebugger nicht aktiviert ist, dann hat dieser Befehl keine Auswirkung.

DebugView ist beispielsweise ein Debugger, der frei verfügbar ist und unter www.sysinternals.com heruntergeladen werden kann.

Siehe auch: andere Debug-Methoden

Verwandte Befehle

FileAppend, Fortsetzungsbereiche

Beispiel

OutputDebug, %A_Now%: Prozess abgebrochen, weil "%TargetWindowTitle%" nicht existiert.
