OutputDebug

Sendet einen String, der von einem beliebigen Debugger abgefangen und angezeigt werden kann (falls vorhanden).

OutputDebug, Text

Parameter

Text Der zu sendende Text, der von einem Debugger abgefangen und angezeigt werden kann. Dieser Text kann Zeilenumbruchszeichen (`n) enthalten, um neue Zeilen beginnen zu können. Außerdem kann mithilfe eines Fortsetzungsbereichs eine lange Zeile in mehreren kurzen Zeilen aufgeteilt werden.

Bemerkungen

Wenn der Prozess des Scripts keinen Debugger hat, wird stattdessen der System-Debugger den String anzeigen. Falls der System-Debugger deaktiviert ist, wird dieser Befehl nicht funktionieren.

Als Debugger kann beispielsweise DebugView verwendet werden, das frei zur Verfügung steht und auf www.sysinternals.com heruntergeladen werden kann.

Siehe auch: andere Debug-Methoden

Verwandte Befehle

FileAppend, Fortsetzungsbereiche

Beispiel

OutputDebug, %A_Now%: Prozess wurde unterbrochen, weil das Fenster "%Zielfenstertitel%" nicht vorhanden ist.
