OutputDebug

Sendet einen String zum Debugger (falls vorhanden), um ihn anzeigen zu lassen.

OutputDebug, Text

Parameter

Text Der Text, der zum Debugger gesendet wird, um ihn anzeigen zu lassen. Mit Zeilenumbruchszeichen (`n) können neue Zeilen begonnen werden. Außerdem kann mithilfe eines Fortsetzungsbereichs eine lange Zeile in mehreren kurzen Zeilen aufgeteilt werden.

Bemerkungen

Ist dem Script-Prozess kein Debugger zugeordnet, wird der System-Debugger den String stattdessen anzeigen. Falls der System-Debugger deaktiviert ist, hat dieser Befehl keine Auswirkung.

Als Debugger kann zum Beispiel DebugView verwendet werden, dieser kann von www.sysinternals.com kostenlos heruntergeladen werden.

Siehe auch: andere Debug-Methoden

Verwandte Befehle

FileAppend, Fortsetzungsbereiche

Beispiel

OutputDebug, %A_Now%: Prozess wurde unterbrochen, weil das Fenster "%Zielfenstertitel%" nicht vorhanden ist.
