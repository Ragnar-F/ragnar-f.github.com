#NoTrayIcon

Deaktiviert das Anzeigen eines Tray-Icons.

#NoTrayIcon

Mit dieser Anweisung irgendwo in einem Script wird das Anzeigen des Tray-Icons verhindert, wenn das Script gestartet wird (auch wenn das Script kompiliert ist).

Wenn diese Anweisung für ein Script mit Hotkeys verwendet werden soll, dann könnte ein Hotkey mit dem ExitApp-Befehl verbunden werden. Ansonsten existiert kein einfacher Weg, das Programm zu beenden (ohne dabei den Computer neu zu starten oder den Prozess zu beenden). Zum Beispiel: #x::ExitApp

Das Tray-Icon kann jederzeit während der Ausführung des Scripts mit dem Befehl Menu, Tray, Icon oder Menu, Tray, NoIcon ein- oder ausgeblendet werden. Der einzige Nachteil von Menu, Tray, NoIcon am Anfang des Scripts ist, dass das Tray-Icon möglicherweise kurz angezeigt wird, wenn das Script startet. Um dies zu verhindern, verwendet stattdessen #NoTrayIcon.

Die interne Variable A_IconHidden enthält 1, wenn das Tray-Icon zurzeit versteckt ist, andernfalls 0.

Verwandte Befehle

Menu, ExitApp

Beispiel

#NoTrayIcon
