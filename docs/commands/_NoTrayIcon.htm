#NoTrayIcon

Deaktiviert die Darstellung des Tray-Icons.

#NoTrayIcon

Diese Direktive irgendwo im Script verhindert, dass das Tray-Icon beim Starten des Scripts angezeigt wird (auch dann, wenn das Script kompiliert ist).

Da es aufgrund dieser Direktive keinen einfachen Weg mehr gibt, das Programm zu beenden (ohne dabei den Computer neu zu starten oder den Prozess zu beenden), wäre es zum Beispiel sinnvoll, einem Hotkey den ExitApp-Befehl zuzuweisen. Zum Beispiel: #x::ExitApp

Während der Ausführung des Scripts kann das Tray-Icon jederzeit per Menu, Tray, Icon eingeblendet oder per Menu, Tray, NoIcon wieder ausgeblendet werden. Die Verwendung von Menu, Tray, NoIcon am Anfang des Scripts hat den Nachteil, dass das Tray-Icon beim Starten des Scripts kurz angezeigt wird. Verwende stattdessen #NoTrayIcon, um das zu vermeiden.

Die interne Variable A_IconHidden enthält 1, wenn das Tray-Icon zurzeit versteckt ist, andernfalls 0.

Verwandte Befehle

Menu, ExitApp

Beispiel

#NoTrayIcon
