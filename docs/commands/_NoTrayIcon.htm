#NoTrayIcon

Deaktiviert die Darstellung des Tray-Icons.

#NoTrayIcon

Diese Direktive irgendwo im Script verhindert, dass das Tray-Icon beim Starten des Scripts angezeigt wird (auch dann, wenn das Script kompiliert ist).

Verwendet man diese Direktive bei einem Script, das Hotkeys hat, empfiehlt es sich, einen Hotkey an den ExitApp-Befehl zu binden. Ansonsten gibt es keine einfache Möglichkeit, das Programm zu beenden (ohne dabei den Computer neu zu starten oder den Prozess zu beenden). Zum Beispiel: #x::ExitApp.

Mit Menu, Tray, Icon oder Menu, Tray, NoIcon kann das Tray-Icon jederzeit ein- oder ausgeblendet werden, während das Script ausgeführt wird. Befindet sich Menu, Tray, NoIcon ganz oben im Script, hat das den Nachteil, dass das Tray-Icon beim Starten des Scripts kurz sichtbar sein wird. Mit #NoTrayIcon kann das stattdessen verhindert werden.

Die interne Variable A_IconHidden enthält 1, wenn das Tray-Icon zurzeit versteckt ist, andernfalls 0.

Verwandte Befehle

Menu, ExitApp

Beispiel

#NoTrayIcon
