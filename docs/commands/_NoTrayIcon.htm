#NoTrayIcon

Deaktiviert das Anzeigen eines Tray-Icons.

#NoTrayIcon

Mit dieser Direktive irgendwo im Script wird das Tray-Icon beim Starten des Scripts verhindert (auch wenn das Script kompiliert ist).

Falls diese Direktive für ein Script mit Hotkeys verwendet werden soll, kann einem Hotkey ein ExitApp-Befehl zugewiesen werden. Ansonsten existiert kein einfacher Weg, das Programm zu beenden (ohne dabei den Computer neu zu starten oder den Prozess zu beenden). Zum Beispiel: #x::ExitApp

Das Tray-Icon kann jederzeit während der Ausführung des Scripts mit dem Befehl Menu, Tray, Icon oder Menu, Tray, NoIcon ein- oder ausgeblendet werden. Der einzige Nachteil von Menu, Tray, NoIcon am Anfang des Scripts ist, dass das Tray-Icon möglicherweise kurz angezeigt wird, wenn das Script startet. Um das zu verhindern, verwendet stattdessen #NoTrayIcon.

Die interne Variable A_IconHidden enthält 1, wenn das Tray-Icon zurzeit versteckt ist, andernfalls 0.

Verwandte Befehle

Menu, ExitApp

Beispiel

#NoTrayIcon
