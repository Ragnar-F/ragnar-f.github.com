{...} (Block)

Zwei geschweifte Klammern kennzeichnen einen Block. Blöcke werden für gewöhnlich bei Funktionen, Else, Loop, While-Loop und If-Befehle benutzt.

{
Beliebig viele Befehle
}

Bemerkungen

Ein Block wird benutzt, um zwei oder mehr Befehle zu vereinen. Es kann auch dazu verwendet werden, die Zugehörigkeiten von IF und ELSE zu ändern. Im nachfolgenden Beispiel wird das ELSE dem ersten statt dem zweiten IF zugeordnet:

If var1 = 1
{
    If var2 = abc
        sleep, 1
}
Else
    Return

Obwohl Blöcke überall verwendet werden können, sind sie zurzeit nur in Verbindung mit Funktionen, Else, Loop oder einem IF-Befehl wie IfEqual oder IfWinExist nützlich.

Wenn IF, ELSE, Loop, While-Loop oder For-Loop nur einen Befehl haben, dann muss dieser Befehl nicht von einem Block umschlossen werden. Das Setzen der Klammern kann aber bei bestimmten Fällen die Lesbarkeit oder Wartbarkeit des Scripts verbessern.

Ein Block kann leer sein (ohne Befehle). Das ist nützlich, falls der Inhalt eines Blocks auskommentiert werden soll, ohne den Block selbst zu entfernen.

One True Brace (OTB, K&R-Stil): Der OTB-Stil darf optional bei folgenden Befehlen verwendet werden: If(Ausdruck), Else, While-Loop, For-Loop, Loop, Funktionsdefinitionen, Try und Catch. Dieser Stil stellt die öffnende Klammer des Blocks auf die gleiche Zeile der Block-Anweisung. Zum Beispiel:

If (x < y) {
    ...
} Else {
    ...
}
While x < y {
    ...
}
For k,v in obj {
    ...
}
Loop %Zählung% {
    ...
}
EigeneFunktion(x, y) {
    ...
}
Try {
    ...
} Catch e {
    ...
}

Ebenso kann ein Befehl oder eine andere Aktion an der rechten Seite der Klammer stehen (außer an der öffnenden Klammer beim OTB-Stil). Zum Beispiel:

If x = 1
{ MsgBox Diese Zeile ist an der rechten Seite der öffnenden Klammer. Wird ausgeführt, sobald IF-Anweisung wahr ist.
    MsgBox Das ist die nächste Zeile.
} MsgBox Diese Zeile ist an der rechten Seite der schließenden Klammer. Sie wird bedingungslos ausgeführt.

Verwandte Befehle

Funktionen, While-Loop, Loop, Else, If, If(Ausdruck)

Beispiel

If x = 1
{
    MsgBox, test1
    Sleep, 5
}
Else
    MsgBox, test2
