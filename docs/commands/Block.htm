{...} (Block)

Zwei geschweifte Klammern kennzeichnen einen Block. Blöcke werden typischerweise bei Funktionen, Else, Loop, While-Loop und If-Befehlen verwendet.

{
Beliebig viele Befehle
}

Bemerkungen

Ein Block wird benutzt, um zwei oder mehr Befehle zu vereinen. Es kann auch dazu verwendet werden, die Zugehörigkeiten von IF und ELSE zu ändern. Im nachfolgenden Beispiel wird das ELSE dem ersten statt dem zweiten IF zugeordnet:

If var1 = 1
{
    If var2 = abc
        sleep, 1
}
Else
    Return

Blöcke sind praktisch überall nutzbar. Sinnvoll sind sie aber nur bei Funktionen, Else, Loop, oder IF-artigen Befehlen wie IfEqual oder IfWinExist.

Wenn ein IF, ELSE, Loop, While-Loop oder For-Loop nur einen einzigen Befehl hat, muss dieser nicht in ein Block gesetzt werden. Allerdings kann das in bestimmten Fällen dabei helfen, das Script besser lesbar oder verwaltbar zu machen.

Ein Block kann leer sein (ohne Befehle), was nützlich sein könnte, wenn der Inhalt eines Blocks auskommentiert werden soll, ohne den Block selbst entfernen zu müssen.

One True Brace (OTB, K&R-Stil): Der OTB-Stil darf optional bei folgenden Befehlen verwendet werden: If (Ausdruck), Else, While-Loop, For-Loop, Loop, Funktionsdefinitionen, Try und Catch. Dieser Stil erlaubt es, dass man die öffnende Klammer des Blocks direkt neben der Block-Anweisung setzen darf, anstatt einzeln auf der nachfolgenden Zeile. Zum Beispiel:

If (x < y) {
    ...
} Else {
    ...
}
While x < y {
    ...
}
For k,v in obj {
    ...
}
Loop %Wiederholungsanzahl% {
    ...
}
MeineFunktion(x, y) {
    ...
}
Try {
    ...
} Catch e {
    ...
}

Ebenso kann ein Befehl oder eine andere Aktion auf der rechten Seite einer Klammer vorkommen (außer die öffnende Klammer des OTB-Stils). Zum Beispiel:

If x = 1
{ MsgBox Diese Zeile befindet sich auf der rechten Seite der öffnenden Klammer. Sie wird ausgeführt, sobald die IF-Anweisung wahr ist.
    MsgBox Das ist die nächste Zeile.
} MsgBox Diese Zeile befindet sich auf der rechten Seite der schließenden Klammer. Sie wird bedingungslos ausgeführt.

Verwandte Befehle

Funktionen, While-Loop, Loop, Else, If, If (Ausdruck)

Beispiel

If x = 1
{
    MsgBox, test1
    Sleep, 5
}
Else
    MsgBox, test2
