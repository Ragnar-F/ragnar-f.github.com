FileCreateDir

Erstellt ein(en) Verzeichnis/Ordner. 

FileCreateDir, Verzeichnisname

Parameter

Verzeichnisname Der Name des Verzeichnisses, das sich im A_WorkingDir-Verzeichnis befindet, solange kein absoluter Pfad angegeben ist.

ErrorLevel

Bei Problemen wird ErrorLevel auf 1 gesetzt, ansonsten enthält ErrorLevel eine 0.

Die interne Variable A_LastError enthält den Rückgabewert der Systemfunktion GetLastError().

Bemerkungen

Dieser Befehl erstellt auch alle übergeordneten Verzeichnisse im Verzeichnisnamen, wenn sie noch nicht vorhanden sind.

Verwandte Befehle

FileRemoveDir

Beispiel

FileCreateDir, C:\Test1\Eigene Bilder\Ordner2
