FileCreateDir

Erstellt ein(en) Verzeichnis/Ordner.

FileCreateDir, Verzeichnisname

Parameter

Verzeichnisname Der Name des Verzeichnisses, das sich im A_WorkingDir-Verzeichnis befindet, solange kein absoluter Pfad angegeben ist.

ErrorLevel

[v1.1.04+] Dieser Befehl ist in der Lage, bei Misserfolg eine Ausnahme auszulösen. Für mehr Informationen, siehe Laufzeitfehler.

ErrorLevel wird bei Misserfolg auf 1 und bei Erfolg auf 0 gesetzt.

Die interne Variable A_LastError enthält den Rückgabewert der Systemfunktion GetLastError().

Bemerkungen

Dieser Befehl erstellt auch alle übergeordneten Verzeichnisse im Verzeichnisnamen, falls sie noch nicht vorhanden sind.

Verwandte Befehle

FileRemoveDir

Beispiel

FileCreateDir, C:\Test1\Eigene Bilder\Ordner2
