WinHide

Versteckt das angegebene Fenster.

WinHide [, Fenstertitel, Fenstertext, Titelausnahme, Textausnahme]

Parameter

Fenstertitel Der Titel oder Teiltitel des Fensters (das Übereinstimmungsverhalten wird mit SetTitleMatchMode bestimmt). Wenn dieser und die anderen 3 Fensterparameter weggelassen werden, wird das zuletzt gefundene Fenster verwendet. Sobald dieser den Buchstaben A enthält und die anderen 3 Fensterparameter weggelassen werden, wird das aktive Fenster verwendet. Bei Verwendung einer Fensterklasse muss ahk_class ExakterKlassenname angegeben werden (wird von Windows Spy angezeigt). Bei Verwendung einer Prozess-Kennung (PID) muss ahk_pid %VariableMitPID% angegeben werden. Um eine Fenstergruppe zu verstecken, verwendet ahk_group EigenerGruppenname (in diesem Fall müssen Fenstertext, Titelausnahme und Textausnahme leer sein). Bei Verwendung einer eindeutigen ID-Nummer eines Fensters muss ahk_id %VariableMitID% angegeben werden. Die Suche kann durch Angabe mehrerer Kriterien eingeschränkt werden. Zum Beispiel: Eigene Datei.txt ahk_class Notepad
Fenstertext In diesem Parameter kann ein Teilstring angegeben werden, der in einem Textelement des Zielfensters vorkommt (werden mit Windows Spy angezeigt). Versteckte Textelemente werden auch erkannt, sofern DetectHiddenText aktiv ist.
Titelausnahme Ignoriert Fenster, deren Titel diesen Wert enthalten.
Textausnahme Ignoriert Fenster, deren Textelemente diesen Wert enthalten.

Bemerkungen

Mit WinShow kann ein verstecktes Fenster wieder sichtbar gemacht werden (DetectHiddenWindows kann entweder On oder Off sein, um das zu bewerkstelligen).

Dieser Befehl bezieht sich immer auf das oberste übereinstimmende Fenster, es sein denn, es befindet sich ahk_group Groupname in Fenstertitel, in diesem Fall sind alle Fenster der Gruppe davon betroffen.

Die Explorer-Taskleiste kann wie folgt versteckt/angezeigt werden:

WinHide ahk_class Shell_TrayWnd
WinShow ahk_class Shell_TrayWnd

Verwandte Befehle

WinShow, SetTitleMatchMode, DetectHiddenWindows, Zuletzt gefundenes Fenster, WinSet

Beispiel

Run, notepad.exe
WinWait, Unbenannt - Editor
Sleep, 500
WinHide ; verwendet das oben gefundene Fenster
Sleep, 1000
WinShow
