WinHide

Versteckt das angegebene Fenster.

WinHide [, Fenstertitel, Fenstertext, Titelausnahme, Textausnahme]

Parameter

Fenstertitel Der Titel oder Teiltitel des Fensters (das Übereinstimmungsverhalten wird mit SetTitleMatchMode bestimmt). Wenn dieser und die anderen 3 Fensterparameter nicht vorhanden sind, dann wird das zuletzt gefundene Fenster verwendet. Wenn dieser den Buchstaben A enthält und die anderen 3 Fensterparameter nicht vorhanden sind, dann wird das aktive Fenster verwendet. Bei Verwendung einer Fensterklasse muss ahk_class ExakterKlassenname angegeben werden (wird von Windows Spy angezeigt). Bei Verwendung einer Prozess-Kennung (PID) muss ahk_pid %VariableMitPID% angegeben werden. Um eine Fenstergruppe zu verstecken, verwendet ahk_group EigenerGruppenname (in diesem Fall müssen Fenstertext, Titelausnahme und Textausnahme leer sein). Bei Verwendung einer eindeutigen ID-Nummer eines Fensters muss ahk_id %VariableMitID% angegeben werden. Die Suche kann durch Angabe mehrerer Kriterien eingeschränkt werden. Zum Beispiel: Eigene Datei.txt ahk_class Notepad
Fenstertext Wenn vorhanden, dann muss dieser Parameter einen Teil der einzelnen Textelemente des Fensters enthalten (wird von Windows Spy angezeigt). Versteckte Textelemente werden erfasst, wenn DetectHiddenText aktiv ist.
Titelausnahme Fenstertitel mit diesem Wert werden nicht berücksichtigt.
Textausnahme Fenstertexte mit diesem Wert werden nicht berücksichtigt.

Bemerkungen

Verwendet WinShow, um ein verstecktes Fenster wieder sichtbar zumachen (DetectHiddenWindows kann dafür entweder ein- oder ausgeschaltet werden).

Dieser Befehl arbeitet immer nur mit dem ersten gefundenen Fenster, es sein denn, es befindet sich ahk_group Groupname in Fenstertitel. In dem Fall sind alle Fenster der Gruppe betroffen.

Die Explorer-Taskleiste kann wie folgt versteckt/angezeigt werden:
WinHide ahk_class Shell_TrayWnd
WinShow ahk_class Shell_TrayWnd

Verwandte Befehle

WinShow, SetTitleMatchMode, DetectHiddenWindows, Zuletzt gefundenes Fenster, WinSet

Beispiel

Run, notepad.exe
WinWait, Unbenannt - Editor
Sleep, 500
WinHide ; verwendet das zuletzt gefundene Fenster
Sleep, 1000
WinShow
