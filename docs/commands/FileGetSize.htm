FileGetSize

Ermittelt die Größe einer Datei.

FileGetSize, Ausgabevariable [, Dateiname, Einheit]

Parameter

Ausgabevariable Der Name der Variable, in der die ermittelte Größe gespeichert wird (auf die nächste ganze Zahl abgerundet).
Dateiname Der Name der Datei, die sich im A_WorkingDir-Verzeichnis befindet, solange kein absoluter Pfad angegeben ist. Falls weggelassen, wird die aktuelle Datei von der innersten dateilesenden Schleife verwendet.
Einheit

Falls vorhanden, bewirkt dieser Parameter, dass das Ergebnis in einer bestimmten Einheit ausgegeben wird (standardmäßig in Bytes):
K = Kilobytes
M = Megabytes

ErrorLevel

[v1.1.04+] Dieser Befehl ist in der Lage, bei Misserfolg eine Ausnahme auszulösen. Für mehr Informationen, siehe Laufzeitfehler.

ErrorLevel wird bei Misserfolg auf 1 und bei Erfolg auf 0 gesetzt.

Die interne Variable A_LastError enthält den Rückgabewert der Systemfunktion GetLastError().

Bemerkungen

Es werden Dateien in beliebiger Größe unterstützt, auch solche, die größer als 4 GB sind, und auch, wenn Einheit in Bytes ist.

Wenn Dateiname ein Verzeichnis ist, wird eine Größe abhängig vom Betriebssystem ausgegeben (in den meisten Fällen wahrscheinlich 0).

Um die Größe des Ordners einschließlich seiner Dateien zu berechnen, siehe folgendes Beispiel:

SetBatchLines, -1  ; Führt die Operation mit maximaler Geschwindigkeit aus.
Ordnergröße = 0
FileSelectFolder, BeliebigerOrdner  ; Fordert den Benutzer auf, einen Ordner auszuwählen.
Loop, %BeliebigerOrdner%\*.*, , 1
    Ordnergröße += %A_LoopFileSize%
MsgBox Der Ordner %BeliebigerOrdner% hat eine Größe von %Ordnergröße% Bytes.

Verwandte Befehle

FileGetAttrib, FileSetAttrib, FileGetTime, FileSetTime, FileGetVersion, dateilesende Schleife

Beispiel

FileGetSize, Ausgabevariable, C:\Eigene Dokumente\test.doc  ; Ermittelt die Größe in Bytes.
FileGetSize, Ausgabevariable, C:\Eigene Dokumente\test.doc, K  ; Ermittelt die Größe in Kilobytes.
