FileRecycleEmpty

Leert den Papierkorb. 

FileRecycleEmpty [, Laufwerksbuchstabe]

Parameter

Laufwerksbuchstabe Wenn nicht vorhanden, dann wird der Papierkorb auf allen Laufwerken geleert. Ansonsten kann ein Laufwerksbuchstabe wie C:\ angegeben werden.

ErrorLevel

Bei Problemen wird ErrorLevel auf 1 gesetzt, ansonsten enthält ErrorLevel eine 0.

Bemerkungen

Dieser Befehl benötigt MS Internet Explorer 4 oder höher.

Verwandte Befehle

FileRecycle, FileDelete, FileCopy, FileMove

Beispiel

FileRecycleEmpty, C:\
