DirCopy

Kopiert einen Ordner mit all seinen Unterordnern und Dateien (ähnlich wie xcopy).

DirCopy, Quelle, Ziel [, Flag]

Parameter

Quelle

Der Name des Quellverzeichnisses (ohne Backslash), das sich im A_WorkingDir-Verzeichnis befindet, solange kein absoluter Pfad angegeben ist. Zum Beispiel: C:\Eigener Ordner

Ziel

Der Name des Zielverzeichnisses (ohne Backslash), das sich im A_WorkingDir-Verzeichnis befindet, solange kein absoluter Pfad angegeben ist. Zum Beispiel: C:\Kopie des eigenen Ordners

Flag

(Optional) Mit diesem Parameter wird bestimmt, ob Dateien überschrieben werden, wenn sie bereits vorhanden sind:

0 (Standard): Vorhandene Dateien werden nicht überschrieben. Der Vorgang schlägt fehl, wenn das Ziel bereits als Datei oder Verzeichnis existiert.

1: Vorhandene Dateien werden überschrieben. Allerdings wird jede Datei oder jeder Unterordner innerhalb vom Ziel ohne entsprechendes Gegenstück in der Quelle nicht gelöscht.

ErrorLevel

[v1.1.04+] Dieser Befehl ist in der Lage, bei Misserfolg eine Ausnahme auszulösen. Für mehr Informationen, siehe Laufzeitfehler.

ErrorLevel wird bei Misserfolg auf 1 und bei Erfolg auf 0 gesetzt. Wenn das Quellverzeichnis eine beliebige gespeicherte Webseite enthält, bestehend aus Seitenname.htm und dem dazugehörigen Verzeichnis mit dem Namen Seitenname_Dateien, dann kann ein Fehler angezeigt werden, selbst wenn das Kopieren erfolgreich war.

Bemerkungen

Wenn die Struktur des Zielverzeichnisses nicht existiert, dann wird sie erstellt, wenn möglich.

Da die Operation einen Ordner mit all seinen Unterordnern und Dateien rekursiv kopiert, ist das Ergebnis unbestimmt, wenn ein Ordner irgendwo in sich selbst kopiert wird. Als Übergangslösung wäre es möglich, eine Kopie des Ordners außerhalb des Zielverzeichnisses zu erstellen und dann mit DirMove die Kopie an die gewünschte Stelle zu verschieben.

DirCopy kopiert einen einzelnen Ordner. Um stattdessen den Inhalt eines Ordners zu kopieren (alle Dateien und Unterordner), sieht die Beispiele bei FileCopy.

Verwandte Befehle

DirMove, FileCopy, FileMove, FileDelete, Datei-Schleife, FileSelectFolder, SplitPath

Beispiele

DirCopy, C:\Eigener Ordner, C:\Kopie des eigenen Ordners

; Beispiel #2: Ein Script, der den Benutzer auffordert, einen Ordner zu kopieren.
FileSelectFolder, Quellordner, , 3, Wähle den Ordner`, der kopiert werden soll
If Quellordner =
    Return
; Ansonsten fortfahren.
FileSelectFolder, Zielordner, , 3, Wähle den Ordner, der die Kopie des Ordners enthalten soll.
If Zielordner =
    Return
; Ansonsten fortfahren.
MsgBox, 4, , Eine Kopie des Ordners "%Quellordner%" wird in "%Zielordner%" erstellt.  Weiter?
If A_MsgBoxResult = "No"
    Return
SplitPath, Quellordner, Quellordnernamen ; Nur den Ordnernamen aus dem vollständigen Pfad extrahieren.
DirCopy, %Quellordner%, %Zielordner%\%Quellordnernamen%
If ErrorLevel
    MsgBox Der Ordner konnte nicht kopiert werden, weil vielleicht ein Ordner mit dem selben Namen in "%Zielordner%" bereits existiert.
Return
