EnvDiv

Bestimmt eine Variable, die sich selbst mit einem vorgegebenen Wert dividiert. Gleichbedeutend mit: Var /= Wert.

EnvDiv, Var, Wert

Parameter

Var Name der Variable, die bearbeitet werden soll.
Wert Integer, Fließkommazahl oder Ausdruck.

Bemerkungen

Dieser Befehl ist gleichbedeutend mit der Kurzfassung: Var /= Wert.

Bei der Division durch 0 erscheint eine Fehlermeldung, wenn das Script geladen wird (wenn möglich), ansonsten wird die Variable leer gemacht.

Wenn Var oder Wert leer ist oder nicht mit einer Zahl beginnt, wird eine 0 als Berechnungsgrundlage verwendet (außer bei interner Verwendung in Ausdrücken wie Var := X /= Y).

Wenn Var oder Wert einen Dezimalpunkt enthält, ist das Endergebnis eine Fließkommazahl, abhängig vom eingestellten Format (SetFormat). Andernfalls wird das Ergebnis gekürzt (z. B. 19 geteilt durch 10 ergibt 1).

Verwandte Befehle

EnvAdd, EnvSub, EnvMult, SetFormat, Ausdrücke, If Var is [not] Typ, SetEnv, bitweise Operationen (Transform)

Beispiel

EnvDiv, EigeneZählung, 2
EigeneZählung /= 2  ; das Gleiche wie oben
