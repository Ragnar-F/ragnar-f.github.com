EnvDiv

Bestimmt eine Variable, die sich selbst mit einem vorgegebenen Wert dividiert. Gleichbedeutend mit: Var /= Wert.

EnvDiv, Var, Wert

Parameter

Var

Name der Variable, die bearbeitet werden soll.

Wert

Integer, Fließkommazahl oder Ausdruck.

Bemerkungen

Dieser Befehl ist gleichbedeutend mit der Kurzfassung: Var /= Wert.

Erfolgt eine Division mit Null, erscheint eine Fehlermeldung beim Laden des Scripts (falls möglich); ansonsten wird die Variable leer gemacht.

Wenn Var oder Wert leer oder keine Zahl ist, wird es bei der Berechnung als 0 angesehen (außer innerhalb eines Audrucks wie Var := X /= Y).

Enthält Var oder Wert einen Dezimalpunkt, wird das Endergebnis eine Fließkommazahl sein, abhängig vom gesetzten Format per SetFormat. Ansonsten wird das Ergebnis gekürzt (z. B. würde 19 durch 10 gleich 1 ergeben).

Beispiel

EnvDiv, MeineZählung, 2
MeineZählung /= 2  ; das Gleiche wie oben
