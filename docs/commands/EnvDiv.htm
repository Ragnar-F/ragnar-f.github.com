EnvDiv

Legt eine Variable an, die sich selbst mit einem vorgegebenen Wert dividiert. Gleichbedeutend mit: Variable /= Wert

EnvDiv, Variable, Wert

Parameter

Variable Der Name der Variable, die bearbeitet wird.
Wert Beliebiger Integer-Wert, Fließkommazahl oder Ausdruck.

Bemerkungen

Dieser Befehl ist gleichbedeutend mit der Kurzfassung: Variable /= Wert

Bei der Division durch 0 erscheint eine Fehlermeldung, wenn das Script geladen wird (wenn möglich), ansonsten wird die Variable leer gemacht.

Wenn entweder Variable oder Wert leer ist oder nicht mit einer Zahl beginnt, dann wird eine 0 als Berechnungsgrundlage verwendet (außer bei interner Verwendung in Ausdrücken wie Variable := X /= Y).

Wenn die Variable oder der Wert einen Dezimalpunkt enthält, dann ist das Endergebnis eine Fließkommazahl, abhängig vom eingestellten Format (SetFormat). Andernfalls wird das Ergebnis abgeschnitten (z. B. 19 geteilt durch 10 ergibt 1).

Verwandte Befehle

EnvAdd, EnvSub, EnvMult, SetFormat, Ausdrücke, If Variable is [not] Typ, SetEnv, bitweise Operationen (Transform)

Beispiel

EnvDiv, EigeneZählung, 2
EigeneZählung /= 2  ; das Gleiche wie oben
