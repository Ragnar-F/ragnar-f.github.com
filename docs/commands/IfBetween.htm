If Variable [not] between Untergrenze and Obergrenze

Überprüft, ob der Inhalt einer Variable zwischen zwei Werten liegt (numerisch oder alphabetisch).

If Variable between Untergrenze and Obergrenze
If Variable not between Untergrenze and Obergrenze

Parameter

Variable Name der Variable, deren Inhalt überprüft wird.
Untergrenze Die Variable liegt im angegebenen Bereich, wenn sie identisch mit oder größer als die Untergrenze ist (String, Nummer oder Variablenverweis).
Obergrenze Die Variable liegt im angegebenen Bereich, wenn sie identisch mit oder kleiner als die Obergrenze ist (String, Nummer oder Variablenverweis).

Bemerkungen

Wenn alle drei Parameter numerisch sind, werden sie als Zahlen anstatt als Strings verglichen. Ansonsten werden sie als Strings alphabetisch verglichen (d. h. die alphabetische Reihenfolge bestimmt, ob die Variable im angegeben Bereich ist). In diesem Fall kann StringCaseSense On verwendet werden, um den Vergleich abhängig von der Groß- und Kleinschreibung zu machen.

Die Operatoren "between", "is", "in" und "contains" werden in Ausdrücken nicht unterstützt.

Verwandte Befehle

IfEqual/Greater/Less, If Variable in/contains Übereinstimmungsliste, If Variable is type, IfInString, StringCaseSense, EnvAdd, Blocks, Else

Beispiel

If Variable between 1 and 5
    MsgBox, %Variable% liegt zwischen 1 und 5.

If Variable not between 0.0 and 1.0
    MsgBox %Variable% liegt nicht zwischen 0.0 und 1.0.

If Variable between %Niedrig% and %Hoch%
    MsgBox %Variable% liegt zwischen %Niedrig% und %Hoch%.

If Variable between Blau and Rot
    MsgBox %Variable% liegt zwischen den Wörtern Blau und Rot (alphabetisch).

Untergrenze = 1
Obergrenze = 10
InputBox, UserInput, Bitte eine Zahl zwischen %Untergrenze% und %Obergrenze% eingeben.
If UserInput not between %Untergrenze% and %Obergrenze%
    MsgBox Ihre Eingabe liegt nicht im gültigen Bereich.
