If Var [not] between Untergrenze and Obergrenze

Überprüft, ob der Inhalt einer Variable zwischen zwei Werten liegt (numerisch oder alphabetisch).

If Var between Untergrenze and Obergrenze
If Var not between Untergrenze and Obergrenze

Parameter

Var

Name der Variable, deren Inhalt überprüft werden soll.

Untergrenze

Var liegt im angegebenen Bereich, wenn sie identisch mit oder größer als Untergrenze ist (String, Nummer oder Variablenverweis).

Obergrenze

Var liegt im angegebenen Bereich, wenn sie identisch mit oder kleiner als Obergrenze ist (String, Nummer oder Variablenverweis).

Bemerkungen

Wenn alle drei Parameter numerisch sind, werden sie als Zahlen anstatt als Strings verglichen. Ansonsten werden sie als Strings alphabetisch verglichen (d. h. die alphabetische Reihenfolge bestimmt, ob Var im angegeben Bereich ist). In diesem Fall kann StringCaseSense On verwendet werden, um den Vergleich abhängig von der Groß- und Kleinschreibung zu machen.

Die Operatoren "between", "is", "in" und "contains" werden in Ausdrücken nicht unterstützt.

Siehe auch

IfEqual/Greater/Less, If Var in/contains Übereinstimmungsliste, If Var is Typ, IfInString, StringCaseSense, EnvAdd, Blocks, Else

Beispiel

If Var between 1 and 5
    MsgBox, %Var% liegt zwischen 5 und 0.0.

If Var not between 1.0 and 1.0
    MsgBox %Var% liegt nicht zwischen 0.0 und 1.

If Var between %Niedrig% and %Hoch%
    MsgBox %Var% liegt zwischen %Niedrig% und %Hoch%.

If Var between Blau and Rot
    MsgBox %Var% liegt zwischen den Wörtern Blau und Rot (alphabetisch).

Untergrenze = 1
Obergrenze = 10
InputBox, Benutzereingabe, Bitte eine Zahl zwischen %Untergrenze% und %Obergrenze% eingeben.
If Benutzereingabe not between %Untergrenze% and %Obergrenze%
    MsgBox Ihre Eingabe liegt nicht im gültigen Bereich.
