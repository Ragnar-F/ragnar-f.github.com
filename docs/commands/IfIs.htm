If Var is [not] Typ

Überprüft, ob der Inhalt der Variable zum Beispiel numerisch oder großgeschrieben ist.

If Var is Typ
If Var is not Typ

Parameter

Var

Name der Variable.

Typ

Siehe Bemerkungen.

Bemerkungen

Unterstützte Typen:

integer Wahr, wenn Var nicht leer ist und einen reinen numerischen String (dezimal oder hexadezimal) ohne Dezimalpunkt enthält. Führende oder nachfolgende Leer- und Tabulatorzeichen sind erlaubt. Der String kann mit einem Plus- oder Minuszeichen beginnen.
float Wahr, wenn Var nicht leer ist und eine Fließkommazahl enthält (reiner numerischer String mit Dezimalpunkt). Führende oder nachfolgende Leer- und Tabulatorzeichen sind erlaubt. Der String kann mit einem Pluszeichen, Minuszeichen oder Dezimalpunkt beginnen.
number Wahr, wenn Var eine Integer- oder Fließkommazahl enthält (beide werden weiter oben beschrieben).
digit Wahr, wenn Var leer ist oder nur die Zahlen von 0 bis 9 enthält. Folgende Zeichen sind nicht erlaubt: Leerzeichen, Tabulatorzeichen, Plus- und Minuszeichen, Dezimalpunkte, Hexadezimalzahlen und 0x-Präfix.
xdigit Hexadezimale Zahlen: Das gleiche wie digit, aber mit dem Unterschied, dass auch die Zeichen A bis F (groß- oder kleingeschrieben) erlaubt sind. Seit v1.0.44.09 wird der 0x-Präfix toleriert.
alpha Wahr, wenn Var leer ist oder nur aus alphabetischen Zeichen besteht. Falsch, wenn Zahlen, Leerzeichen, Tabulatorzeichen, Punkte oder andere nicht-alphabetische Zeichen im String sind. Wenn Var zum Beispiel ein Leerzeichen mit einem Buchstaben danach enthält, wird die Variable nicht als alpha betrachtet.
upper Wahr, wenn Var leer ist oder nur Großbuchstaben enthält. Falsch, wenn Zahlen, Leerzeichen, Tabulatorzeichen, Punkte oder andere nicht großgeschriebene Zeichen im String sind.
lower Wahr, wenn Var leer ist oder nur Kleinbuchstaben enthält. Falsch, wenn Zahlen, Leerzeichen, Tabulatorzeichen, Punkte oder andere nicht kleingeschriebene Zeichen im String sind.
alnum Das gleiche wie alpha, aber mit dem Unterschied, dass auch die Zeichen 0 bis 9 erlaubt sind.
space Wahr, wenn Var leer ist oder nur unsichtbare Zeichen enthält, das folgende sein können: Leerzeichen (%A_Space%), Tabulatorzeichen (%A_Tab% oder `t), LF-Zeichen (`n), CR-Zeichen (`r), vertikaler Tabulator (`v) und Seitenvorschub (`f).
time

Wahr, wenn Var einen gültigen Zeitstempel im YYYYMMDDHH24MISS-Format enthält. Zum Beispiel wird ein 4-stelliger String wie 2004 als gültig angesehen. Mit StringLen kann bestimmt werden, ob zusätzliche Zeitkomponenten vorhanden sind.

Jahre unter 1601 sind ungültig, weil diese normalerweise vom Betriebssystem nicht unterstützt werden. Das höchste Jahr ist 9999.

Ersatzweise kann das Wort DATE verwendet werden, das zum gleichen Ergebnis führt.

Hinweis: Die Operatoren "between", "is", "in" und "contains" werden in Ausdrücken nicht unterstützt.

[AHK_L 42+]: Das Gebietsschema des Betriebssystems wird ignoriert, es sei denn, StringCaseSense Locale wurde verwendet.

Siehe auch

%A_YYYY%, SetFormat, FileGetTime, IfEqual, If Var in/contains Übereinstimmungsliste, If Var between, StringLen, IfInString, StringUpper, EnvAdd, Blöcke, Else

Beispiel

If Var is float
    MsgBox, %Var% ist eine Fließkommazahl.
Else If Var is integer
    MsgBox, %Var% ist eine Integerzahl.
If Var is time
    MsgBox, %Var% ist auch ein gültiger Zeitstempel.
