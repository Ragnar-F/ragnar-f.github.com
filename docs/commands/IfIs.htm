If Variable is [not] Typ

Überprüft, ob der Inhalt der Variable numerisch, groß geschrieben, etc.

If Variable is Typ
If Variable is not Typ

Parameter

Variable Der Name der Variable.
Typ Siehe Bemerkungen.

Bemerkungen

Unterstützte Typen:

integer Wahr, wenn die Variable nicht leer ist und einen reinen numerischen String (dezimal oder hexadezimal) ohne Dezimalpunkt enthält. Führende oder nachfolgende Leerzeichen und Tabulatoren sind erlaubt. Der String kann mit einem Plus- oder Minuszeichen beginnen.
float Wahr, wenn die Variable nicht leer ist und eine Fließkommazahl enthält; d. h. ein reiner numerischer String mit Dezimalpunkt. Führende oder nachfolgende Leerzeichen und Tabulatoren sind erlaubt. Der String kann mit einem Pluszeichen, Minuszeichen oder Dezimalpunkt beginnen.
number Wahr, wenn die Variable eine Integer- oder Fließkommazahl enthält (beide werden weiter oben beschrieben).
digit Wahr, wenn die Variable leer ist oder nur Zahlen enthält, die nur aus den Zeichen 0 bis 9 bestehen. Folgende Zeichen sind nicht erlaubt: Leerzeichen, Tabulatoren, Plus- und Minuszeichen, Dezimalpunkte, Hexadezimalzahlen und 0x-Präfix.
xdigit Hexadezimale Zahlen: Das gleiche wie digit mit der Ausnahme, dass die Zeichen A bis F (groß oder klein geschrieben) auch erlaubt sind. Seit v1.0.44.09 wird der 0x-Präfix toleriert.
alpha Wahr, wenn die Variable leer ist oder nur aus alphabetischen Zeichen besteht. Falsch, wenn Zahlen, Leerzeichen, Tabulatoren, Punkte oder andere nicht-alphabetische Zeichen im String sind. Wenn zum Beispiel die Variable ein Leerzeichen, gefolgt von einem Buchstaben, enthält, dann wird die Variable nicht als alpha betrachtet.
upper Wahr, wenn die Variable leer ist oder nur Großbuchstaben enthält. Falsch, wenn Zahlen, Leerzeichen, Tabulatoren, Punkte oder andere nicht-großgeschriebene Zeichen im String sind.
lower Wahr, wenn die Variable leer ist oder nur Kleinbuchstaben enthält. Falsch, wenn Zahlen, Leerzeichen, Tabulatoren, Punkte oder andere nicht-kleingeschriebene Zeichen im String sind.
alnum Das gleiche wie alpha, nur mit der Ausnahme, dass die Zeichen 0 bis 9 auch erlaubt sind.
space Wahr, wenn die Variable leer ist oder nur unsichtbare Zeichen enthält, welche folgende sein können: Leerzeichen (%A_Space%), Tabulator (%A_Tab% oder `t), LF-Zeichen (`n), CR-Zeichen (`r), vertikaler Tabulator (`v) und Seitenvorschub (`f).
time

Wahr, wenn die Variable einen gültigen Zeitstempel im YYYYMMDDHH24MISS-Format enthält. Zum Beispiel wird ein 4-stelliger String wie 2004 als gültig angesehen. Mit StringLen kann bestimmt werden, ob zusätzliche Zeitkomponenten vorhanden sind.

Jahre unter 1601 sind ungültig, weil diese vom Betriebssystem in der Regel nicht unterstützt werden. Das höchste Jahr ist 9999.

Das Wort DATE kann als Ersatz für das Wort TIME verwendet werden, mit dem gleichen Ergebnis.

Hinweis: Die Operatoren "between", "is", "in" und "contains" werden in Ausdrücken nicht unterstützt.

[AHK_L 42+]: Das Gebietsschema des Betriebssystems wird ignoriert, es sei denn, StringCaseSense Locale wurde verwendet.

Verwandte Befehle

%A_YYYY%, SetFormat, FileGetTime, IfEqual, If Variable in/contains Übereinstimmungsliste, If Variable between, StringLen, IfInString, StringUpper, EnvAdd, Blocks, Else

Beispiel

If Variable is float
    MsgBox, %Variable% ist eine Fließkommazahl.
Else If Variable is integer
    MsgBox, %Variable% ist eine Integerzahl.
If Variable is time
    MsgBox, %Variable% ist auch ein gültiger Zeitstempel.
