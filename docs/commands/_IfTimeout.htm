#IfTimeout [AHK_L 8+]

Bestimmt die Zeit, die maximal für die Auswertung des #If-Ausdrucks aufgebracht werden kann.

#IfTimeout Zeitüberschreitung

Parameter

Zeitüberschreitung Der Wert für die globale Zeitüberschreitung in Millisekunden.

Bemerkungen

Eine Zeitüberschreitung wird implementiert, um lang laufende Ausdrücke daran zu hindern, den Tastatureingabeprozess zu unterbrechen. Wenn das Zeitlimit überschritten wird, dann wird der Ausdruck weiterhin ausgewertet, obwohl der Tastatur-Hook den Ausdruck bereits als falsch interpretiert.

Wenn diese Anweisung nicht angegeben ist, dann wird standardmäßig 1000 verwendet.

Beachtet, dass das System eine eigene Zeitüberschreitung implementiert hat. Wenn das Zeitlimit des Systems überschritten wird, dann wird sowohl der Hotkey als auch die ursprüngliche Funktion der Taste aktiviert. Das Zeitlimit des Systems wird vom DWORD-Wert LowLevelHooksTimeout im folgenden Registrierungsschlüssel definiert:

HKEY_CURRENT_USER\Control Panel\Desktop

Verwandte Befehle

#If

Beispiel

#IfTimeout 10 ; Setzt die Zeitüberschreitung auf 10 ms.
