DateDiff

Vergleicht zwei Datum-Zeit-Werte und gibt deren Differenz zurück.

Ausgabevariable := DateDiff(DatumZeit1, DatumZeit2, Zeiteinheit)

Parameter

DatumZeit1,
DatumZeit2

Datum-Zeit-Stempel im YYYYMMDDHH24MISS-Format.

Wenn DatumZeit1 früher als DatumZeit2 ist, dann ist das Ergebnis eine negative Zahl.

Zeiteinheit

Die Einheit zur Bemessung der Differenz. Die Zeiteinheit enthält entweder Seconds, Minutes, Hours oder Days (oder jeweils nur den ersten Buchstaben).

Rückgabewert

Sobald DatumZeit einen ungültigen Zeitstempel oder ein Jahr vor 1601 enthält, oder Zeit nicht numerisch ist, wird ein leerer String zurückgegeben, um den Fehler zu kennzeichnen.

Ansonsten ist der Rückgabewert die Differenz zwischen zwei Zeitstempeln - in der angegebenen Einheit von Zeiteinheit. Das Ergebnis wird immer auf den nächsten Integer abgerundet. Wenn beispielsweise die aktuelle Differenz zwischen zwei Zeitstempeln 1.999 Tage ist, dann wird die Ausgabe auf 1 Tag gesetzt. Für eine höhere Genauigkeit muss Seconds in Zeiteinheit verwendet und das Ergebnis durch 60.0, 3600.0 oder 86400.0 geteilt werden.

Bemerkungen

Die interne Variable A_Now enthält die aktuelle Ortszeit im YYYYMMDDHH24MISS-Format.

Zur genauen Bestimmung der verstrichenen Zeit zwischen zwei Ereignissen wird die A_TickCount-Methode verwendet, weil A_TickCount eine Präzision im Millisekundenbereich unterstützt.

Für das Addieren oder Subtrahieren von Sekunden, Minuten, Stunden oder Tagen einer bestimmten Zahl mit einem Zeitstempel wird DateAdd verwendet (mit einer negativen Zahl wird eine Subtraktion erreicht).

Verwandte Befehle

DateAdd, FileGetTime, FormatTime

Beispiel

Variable1 := "20050126"
Variable2 := "20040126"
Differenz := DateDiff(Variable1, Variable2, "days")
MsgBox %Differenz%  ; Das Ergebnis ist 366, da 2004 ein Schaltjahr ist.
