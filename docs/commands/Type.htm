Type

Ermittelt den genauen Typ des Wertes.

Ausgabevariable := Type(Wert)

Rückgabewert

Der Rückgabewert ist ein String, der den Typ des Wertes kennzeichnet. Folgende Werte können enthalten sein:

  • String
  • Integer
  • Float
  • Object

Falls der Wert ein Objekt ist, könnte der Rückgabewert entweder "Object" oder ein speziellerer interner Typ wie "Func", "FileObject" oder "ComObject" sein. Zur Überprüfung eines beliebigen Objekttyps kann IsObject(Wert) verwendet werden.

Bemerkungen

Diese Funktion sollte normalerweise nicht verwendet werden, um festzustellen, ob ein Wert numerisch ist, weil numerische Strings sowohl in mathematischen Ausdrücken als auch bei den meisten internen Funktionen gültig sind. In einigen Fällen jedoch ist der genaue Typ eines Wertes von größerer Wichtigkeit. Wenn zum Beispiel NumPut eine Variable übergeben wird, die einen puren Integer enthält (keinen numerischen String), dann wird anstelle der Variablenadresse der Integer verwendet.

Beispiele

a := 1, b := 2.0, c := "3"
MsgBox % Type(a)  ; Integer
MsgBox % Type(b)  ; Float
MsgBox % Type(c)  ; String
