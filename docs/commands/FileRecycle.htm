FileRecycle

Verschiebt eine Datei oder ein Verzeichnis in den Papierkorb, falls möglich.

FileRecycle, Dateimuster

Parameter

Dateimuster

Name einer Datei oder eine musterbasierte Suche wie C:\Temp\*.tmp. Dateimuster befindet sich im A_WorkingDir-Verzeichnis, solange kein absoluter Pfad angegeben ist.

Um ein ganzes Verzeichnis zu recyclen, muss dessen Namen ohne nachfolgenden Backslash angegeben werden.

ErrorLevel

[v1.1.04+] Dieser Befehl ist in der Lage, bei Misserfolg eine Ausnahme auszulösen. Für mehr Informationen, siehe Laufzeitfehler.

ErrorLevel wird bei Misserfolg auf 1 und bei Erfolg auf 0 gesetzt.

Bemerkungen

Dieser Befehl verwendet die systeminterne Funktion SHFileOperation. Es ist möglich, dass diese Funktion eine Datei entgültig löscht, falls diese nicht in den Papierkorb passt. Seit v1.0.96 erscheint eine Warnung, bevor das geschieht.

Verwandte Befehle

FileRecycleEmpty, FileDelete, FileCopy, FileMove

Beispiel

FileRecycle, C:\Temp-Dateien\*.tmp
