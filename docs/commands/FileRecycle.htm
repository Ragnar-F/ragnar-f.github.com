FileRecycle

Wenn möglich, verschiebt dieser Befehl eine Datei oder ein Verzeichnis in den Papierkorb.

FileRecycle, Dateimuster

Parameter

Dateimuster

Der Name einer Datei oder eine musterbasierte Suche wie C:\Temp\*.tmp. Das Dateimuster befindet sich im A_WorkingDir-Verzeichnis, solange kein absoluter Pfad angegeben ist.

Um ein ganzes Verzeichnis zu entsorgen, gebt den Namen ohne nachfolgenden Backslash an.

ErrorLevel

[v1.1.04+] Dieser Befehl ist in der Lage, bei Misserfolg eine Ausnahme auszulösen. Für mehr Informationen, siehe Laufzeitfehler.

ErrorLevel wird bei Misserfolg auf 1 und bei Erfolg auf 0 gesetzt.

Bemerkungen

SHFileOperation wird verwendet, um die eigentliche Arbeit zu machen. Diese Funktion kann dauerhaft die Datei löschen, wenn sie nicht mehr im Papierkorb passt, weil sie zu groß ist. Seit v1.0.96 erscheint eine Warnung, bevor das geschieht.

Verwandte Befehle

FileRecycleEmpty, FileDelete, FileCopy, FileMove

Beispiel

FileRecycle, C:\Temp-Dateien\*.tmp
