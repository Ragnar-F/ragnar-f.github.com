FileGetTime

Ermittelt den Zeitstempel einer Datei oder eines Ordners.

FileGetTime, Ausgabevariable [, Dateiname, Zeitstempeltyp]

Parameter

Ausgabevariable Der Name der Variable, in der der ermittelte Zeitstempel im YYYYMMDDHH24MISS-Format gespeichert wird. Die Zeit ist die lokale Zeit des Betriebssystems, nicht UTC/GMT.
Dateiname Der Name der Datei oder des Ordners, die sich im A_WorkingDir-Verzeichnis befinden, solange kein absoluter Pfad angegeben ist. Falls weggelassen, wird die aktuelle Datei von der innersten dateilesenden Schleife verwendet.
Zeitstempeltyp Welcher Zeitstempel abgerufen werden soll:
M = Änderungsdatum (Standard, wenn Parameter nicht vorhanden ist)
C = Erstellungsdatum
A = Zugriffsdatum

ErrorLevel

[v1.1.04+] Dieser Befehl ist in der Lage, bei Misserfolg eine Ausnahme auszulösen. Für mehr Informationen, siehe Laufzeitfehler.

ErrorLevel wird bei Misserfolg auf 1 und bei Erfolg auf 0 gesetzt.

Die interne Variable A_LastError enthält den Rückgabewert der Systemfunktion GetLastError().

Bemerkungen

In Win9x ist das Zugriffsdatum einer Datei möglicherweise ungenauer als in NT-basierte Betriebssysteme.

Siehe YYYYMMDDHH24MISS, um mehr über Datum- und Zeitangaben zu erfahren.

Verwandte Befehle

FileSetTime, FormatTime, If Var is [not] Typ, FileGetAttrib, FileSetAttrib, FileGetSize, FileGetVersion, dateilesende Schleife, EnvAdd, EnvSub

Beispiel

FileGetTime, Ausgabevariable, C:\Eigene Dokumente\test.doc  ; Ermittelt standardmäßig das Änderungsdatum.
FileGetTime, Ausgabevariable, C:\Eigene Dokumente\test.doc, C  ; Ermittelt das Erstellungsdatum.
