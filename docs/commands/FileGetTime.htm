FileGetTime

Ermittelt den Zeitstempel einer Datei oder eines Ordners.

FileGetTime, Ausgabevariable [, Dateiname, Zeitstempeltyp]

Parameter

Ausgabevariable Der Name der Variable, in der der ermittelte Zeitstempel im YYYYMMDDHH24MISS-Format gespeichert wird. Die Zeit ist die lokale Zeit des Betriebssystems, nicht UTC/GMT.
Dateiname Der Name der Datei oder des Ordners, die sich im A_WorkingDir-Verzeichnis befinden, solange kein absoluter Pfad angegeben ist. Wenn nicht vorhanden, dann wird die aktuelle Datei in der innersten dateilesende Schleife verwendet.
Zeitstempeltyp Welcher Zeitstempel abgerufen werden soll:
M = Änderungsdatum (Standard, wenn Parameter nicht vorhanden ist)
C = Erstelldatum
A = Zugriffsdatum

ErrorLevel

[v1.1.04+] Dieser Befehl ist in der Lage, bei Misserfolg eine Ausnahme auszulösen. Für mehr Informationen, siehe Laufzeitfehler.

Bei Problemen wird ErrorLevel auf 1 gesetzt, ansonsten enthält ErrorLevel eine 0.

Die interne Variable A_LastError enthält den Rückgabewert der Systemfunktion GetLastError().

Bemerkungen

Das Zugriffsdatum einer Datei ist in Win9x nicht so genau wie in NT-basierte Betriebssysteme.

Siehe YYYYMMDDHH24MISS für eine Erklärung der Datums-und Zeitangaben.

Verwandte Befehle

FileSetTime, FormatTime, If var is [not] type, FileGetAttrib, FileSetAttrib, FileGetSize, FileGetVersion, dateilesende Schleife, EnvAdd, EnvSub

Beispiel

FileGetTime, Ausgabevariable, C:\Eigene Dokumente\test.doc  ; Ermittelt standardmäßig das Änderungsdatum.
FileGetTime, Ausgabevariable, C:\Eigene Dokumente\test.doc, C  ; Ermittelt das Erstelldatum.
