StrLower / StrUpper

Wandelt den String in Klein- oder Großbuchstaben um.

StrLower, Ausgabevariable, String [, T]
StrUpper, Ausgabevariable, String [, T]

Parameter

Ausgabevariable

Der Name der Variable, in der der neu umgewandelte String gespeichert wird.

String

Der String, der umgewandelt wird.

T

Wenn dieser Parameter den Buchstaben T enthält, dann wird der String in einem Titel umgewandelt. Zum Beispiel wird "VOM Winde VERWEHT" zu "Vom Winde Verweht".

Bemerkungen

Um festzustellen, ob ein Zeichen oder String komplett in Groß- oder Kleinbuchstaben ist, verwendet RegExMatch(). Zum Beispiel:

var := "abc"
if RegExMatch(var, "^[a-z]$")
    MsgBox var besteht nur aus Kleinbuchstaben.
if !RegExMatch(var, "[A-Z]")
    MsgBox var enthält keine Großbuchstaben.

Verwandte Befehle

InStr, SubStr, StrLen, StrReplace

Beispiel

StrUpper, String1, String1  ; Ausgabe kann das gleiche wie Eingabe sein.
StrLower, String2, String2
