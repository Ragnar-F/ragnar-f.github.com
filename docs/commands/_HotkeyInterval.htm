#HotkeyInterval

Zusammen mit #MaxHotkeysPerInterval kann die Frequenz der Hotkey-Aktivierungen angegeben werden, bei dessen Überschreitung eine Warnmeldung angezeigt wird.

#HotkeyInterval Millisekunden

Parameter

Millisekunden Die Länge des Intervalls in Millisekunden.

Bemerkungen

Ist diese Direktive im Script nicht angegeben, gilt standardmäßig 2000.

Für Details und Bemerkungen, siehe #MaxHotkeysPerInterval.

Verwandte Befehle

#MaxHotkeysPerInterval

Beispiel

#HotkeyInterval 2000  ; Standardwert (Millisekunden).
#MaxHotkeysPerInterval 200
