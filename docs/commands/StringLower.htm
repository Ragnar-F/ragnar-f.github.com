StringLower / StringUpper

Wandelt den String in Klein- oder Großbuchstaben um.

StringLower, Ausgabevariable, Eingabevariable [, T]
StringUpper, Ausgabevariable, Eingabevariable [, T]

Parameter

Ausgabevariable Name der Variable, in die der neu umgewandelte String gespeichert wird.
Eingabevariable Name der Variable, deren Inhalt eingelesen wird. Der Name sollte nicht mit Prozentzeichen umschlossen werden, sofern nicht die Absicht besteht, den Inhalt der Variable als Namen zu verwenden.
T Sobald dieser Parameter der Buchstabe T ist, wird der String in ein Titel umgewandelt. Zum Beispiel wird "VOM Winde VERWEHT" zu "Vom Winde Verweht".

Bemerkungen

Mit "If Var is [not] upper/lower" kann überprüft werden, ob ein Zeichen oder String vollständig groß- oder kleingeschrieben ist.

Bei diesen und allen anderen Befehlen kann die Ausgabevariable auch die Eingabevariable sein, und umgekehrt.

Verwandte Befehle

IfInString, StringGetPos, StringMid, StringTrimLeft, StringTrimRight, StringLeft, StringRight, StringLen, StringReplace

Beispiel

StringUpper, String1, String1  ; Ausgabe kann das gleiche wie Eingabe sein.
StringLower, String2, String2
