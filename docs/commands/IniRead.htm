IniRead

Liest einen Wert aus einer Standard-INI-Datei.

IniRead, Ausgabevariable, Dateiname [, Sektion, Schlüssel, Standardwert]

Parameter

Ausgabevariable

Der Name der Variable, in der der ermittelte Wert gespeichert wird. Falls der Wert nicht ermittelbar ist, enthält die Variable den Standardwert (siehe unten).

[AHK_L 57+]: Sobald der Schlüssel weggelassen wird, wird der Inhalt der angegebenen Sektion abgerufen, ohne jegliche Kommentare. Sobald auch die Sektion weggelassen wird, enthält die Ausgabevariable eine Liste mit Sektionsnamen, die durch LF-Zeichen (`n) getrennt sind.
Dateiname Der Name der Datei, die sich im A_WorkingDir-Verzeichnis befindet, solange kein absoluter Pfad angegeben ist. 
Sektion Name der Sektion aus der INI-Datei. Solche Sektionen werden mit eckigen Klammern dargestellt (die Klammern dürfen nicht in diesem Parameter stehen).
Schlüssel Name des Schlüssels aus der INI-Datei.
Standardwert

Dieser Wert wird in die Ausgabevariable gespeichert, wenn der angeforderte Schlüssel nicht gefunden wird. Ist dieser Parameter nicht vorhanden, wird ERROR als Standardwert verwendet. Als leeren Wert kann %A_Space% benutzt werden.

[AHK_L 57+]: Dieser Parameter entfällt, sobald der Schlüssel weggelassen wird.

ErrorLevel

ErrorLevel wird nicht durch diesen Befehl beeinflusst. Bei einem Problem wird die Ausgabevariable auf den oben genannten Standardwert gesetzt.

Bemerkungen

Das Betriebssystem entfernt automatisch führende oder nachfolgende Leer-/Tabulatorzeichen in der Ausgabevariable.

Eine typische INI-Datei sieht wie folgt aus:

[Sektionsname]
Schlüssel=Wert

Unicode: IniRead und IniWrite sind auf die externen Funktionen GetPrivateProfileString und WritePrivateProfileString angewiesen, um Werte zu lesen und zu schreiben. Diese Funktionen unterstützen Unicode nur in UTF-16-Dateien; alle anderen Dateien verwenden den Standard-ANSI-Zeichensatz des Systems.

Verwandte Befehle

IniDelete, IniWrite, RegRead, dateilesende Schleife, FileRead

Beispiel

IniRead, Ausgabevariable, C:\Temp\EigeneDatei.ini, Sektion2, Schlüssel
MsgBox, Der Wert ist %Ausgabevariable%.
