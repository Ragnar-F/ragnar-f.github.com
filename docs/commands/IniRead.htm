IniRead

Liest einen Wert von einer INI-Datei im Standardformat.

IniRead, Ausgabevariable, Dateiname [, Sektion, Schlüssel, Standardwert]

Parameter

Ausgabevariable

Der Name der Variable, in der der ermittelte Wert gespeichert wird. Wenn der Wert nicht ermittelt werden kann, dann enthält die Variable den Standardwert (siehe unten).

Wenn der Schlüssel weggelassen wird, dann wird der Inhalt der angegebenen Sektion abgerufen, ohne jegliche Kommentare.

Sobald der Schlüssel als auch die Sektion weggelassen werden, enthält die Ausgabevariable eine Liste mit Sektionsnamen, die durch LF-Zeichen (`n) getrennt sind.

Dateiname

Der Name der Datei, die sich im A_WorkingDir-Verzeichnis befindet, solange kein absoluter Pfad angegeben ist. 

Sektion

Name der Sektion in der INI-Datei. Solche Sektionen werden mit eckigen Klammern dargestellt (die Klammern dürfen nicht in diesem Parameter stehen).

Schlüssel

Name des Schlüssels in der INI-Datei.

Standardwert

Dieser Wert wird in die Ausgabevariable gespeichert, wenn der angeforderte Schlüssel nicht gefunden wird. Wenn weggelassen, dann wird ein leerer String als Standardwert verwendet.

Dieser Parameter wird nicht verwendet, wenn der Schlüssel weggelassen wird.

ErrorLevel

[v1.1.04+] Dieser Befehl ist in der Lage, bei Misserfolg eine Ausnahme auszulösen. Für mehr Informationen, siehe Laufzeitfehler.

ErrorLevel wird bei Misserfolg auf 1 und bei Erfolg auf 0 gesetzt.

Bemerkungen

Das Betriebssystem entfernt automatisch führende oder nachfolgende Leerzeichen/Tabulatoren in der Ausgabevariable.

Eine typische INI-Datei sieht wie folgt aus:

[Sektionsname]
Schlüssel=Wert

Unicode: IniRead und IniWrite sind auf die externen Funktionen GetPrivateProfileString und WritePrivateProfileString angewiesen, um Werte zu lesen und zu schreiben. Diese Funktionen unterstützen Unicode nur in UTF-16-Dateien; alle anderen Dateien verwenden den Standard-ANSI-Zeichensatz des Systems.

Verwandte Befehle

IniDelete, IniWrite, RegRead, dateilesende Schleife, FileRead

Beispiel

IniRead, Ausgabevariable, C:\Temp\EigeneDatei.ini, Sektion2, Schlüssel
MsgBox, Der Wert ist %Ausgabevariable%.
