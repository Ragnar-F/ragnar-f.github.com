DriveGet

Ermittelt unterschiedliche Informationen über die Laufwerke eines Computers.

DriveGet, AusgabeVar, Befehl [, Wert]

Parameter

AusgabeVar

Name der Variable, in der der Rückgabewert von Befehl gespeichert werden soll.

Befehl, Wert
Siehe untere Liste.

Befehl, Wert

Die Parameter Befehl und Wert sind voneinander abhängig und können wie folgt verwendet werden. Bei Misserfolg wird AusgabeVar leer gemacht und ErrorLevel auf 1 gesetzt.

List [, Typ]: Speichert in AusgabeVar ein Zeichenfolge bestehend aus Laufwerksbuchstaben. Zum Beispiel: ACDEZ. Wenn Typ fehlt, werden alle Laufwerksarten abgerufen. Ansonsten kann bei Typ eines der folgenden Wörter angegeben werden, um nur einen bestimmten Laufwerkstypen abzurufen: CDROM, REMOVABLE, FIXED, NETWORK, RAMDISK, UNKNOWN.

Capacity (oder Cap), Pfad: Ermittelt die Gesamtkapazität von Pfad (z. B. C:\) in Megabyte. Mit DriveSpaceFree kann der freie Speicherplatz abgerufen werden.

Filesystem (oder FS), Laufwerk: Ermittelt den Typ des Dateisystems von Laufwerk. Laufwerk ist der Laufwerksbuchstabe, gefolgt von einem Doppelpunkt und einem umgekehrten Schrägstrich, oder einem Netzwerkpfad im UNC-Format wie \server1\share1. AusgabeVar wird auf eines der folgenden Wörter gesetzt: FAT, FAT32, NTFS, CDFS (typisch für CDs), UDF (typisch für DVDs). AusgabeVar wird leer gemacht und ErrorLevel auf 1 gesetzt, wenn das Laufwerk kein formatiertes Medium enthält.

Label, Laufwerk: Ermittelt die Datenträgerbezeichnung von Laufwerk. Laufwerk ist der Laufwerksbuchstabe, gefolgt von einem Doppelpunkt und einem umgekehrten Schrägstrich, oder einem Netzwerkpfad im UNC-Format wie \server1\share1. Die Bezeichnung kann wie folgt geändert werden: Drive, Label, C:, MeineBezeichnung.

Serial, Laufwerk: Ermittelt die Datenträgerseriennummer von Laufwerk im dezimalen Integer-Format. Laufwerk ist der Laufwerksbuchstabe, gefolgt von einem Doppelpunkt und einem umgekehrten Schrägstrich, oder einem Netzwerkpfad im UNC-Format wie \server1\share1. Mit SetFormat könnte man diesen Wert in ein hexadezimales Format umwandeln.

Type, Pfad: Ermittelt den Laufwerkstypen von Pfad, und setzt dabei AusgabeVar auf eines der folgenden Wörter: Unknown, Removable, Fixed, Network, CDROM, RAMDisk.

Status, Pfad: Ermittelt den Status von Pfad, und setzt dabei AusgabeVar auf eines der folgenden Wörter: Unknown (könnte auf unformatiert/RAW hindeuten), Ready, NotReady (typisch für Wechseldatenträger ohne Inhalt), Invalid (Pfad existiert nicht oder ist ein Netzlaufwerk, das gerade nicht zugänglich ist, usw.)

StatusCD [, Laufwerk]: Ermittelt den Medienstatus eines CD- oder DVD-Laufwerks. Laufwerk ist der Laufwerksbuchstabe, gefolgt von einem Doppelpunkt (wenn Laufwerk fehlt, wird das Standard-CD/DVD-Laufwerk verwendet). AusgabeVar wird leer gemacht, wenn der Status nicht ermittelt werden kann. Ansonsten wird es auf einen der folgenden Strings gesetzt:

not ready Das Laufwerk ist gerade beschäftigt, weil es vermutlich an einer Schreiboperation beteiligt ist. Bekannte Einschränkung: "not ready" tritt auch auf, wenn das Laufwerk keine CD, sondern eine DVD enthält.
open Das Laufwerk enthält keine Disc, oder ist offen.
playing Das Laufwerk spielt gerade eine Disc ab.
paused Das zuvor abgespielte Audio oder Video ist nun pausiert.
seeking Das Laufwerk ist am Suchen.
stopped Das Laufwerk enthält eine CD, auf der gerade nicht zugegriffen werden kann.

Es kann sein, dass dieser Befehl nicht funktionieren wird, wenn das Laufwerk über das Netzwerk zugreifbar ist, oder kein CD/DVD-Laufwerk ist; schlägt es in solchen oder sonstigen Fällen fehl, wird AusgabeVar leer gemacht und ErrorLevel auf 1 gesetzt.

Wenn das Laufwerk gerade geschlossen wurde, kann zu einer Verzögerung kommen, bevor der Befehl endet.

Mit dem Drive-Befehl kann ein Laufwerk ein- oder ausgefahren werden.

ErrorLevel

[v1.1.04+] Dieser Befehl ist in der Lage, bei Misserfolg eine Ausnahme auszulösen. Für mehr Informationen, siehe Laufzeitfehler.

ErrorLevel wird bei Misserfolg auf 1 und bei Erfolg auf 0 gesetzt.
