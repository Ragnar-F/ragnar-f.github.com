FileDelete

Löscht eine oder mehrere Dateien.

FileDelete, Dateimuster

Parameter

Dateimuster

Der Name einer Datei oder eine musterbasierte Suche wie C:\Temp\*.tmp. Das Dateimuster befindet sich im A_WorkingDir-Verzeichnis, solange kein absoluter Pfad angegeben ist.

Um einen ganzen Ordner zu löschen, einschließlich seine Unterordner und Dateien, verwendet FileRemoveDir.

ErrorLevel

[v1.1.04+] Dieser Befehl ist in der Lage, bei Misserfolg eine Ausnahme auszulösen. Für mehr Informationen, siehe Laufzeitfehler.

Bei Problemen enthält ErrorLevel die Anzahl an Dateien, die nicht gelöscht werden konnten, ansonsten wird ErrorLevel auf 0 gesetzt. Das Löschen einer musterbasierten Suche wie *.tmp wird als Erfolg angesehen, selbst wenn keine Dateien gefunden werden; daher wird ErrorLevel auf 0 gesetzt.

Falls Dateien gefunden wurden, wird A_LastError auf 0 gesetzt, oder auf den aktuellsten Rückgabewert der Systemfunktion GetLastError(). Ansonsten enthält A_LastError einen Fehlercode, der darauf hinweist, warum keine Dateien gefunden wurden.

Bemerkungen

Eine schreibgeschützte Datei kann nur gelöscht werden, wenn vorher deren Schreibschutzattribut entfernt wird. Zum Beispiel: FileSetAttrib, -R, C:\Eigene Datei.txt

Verwandte Befehle

FileRecycle, FileRemoveDir, FileCopy, FileMove

Beispiel

FileDelete, C:\Temp-Dateien\*.tmp
