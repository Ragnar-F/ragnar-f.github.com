FileDelete

Löscht beliebig viele Dateien.

FileDelete, Dateimuster

Parameter

Dateimuster

Der Name einer Datei oder eine musterbasierte Suche wie C:\Temp\*.tmp. Das Dateimuster befindet sich im A_WorkingDir-Verzeichnis, solange kein absoluter Pfad angegeben ist.

Mit FileRemoveDir kann ein ganzer Ordner mit all seinen Unterordnern und Dateien entfernt werden.

ErrorLevel

[v1.1.04+] Dieser Befehl ist in der Lage, bei Misserfolg eine Ausnahme auszulösen. Für mehr Informationen, siehe Laufzeitfehler.

Bei Problemen enthält ErrorLevel die Anzahl an Dateien, die nicht gelöscht werden konnten, ansonsten wird ErrorLevel auf 0 gesetzt. Das Löschen einer musterbasierten Suche wie *.tmp wird als erfolgreich angesehen, selbst wenn keine Dateien gefunden werden; daher wird ErrorLevel auf 0 gesetzt.

Wenn Dateien gefunden wurden, enthält A_LastError eine Null oder den Rückgabewert der Systemfunktion GetLastError()-Funktion unmittelbar nach dem letzten Fehler. Ansonsten enthält A_LastError einen Fehlercode, der darauf hinweist, warum keine Dateien gefunden wurden.

Bemerkungen

Um eine schreibgeschützte Datei zu löschen, muss zuerst der Schreibschutz entfernt werden. Zum Beispiel: FileSetAttrib, -R, C:\Eigene Datei.txt

Verwandte Befehle

FileRecycle, FileRemoveDir, FileCopy, FileMove

Beispiel

FileDelete, C:\Temp-Dateien\*.tmp
