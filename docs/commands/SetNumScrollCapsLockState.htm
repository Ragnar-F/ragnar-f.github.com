SetCapsLockState/SetNumLockState/SetScrollLockState

Setzt den Status der FESTSTELL/NUM/ROLLEN-Taste. Kann die Taste auch dazu zwingen, an oder aus zubleiben.

SetCapsLockState [, Status]
SetNumLockState [, Status]
SetScrollLockState [, Status]

Parameter

Status

Ist dieser Parameter nicht vorhanden, wird das AlwaysOn/Off-Attribut von der Taste entfernt (falls vorhanden). Ansonsten kann eins der folgenden Wörter angegeben werden:

On: Aktiviert die Taste und entfernt deren AlwaysOn/Off-Attribut (falls vorhanden).

Off: Deaktiviert die Taste und entfernt deren AlwaysOn/Off-Attribut (falls vorhanden).

AlwaysOn: Zwingt die Taste dazu, dauerhaft anzubleiben.

AlwaysOff: Zwingt die Taste dazu, dauerhaft auszubleiben.

Bemerkungen

Es ist auch möglich, per Send-Befehl eine Taste auf den entgegengesetzten Zustand umzuschalten: Send {Capslock}.

Die Aufrechterhaltung einer Taste mit AlwaysOn oder AlwaysOff erfordert den Tastatur-Hook, der in solchen Fällen automatisch installiert wird.
