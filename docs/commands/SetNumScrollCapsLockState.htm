SetCapsLockState/SetNumLockState/SetScrollLockState

Setzt den Status der FESTSTELL/NUM/ROLLEN-Taste. Kann die Taste auch dazu zwingen, an oder aus zubleiben.

SetCapsLockState [, Status]
SetNumLockState [, Status]
SetScrollLockState [, Status]

Parameter

Status

Ist dieser Parameter nicht vorhanden, wird das AlwaysOn/Off-Attribut von der Taste entfernt (falls vorhanden). Ansonsten kann eins der folgenden Wörter angeben werden:

On: Aktiviert die Taste und entfernt deren AlwaysOn/Off-Attribut (falls vorhanden).

Off: Deaktiviert die Taste und entfernt deren AlwaysOn/Off-Attribut (falls vorhanden).

AlwaysOn: Zwingt die Taste dazu, dauerhaft anzubleiben (hat keine Auswirkung auf Windows 9x).

AlwaysOff: Zwingt die Taste dazu, dauerhaft auszubleiben (hat keine Auswirkung auf Windows 9x).

Bemerkungen

In Windows Me/98/95 sind diese Befehle zurzeit augrund von Einschränkungen nicht hundertprozentig zuverlässig. Alternativ ist es möglich, per Send-Befehl eine Taste in Win9x zuverlässig auf den entgegengesetzten Zustand umzuschalten: Send {Capslock}.

Die Aufrechterhaltung einer Taste mit AlwaysOn oder AlwaysOff erfordert den Tastatur-Hook, der in solchen Fällen automatisch installiert wird.

Verwandte Befehle

SetStoreCapslockMode, GetKeyState

Beispiele

SetNumlockState, on
SetScrollLockState, AlwaysOff
