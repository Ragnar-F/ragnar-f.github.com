Random

Erzeugt eine Pseudo-Zufallszahl.

Random, Ausgabevariable [, Min, Max]
Random, , NewSeed

Parameter

Ausgabevariable Der Name der Variable, in der das Ergebnis gespeichert wird. Das Format der gespeicherten Fließkommazahlen wird durch SetFormat bestimmt.
Min Die kleinste Zahl, die erzeugt werden kann, welche negativ, eine Fließkommazahl oder ein Ausdruck sein kann. Wenn weggelassen, dann ist die kleinste Zahl 0. Der niedrigste zulässige Wert für Integer ist -2147483648, Fließkommazahlen hingegen haben keine Beschränkungen.
Max Die größte Zahl, die erzeugt werden kann, welche negativ, eine Fließkommazahl oder ein Ausdruck sein kann. Wenn weggelassen, dann ist die größte Zahl 2147483647 (der größte zulässige Integer-Wert, Fließkommazahlen hingegen haben keine Beschränkungen).
NewSeed

Dieser Modus erneuert den Startpunkt im Zufallszahlengenerator mit NewSeed (kann ein Ausdruck sein). Das betrifft alle danach generierten Zufallszahlen. NewSeed ist ein Integer-Wert zwischen 0 und 4294967295 (0xFFFFFFFF). Dieses Verfahren kann die Qualität/Sicherheit von erzeugten Zufallszahlen verbessern, besonders wenn NewSeed eine wirkliche Zufallszahl ist, anstatt eine mit geringerer Qualität wie eine Pseudo-Zufallszahl. Im Allgemeinen muss dieses Verfahren nicht mehr als einmal durchgeführt werden.

Wenn das Neusetzen des Startpunkts nie vom Script durchgeführt wird, dann beginnt er bei niedrigwertigen 32-Bits vom 64-Bit-Wert, dass die Anzahl der 100-Nanosekunden-Intervalle seit 1. Januar 1601 ist. Dieser Wert wandert alle ~7.2 Minuten von 0 bis 4294967295.

Bemerkungen

Dieser Befehl erzeugt eine Pseudo-Zufallszahl, die eine wahre Zufallszahl simuliert, aber in Wirklichkeit eine Zahl ist, die von einer komplizierten Formel abgeleitet wird, um das Bestimmen/Erraten der nächsten Zahl zu erschweren.

Alle Zahlen im angegebenen Bereich haben ungefähr die gleiche Wahrscheinlichkeit, erzeugt zu werden (siehe aber "bekannte Einschränkungen" weiter unten).

Wenn Min oder Max einen Dezimalpunkt enthält, dann ist das Endergebnis eine Fließkommazahl, abhängig vom eingestellten Format (SetFormat). Andernfalls wird das Ergebnis ein Integer sein.

Bekannte Einschränkungen für Fließkommazahlen:

  • Nur etwa 4294967296 verschiedene Zahlen können für jeden bestimmten Bereich erzeugt werden, alle anderen Zahlen im Bereich werden somit nie erzeugt.
  • Gelegentlich kann ein Ergebnis etwas größer als der angegebene Max-Wert sein (zum Teil durch die Ungenauigkeit in Fließkommazahlen).

Verwandte Befehle

SetFormat

Beispiele

Random, rand, 1, 10
Random, rand, 0.0, 1.0

Kommentare vom Original

This function uses the Mersenne Twister random number generator, MT19937, written by Takuji Nishimura and Makoto Matsumoto, Shawn Cokus, Matthe Bellew and Isaku Wada.

The Mersenne Twister is an algorithm for generating random numbers. It was designed with consideration of the flaws in various other generators. The period, 219937-1, and the order of equidistribution, 623 dimensions, are far greater. The generator is also fast; it avoids multiplication and division, and it benefits from caches and pipelines.
