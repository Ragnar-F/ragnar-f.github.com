SetBatchLines

Bestimmt, wie schnell ein Script ausgeführt wird (betrifft die CPU-Auslastung).

SetBatchLines, 20ms
SetBatchLines, Zeilenanzahl

Parameter

20ms (20ms ist nur ein Beispiel) Wenn der Wert mit ms endet, wird angegeben, wie oft ein Script ruhen soll (jede Ruhephase dauert 10 ms). Im folgenden Beispiel ruht das Script jeweils für 10ms, nachdem es 20ms lief: SetBatchLines, 20ms.
Zeilenanzahl Die Anzahl der Zeilen im Script, die ausgeführt werden, bevor es 10ms ruht. Der Wert hat eine maximale Größe von 9223372036854775807. Außerdem schließen sich dieser Modus und der 20ms-Modus vom vorherigem Abschnitt gegenseitig aus, d. h. es kann jeweils nur ein Modus zur selben Zeit aktiv sein.

Bemerkungen

Mit SetBatchLines -1 kann festgelegt werden, dass das Script nie ruht (das heißt, dass das Script mit maximaler Geschwindigkeit läuft).

Die Standardeinstellung ist 10ms, außer in den Versionen vor v1.0.16, die stattdessen 10 (Zeilen) verwenden.

Die "ms"-Methode ist für Scripts empfehlenswert, denen Geschwindigkeit und Zusammenarbeit wichtig ist. Zum Beispiel verhindert die Angabe von 10ms bei den meisten Betriebssystemen, dass die CPU-Auslastung des Scripts größer als 50% ist. Dadurch können die Scripts schnell ausgeführt werden, während weiterhin ein hohes Maß an Zusammenarbeit mit CPU-empfindlichen Anwendungen wie Spiele oder Video-Aufnahme/Wiedergabe gewährleistet wird.

Die interne Variable A_BatchLines enthält die aktuelle Einstellung.

Zudem kann die Geschwindigkeit eines Scripts je nach Art von folgenden Befehlen beeinflusst werden: SetWinDelay, SetControlDelay, SendMode, SetKeyDelay, SetMouseDelay und SetDefaultMouseSpeed.

Jeder neu gestartete Thread (z. B. ein Hotkey, ein benutzerdefinierter Menüpunkt oder eine zeitlich festgelegte Subroutine) beginnt mit der Standardeinstellung des Befehls. Diese Standardeinstellung kann geändert werden, indem dieser Befehl im automatischen Ausführungsbereich (am Anfang des Scripts) verwendet wird.

Verwandte Befehle

SetWinDelay, SetControlDelay, SendMode, SetKeyDelay, SetMouseDelay, SetDefaultMouseSpeed, Critical

Beispiel

SetBatchLines, 10ms
SetBatchLines, 1000
