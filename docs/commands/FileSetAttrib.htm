FileSetAttrib

Ändert die Attribute von einer oder mehreren Dateien (auch Ordner). Platzhalter werden unterstützt.

FileSetAttrib, Attribute [, Dateimuster, OrdnerEinbeziehen?, Rekursiv?]

Parameter

Attribute

Die Attribute, die geändert werden sollen (siehe Bemerkungen).

Dateimuster
Name einer Datei, eines Ordners oder einer musterbasierten Suche wie C:\Temp\*.tmp. Dateimuster befindet sich im A_WorkingDir-Verzeichnis, solange kein absoluter Pfad angegeben ist.

Ist dieser Parameter nicht vorhanden, wird die aktuelle Datei von der innersten Dateimuster-Schleife verwendet.

OrdnerEinbeziehen?

0 (Standard) Ordner werden nicht mit einbezogen (nur Dateien).
1 Alle Dateien und Ordner, die mit der musterbasierten Suche übereinstimmen, werden mit einbezogen.
2 Nur Ordner werden mit einbezogen (keine Dateien).

Hinweis: Falls Dateimuster ein einzelner Ordner anstatt eine musterbasierte Suche ist, wird er immer mit einbezogen, unabhängig von dieser Einstellung.

Dieser Parameter kann ein Ausdruck sein.

Rekursiv?

0 (Standard) Unterordner werden nicht rekursiv behandelt.
1 Unterordner werden rekursiv behandelt, so dass die darin enthaltenen Dateien und Unterordner mit einbezogen werden, wenn sie mit Dateimuster übereinstimmen. Alle Unterordner werden rekursiv behandelt, nicht nur solche, die mit Dateimuster übereinstimmen. Allerdings werden Dateien und Ordner, deren kompletter Pfadname länger als 259 Zeichen ist, übersprungen, als würde sie nicht existieren. Solche Dateien sind normalerweise selten, weil das Betriebssystem nicht erlaubt, dass diese erstellt werden können.

Dieser Parameter kann ein Ausdruck sein.

ErrorLevel

[v1.1.04+] Dieser Befehl ist in der Lage, bei Misserfolg eine Ausnahme auszulösen. Für mehr Informationen, siehe Laufzeitfehler.

Bei Misserfolg wird ErrorLevel auf die Anzahl der Dateien gesetzt, die nicht geändert werden konnten, ansonsten auf 0.

Falls Dateien gefunden wurden, wird A_LastError auf 0 gesetzt, oder auf den aktuellsten Rückgabewert der Systemfunktion GetLastError(). Ansonsten enthält A_LastError einen Fehlercode, der darauf hinweist, warum keine Dateien gefunden wurden.

Bemerkungen

Der Attribute-Parameter besteht aus mehreren Operatoren und Attributsbuchstaben.

Operatoren:

+ Aktiviert das Attribut
- Deaktiviert das Attribut
^ Schaltet das Attribut um (auf den entgegengesetzten Wert)


Attributsbuchstaben:
R = READONLY (schreibgeschützt)
A = ARCHIVE (archiviert)
S = SYSTEM (Systemdatei)
H = HIDDEN (versteckt)
N = NORMAL (nur gültig, wenn dieser ohne den anderen Attributen verwendet wird)
O = OFFLINE
T = TEMPORARY (temporär)

Hinweis: Zurzeit ist es mit diesem Befehl nicht möglich, den Komprimierungsstatus einer Datei zu ändern.

Siehe auch

FileGetAttrib, FileGetTime, FileSetTime, FileGetSize, FileGetVersion, Dateimuster-Schleife

Beispiele

FileSetAttrib, +RH, C:\MeineDateien\*.*, 1  ; +RH ist identisch mit +R+H
FileSetAttrib, ^H, C:\MeineDateien  ; Schaltet das HIDDEN-Attribut des Ordners um.
FileSetAttrib, -R+A, C:\Neue Textdatei.txt
FileSetAttrib, +A, C:\*.ini, , 1  ; Alle INI-Dateien auf Laufwerk C: werden rekursiv behandelt.
