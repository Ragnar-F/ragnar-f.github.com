FileSetAttrib

Ändert die Attribute von einer oder mehreren Dateien (auch Ordner). Platzhalter werden unterstützt.

FileSetAttrib, Attribute [, Dateimuster, OrdnerEinbeziehen?, Rekursiv?]

Parameter

Attribute Die Attribute, die geändert werden (siehe Bemerkungen).
Dateimuster Der Name einer Datei, eines Ordners oder einer musterbasierten Suche wie C:\Temp\*.tmp. Das Dateimuster befindet sich im A_WorkingDir-Verzeichnis, solange kein absoluter Pfad angegeben ist.

Falls weggelassen, wird die aktuelle Datei von der innersten dateilesenden Schleife verwendet.
OrdnerEinbeziehen?

0 (Standard) Ordner werden nicht mit einbezogen (nur Dateien).
1 Alle Dateien und Ordner, die der musterbasierten Suche entsprechen, werden mit einbezogen.
2 Nur Ordner werden mit einbezogen (keine Dateien).

Hinweis: Wenn das Dateimuster ein einzelner Ordner anstelle einer musterbasierten Suche ist, wird dieser immer mit einbezogen, unabhängig dieser Einstellung.

Dieser Parameter kann ein Ausdruck sein.
Rekursiv?

0 (Standard) Unterordner werden nicht rekursiv behandelt.
1 Unterordner werden rekursiv behandelt, damit die enthaltenen Dateien und Unterordner mit einbezogen werden, wenn sie mit dem Dateimuster übereinstimmen. Alle Unterordner werden rekursiv behandelt, nicht nur solche, die mit dem Dateimuster übereinstimmen. Allerdings werden Dateien und Ordner mit einem kompletten Pfad länger als 259 Zeichen ignoriert, als würde sie nicht existieren. Solche Dateien sind selten, weil normalerweise das Betriebssystem das Erstellen solcher Dateien verbietet.

Dieser Parameter kann ein Ausdruck sein.

ErrorLevel

[v1.1.04+] Dieser Befehl ist in der Lage, bei Misserfolg eine Ausnahme auszulösen. Für mehr Informationen, siehe Laufzeitfehler.

Bei Problemen enthält ErrorLevel die Anzahl an Dateien, die nicht geändert werden konnten, ansonsten wird ErrorLevel auf 0 gesetzt.

Falls Dateien gefunden wurden, wird A_LastError auf 0 gesetzt, oder auf den aktuellsten Rückgabewert der Systemfunktion GetLastError(). Ansonsten enthält A_LastError einen Fehlercode, der darauf hinweist, warum keine Dateien gefunden wurden.

Bemerkungen

Der Attribute-Parameter besteht aus mehreren Operatoren und Attributsbuchstaben.

Operatoren:

+ Aktiviert das Attribut
- Deaktiviert das Attribut
^ Schaltet das Attribut um (auf den entgegengesetzten Wert)

Attributsbuchstaben:
R = READONLY
A = ARCHIVE
S = SYSTEM
H = HIDDEN
N = NORMAL (nur gültig, wenn diese Attribut alleine steht)
O = OFFLINE
T = TEMPORARY

Hinweis: Derzeit kann der Komprimierungsstatus einer Datei nicht mit diesem Befehl geändert werden.

Verwandte Befehle

FileGetAttrib, FileGetTime, FileSetTime, FileGetSize, FileGetVersion, dateilesende Schleife

Beispiele

FileSetAttrib, +RH, C:\EigeneDateien\*.*, 1  ; +RH ist identisch mit +R+H
FileSetAttrib, ^H, C:\EigeneDateien  ; Schaltet das HIDDEN-Attribut des Ordners um.
FileSetAttrib, -R+A, C:\Neue Textdatei.txt
FileSetAttrib, +A, C:\*.ini, , 1  ; Alle INI-Dateien auf Laufwerk C: werden rekursiv behandelt.
