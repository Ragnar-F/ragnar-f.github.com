#MaxMem

Setzt die maximale Kapazität jeder Variable auf die angegebene Anzahl an Megabytes.

#MaxMem Megabytes

Parameter

Megabytes Die Anzahl an erlaubten Megabytes für jede Variable. Ein Wert größer als 4095 bleibt auf 4095. Ein Wert kleiner als 1 bleibt auf 1.

Bemerkungen

Wenn diese Direktive nicht angegeben ist, dann wird standardmäßig 64 verwendet.

Die Kapazitätsbeschränkung jeder Variable verhindert, dass ein fehlerhaftes Script den gesamten Arbeitsspeicher verbraucht. Das Anheben oder Absenken der Beschränkung hat weder Auswirkung auf die Performance eines Scripts, noch wird dadurch der aktuelle Speicherverbrauch des Scripts verändert (außer im Fall von WinGetText und ControlGetText, die in der Lage sind, mehr Text abzurufen, wenn #MaxMem erhöht wird).

Diese Einstellung ist global, daher muss diese Direktive irgendwo im Script nur einmal angegeben werden, um das Verhalten des ganzen Scripts zu beeinflussen.

#MaxMem limitiert nur die automatische Erweiterung der Variable. VarSetCapacity wird aber dadurch nicht beeinflusst.

Verwandte Befehle

VarSetCapacity, Variablen, Sort, WinGetText, ControlGetText, #MaxThreads

Beispiel

#MaxMem 256  ; Erlaubt 256 MB pro Variable.
