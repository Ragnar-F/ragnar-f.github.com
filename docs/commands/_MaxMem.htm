#MaxMem

Setzt die maximale Kapazität jeder Variable auf die angegebene Anzahl an Megabytes.

#MaxMem Megabytes

Parameter

Megabytes

Die Anzahl an Megabytes, die jede Variable maximal haben darf. Ein Wert größer als 4095 wird automatisch auf 4095 gesetzt. Ein Wert kleiner als 1 wird automatisch auf 1 gesetzt.

Bemerkungen

Ist diese Direktive im Script nicht angegeben, gilt standardmäßig 64.

Das Begrenzen der Kapazität einer Variable soll verhindern, dass ein fehlerhaftes Script den gesamten Arbeitsspeicher verbraucht. Das Erhöhen oder Verringern der Limitierung hat weder Auswirkung auf die Performance eines Scripts, noch ändert es, wieviel Speicher das Script aktuell verwendet (außer bei WinGetText und ControlGetText, die mehr Text abrufen können, wenn #MaxMem erhöht wird).

Diese Einstellung ist global - das heißt, dass die Direktive nur einmal angegeben werden muss (egal wo), um das Verhalten des ganzen Scripts zu beeinflussen.

#MaxMem limitiert nur die automatische Erweiterung, die eine Variable für sich selbst durchführt. Diese Direktive hat keinen Einfluss auf VarSetCapacity.

Beispiel
