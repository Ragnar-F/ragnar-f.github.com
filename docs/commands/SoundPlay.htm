SoundPlay

Abspielen einer Sound-, Video- oder andere unterstützte Dateien.

SoundPlay, Dateiname [, Warten]

Parameter

Dateiname

Der Name der abspielenden Datei, die sich im A_WorkingDir-Verzeichnis befindet, solange kein absoluter Pfad angegeben ist.

Um Standard-Systemsounds zu erzeugen, verwendet ein Sternchen vor einer Nummer, wie unten angezeigt. Hinweis: der Warten-Parameter funktioniert nicht in diesem Modus.
*-1: Einfacher Piepton. Wenn die Soundkarte nicht verfügbar ist, dann wird der Ton mithilfe des PC-Lautsprechers erzeugt.
*16: Hand (Stopp/Fehler)
*32: Frage
*48: Ausruf
*64: Sternchen (Info)
Warten

Wenn nicht vorhanden, dann springt der aktuelle Thread des Scripts auf dem nachfolgenden Befehl, während die Datei abgespielt wird. Um dies zu vermeiden, verwendet eine 1 oder das Wort WAIT, damit der aktuelle Thread wartet, bis die Datei fertig abgespielt wurde. Selbst beim Warten können neue Threads mittels Hotkey, benutzerdefinierte Menüeinträge oder Timer gestartet werden.

Bekannte Einschränkung: Wenn der Warten-Parameter weggelassen wird, dann sieht das Betriebssystem die abspielende Datei als "verwendet", bis das Script beendet oder eine andere Datei abgespielt wird (auch eine nicht vorhandene Datei).

ErrorLevel

[v1.1.04+] Dieser Befehl ist in der Lage, bei Misserfolg eine Ausnahme auszulösen. Für mehr Informationen, siehe Laufzeitfehler.

ErrorLevel wird bei Misserfolg auf 1 und bei Erfolg auf 0 gesetzt.

Bemerkungen

Alle Windows-Betriebssysteme sollten in der Lage sein, WAV-Dateien abzuspielen. Allerdings können möglicherweise andere Dateien (MP3, AVI usw.) nicht abgespielt werden, wenn die richtigen Codecs oder Funktionen auf dem Betriebssystem fehlen.

Wenn das aktuelle Script eine Datei abspielt, während eine andere Datei abgespielt wird, dann wird die erste Datei gestoppt, damit die zweite Datei abgespielt werden kann. Auf einigen Systemen können bestimmte Dateitypen auch gestoppt werden, selbst wenn ein völlig separates Script eine neue Datei abspielt.

Um eine Datei zu stoppen, die gerade abgespielt wird, verwendet SoundPlay mit einem nicht vorhandenen Dateinamen, zum Beispiel: SoundPlay, Nonexistent.avi

Wenn das Script beendet wird, dann werden alle aktuell abgespielten Dateien gestoppt.

Verwandte Befehle

SoundBeep, SoundGet, SoundSet, SoundGetWaveVolume, SoundSetWaveVolume, MsgBox, Threads

Beispiel

SoundPlay, %A_WinDir%\Media\ding.wav
SoundPlay *-1  ; Einfacher Piepton. Wenn die Soundkarte nicht verfügbar ist, dann wird der Ton mithilfe des PC-Lautsprechers erzeugt.
