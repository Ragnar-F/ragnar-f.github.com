ObjAddRef() / ObjRelease() [AHK_L 53+]

Erhöht oder verringert die Referenzzählung eines Objekts.

ObjAddRef(Ptr)
ObjRelease(Ptr)

Parameter

Ptr Ein nicht verwalteter Objekt-Pointer oder COM-Interface-Pointer.

Rückgabewert

Diese Funktionen geben die neue Referenzzählung zurück. Dieser Wert sollte nur zum Debuggen verwendet werden.

Verwandte Befehle

Die folgenden Artikel beziehen sich auf die Referenzzählung von COM. Allerdings decken sie einige wichtige Konzepte und Regeln ab, die im Allgemeinen auch für AutoHotkey-Objekte gelten: IUnknown::AddRef, IUnknown::Release, Regeln bei der Referenzzählung.

Beispiele

Siehe ComObjConnect.

obj := Object()

; Die folgenden zwei Zeilen sind gleichwertig:
ptr1 := Object(obj)
ptr2 := ObjectToPointer(obj)

ObjectToPointer(obj) {
    If !IsObject(obj)
        Return ""
    ptr := &obj

    ObjAddRef(ptr)
    Return ptr
}

; Jeder Pointer, der über Object() oder ObjectToPointer() ermittelt wird, muss manuell freigegeben werden,
; damit das Objekt freigegeben und der verwendete Speicher wiedergewonnen werden kann.
ObjRelease(ptr2)
ObjRelease(ptr1)
