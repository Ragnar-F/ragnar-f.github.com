SoundGetWaveVolume

Ermittelt die Wave-Wiedergabelautstärke eines Audio-Geräts.

SoundGetWaveVolume, Ausgabevariable [, Gerätenummer]

Parameter

Ausgabevariable Der Name der Variable, in der die abgerufene Lautstärke gespeichert wird, welche eine Fließkommazahl zwischen 0 und 100 ist. Die Variable wird leer gemacht, wenn ein Problem beim Abrufen der Lautstärke aufgetreten ist. Das Format der Fließkommazahl (eine Zahl mit Dezimalstellen) wird durch SetFormat festgelegt.
Gerätenummer Wenn dieser Parameter weggelassen wird, dann standardmäßig eine 1 (das erste Audiogerät), welches in der Regel das Standardgerät für die Aufnahme und Wiedergabe ist. Verwendet eine Zahl größer als 1, um ein anderes Audiogerät zu bearbeiten.

ErrorLevel

[v1.1.04+] Dieser Befehl ist in der Lage, bei Misserfolg eine Ausnahme auszulösen. Für mehr Informationen, siehe Laufzeitfehler.

ErrorLevel wird bei Misserfolg auf 1 und bei Erfolg auf 0 gesetzt.

Bemerkungen

SoundGetWaveVolume wird zurzeit nicht von Windows Vista und höher unterstützt (es sei denn, AutoHotkey.exe läuft im XP-Kompatibilitätsmodus), dadurch wird die Ausgabevariable leer gemacht.

Die derzeitige Wave-Wiedergabelautstärke kann mit SoundSetWaveVolume eingestellt werden. Einstellungen wie Gesamtlautstärke, Synthesizer, Mikrofon, Stummschaltung, Höhen und Tiefen können mithilfe von SoundSet und SoundGet gesetzt oder abgerufen werden.

Verwandte Befehle

SoundSetWaveVolume, SoundSet, SoundGet, SoundPlay

Beispiel

SoundGetWaveVolume, Ausgabevariable
MsgBox, Die aktuelle Wave-Wiedergabelautstärke liegt bei %Ausgabevariable%`%.
