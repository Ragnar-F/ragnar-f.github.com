SoundGetWaveVolume

Ermittelt die Wave-Wiedergabelautstärke eines Audio-Geräts.

SoundGetWaveVolume, AusgabeVar [, Gerätenummer]

Parameter

AusgabeVar Name der Variable, in der die abgerufene Lautstärke gespeichert werden soll, welche eine Fließkommazahl zwischen 0 und 100 ist. Die Variable wird leer gemacht, wenn ein Problem beim Abrufen der Lautstärke aufgetreten ist. Das Format der Fließkommazahl, z. B. deren Dezimalstellen, wird über SetFormat bestimmt.
Gerätenummer Ist dieser Parameter nicht vorhanden, gilt standardmäßig eine 1 (das erste Audiogerät), das für gewöhnlich das Standardgerät des Systems für Aufnahme und Wiedergabe ist. Mit einer Zahl größer als 1 kann ein anderes Audiogerät bearbeitet werden.

ErrorLevel

[v1.1.04+] Dieser Befehl ist in der Lage, bei Misserfolg eine Ausnahme auszulösen. Für mehr Informationen, siehe Laufzeitfehler.

ErrorLevel wird bei Misserfolg auf 1 und bei Erfolg auf 0 gesetzt.

Bemerkungen

Die derzeitige Wave-Wiedergabelautstärke kann über SoundSetWaveVolume bestimmt werden. Einstellungen wie Gesamtlautstärke, Synthesizer, Mikrofon, Stummschaltung, Höhe und Bass können mithilfe von SoundSet und SoundGet gesetzt oder abgerufen werden.

Windows Vista und höher (v1.1.10+): Dieser Befehl ist das gleiche wie SoundGet, wenn Komponententyp auf Wave und Kontrolltyp auf Volume gesetzt wird.

Verwandte Befehle

SoundSetWaveVolume, SoundSet, SoundGet, SoundPlay

Beispiel

SoundGetWaveVolume, AusgabeVar
MsgBox, Die aktuelle Wave-Wiedergabelautstärke ist bei %AusgabeVar%`%.
