Edit

Öffnet das aktuelle Script mit dem Standard-Editor.

Edit

Dieser Befehl verwendet dazu das Verb "edit" in der Registrierung (oder den Windows-Editor, falls kein Verb vorhanden ist), um das aktuelle Script zur Bearbeitung zu öffnen. Ist das Script jedoch bereits in einem Editor-Fenster geöffnet (abhängig vom Fenstertitel), wird dieses Fenster aktiviert, anstatt eine neue Instanz des Editors zu öffnen.

Dieser Befehl hat keine Wirkung, wenn er innerhalb eines kompilierten Scripts verwendet wird.

Beachtet zudem, dass die Syntax-Hervorhebung von AutoHotkey für verschiedene Editoren aktiviert werden kann (siehe unten). Weiterhin kann die kontextabhängige Hilfe für AutoHotkey-Befehle mit diesem Beispiel aktiviert werden. Außerdem kann die eigene Produktivität erhöht werden, wenn Programme wie ISense verwendet werden, um bestimmte Eingaben bei einem beliebigen Editor automatisch zu vervollständigen. Solche Programme beobachten die Benutzereingabe und zeigen entsprechende Menüs und Parameterlisten an, um das Schreiben teilweise abzunehmen und den Benutzer daran zu erinnern, wie die richtige Parameterreihenfolge lautet.

Verwandte Befehle

Reload

Beispiel

Edit ; Das Script wird zur Bearbeitung geöffnet. 
; Wenn die Befehlszeile des Editors wie folgt aufgebaut ist,
; kann er als Standard-Editor für ahk-Dateien verwenden werden:
;
;   Editor.exe "Vollständiger Pfad des Scripts.ahk"
;
; Beim Ausführen des Scripts wird der Benutzer aufgefordert,
; die ausführbare Datei des Editors auszuwählen.
;
FileSelectFile Editor, 2,, Editor auswählen, Programme (*.exe)
If ErrorLevel
    ExitApp
RegWrite REG_SZ, HKCR, AutoHotkeyScript\Shell\Edit\Command,, "%Editor%" "`%1"

Editoren mit AutoHotkey-Unterstützung

SciTE4AutoHotkey ist eine angepasste Version des Text-Editors SciTE, der für die Bearbeitung von AutoHotkey-Scripts angepasst ist. Seine Eigenschaften umfassen:

  • Syntax-Hervorhebung
  • Intelligente automatische Einrückung
  • Automatische Vervollständigung
  • Aufruftipps (auch als IntelliSense bekannt)
  • Klappbarer Code
  • Unterstützung für interaktive Fehlerbehebung.
  • Weitere Tools für AutoHotkey-Scripts

SciTE4AutoHotkey kann hier gefunden werden: http://www.autohotkey.com/forum/topic58820.html
