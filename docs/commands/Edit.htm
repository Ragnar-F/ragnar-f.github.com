Öffnet das aktuelle Script mit dem Standard-Editor.

Edit

Der Edit-Befehl öffnet mithilfe des entsprechendem Verbs "edit" in der Registrierung (oder mit dem Editor von Windows, falls kein Verb vorhanden ist) das aktuelle Script zur Bearbeitung. Wenn ein Editor das Script allerdings bereits geöffnet hat (abhängig vom Fenstertitel), dann wird das Fenster aktiviert, anstatt eine neue Instanz des Editors zu öffnen.

Dieser Befehl funktioniert nicht, wenn er innerhalb eines kompilierten Scripts verwendet wird.

Weiterhin kann die Syntax-Hervorhebung von AutoHotkey für verschiedene Editoren aktiviert werden (siehe unten). Darüber hinaus kann die kontextabhängige Hilfe für AutoHotkey-Befehle in einem beliebigen Editor mit diesem Beispiel aktiviert werden. Außerdem kann die eigene Produktivität mithilfe eines Tools zur Auto-Vervollständigung wie z. B. ISense erhöht werden. ISense beobachtet die Eingaben des Benutzer und zeigt Menüs sowie Parameterlisten an, die ihm das Schreiben teilweise abnehmen und ihn daran erinnern, wie die Reihenfolge der Parameter ist.

Verwandte Befehle

Reload

Beispiel

Edit ; Das Script wird zur Bearbeitung geöffnet. 
; Wenn die Befehlszeilen des Editors wie folgt aussehen,
; dann kann er als Standard-Editor für ahk-Dateien verwenden werden:
;
;   Editor.exe "Vollständiger Pfad des Scripts.ahk"
;
; Beim Ausführen des Scripts erscheint die Aufforderung,
; die ausführbare Datei des Editors auszuwählen.
;
FileSelectFile Editor, 2,, Editor auswählen, Programme (*.exe)
If ErrorLevel
    ExitApp
RegWrite REG_SZ, HKCR, AutoHotkeyScript\Shell\Edit\Command,, "%Editor%" "`%1"

Editoren mit AutoHotkey-Unterstützung

SciTE4AutoHotkey ist eine angepasste Version des Text-Editors SciTE, der für die Bearbeitung von AutoHotkey-Scripts angepasst ist. Seine Eigenschaften umfassen:

  • Syntax-Hervorhebung
  • Intelligente automatische Einrückung
  • Automatische Vervollständigung
  • Aufruftipps (auch als IntelliSense bekannt)
  • Klappbarer Code
  • Unterstützung für interaktive Fehlerbehebung.
  • Weitere Tools für AutoHotkey-Scripts

SciTE4AutoHotkey kann hier gefunden werden: http://www.autohotkey.com/forum/topic58820.html
