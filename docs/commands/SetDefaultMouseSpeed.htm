SetDefaultMouseSpeed

Bestimmt die Mausgeschwindigkeit, die verwendet wird, wenn sie noch nicht in Click und MouseMove/Click/Drag angegeben ist.

SetDefaultMouseSpeed, Geschwindigkeit

Parameter

Geschwindigkeit Die Bewegungsgeschwindigkeit der Maus von 0 (am schnellsten) bis 100 (am langsamsten).  Hinweis: mit der Geschwindigkeit von 0 wird die Maus sofort bewegt. Dieser Parameter kann ein Ausdruck sein.

Bemerkungen

SetDefaultMouseSpeed wird bei der SendInput- oder SendPlay-Methode ignoriert: damit wird die Maus sofort bewegt (obwohl SetMouseDelay einen Modus hat, der für SendPlay gilt). Um die Maus sichtbar langsamer zu bewegen (z. B. für ein Script, dass eine Demonstration für das Publikum vorführt), verwendet SendEvent {Click 100, 200} oder SendMode Event (optional in Verbindung mit BlockInput).

Wenn dieser Befehl nicht verwendet wird, dann ist 2 die Standard-Mausgeschwindigkeit. Die interne Variable A_DefaultMouseSpeed enthält die aktuelle Einstellung.

Die Befehle MouseClick, MouseMove und MouseClickDrag haben alle einen Parameter, um die Standard-Mausgeschwindigkeit zu überschreiben.

Sobald die Geschwindigkeit größer als Null ist, beeinflusst auch SetMouseDelay die Geschwindigkeit, indem eine Verzögerung nach jeder Mausbewegung erfolgt, schrittweise Richtung Ziel.

Jeder neu gestartete Thread (z. B. ein Hotkey, ein benutzerdefinierter Menüpunkt oder eine zeitlich festgelegte Subroutine) beginnt mit der Standardeinstellung des Befehls. Diese Standardeinstellung kann geändert werden, indem dieser Befehl im automatischen Ausführungsbereich (am Anfang des Scripts) verwendet wird.

Verwandte Befehle

SetMouseDelay, SendMode, Click, MouseClick, MouseMove, MouseClickDrag, SetWinDelay, SetControlDelay, SetKeyDelay, SetKeyDelay

Beispiel

SetDefaultMouseSpeed, 0 ; Bewegt sofort die Maus.
