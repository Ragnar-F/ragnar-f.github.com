SetDefaultMouseSpeed

Bestimmt die Mausgeschwindigkeit, die verwendet werden soll, wenn sie in Click und MouseMove/Click/Drag noch nicht angegeben ist.

SetDefaultMouseSpeed, Geschwindigkeit

Parameter

Geschwindigkeit Wie schnell die Maus bewegt werden soll, von 0 (schnell) bis 100 (langsam). Hinweis: Eine Geschwindigkeit von 0 wird die Maus sofort bewegen. Dieser Parameter kann ein Ausdruck sein.

Bemerkungen

SetDefaultMouseSpeed wird bei den Methoden SendInput und SendPlay ignoriert; sie bewegen die Maus sofort (allerdings hat SetMouseDelay einen Modus, der für SendPlay verwendet werden kann). Wenn die Maus sichtbar langsamer bewegt werden soll (z. B. um einem Publikum etwas vorzuführen), verwende SendEvent {Click 100, 200} oder SendMode Event (optional in Verbindung mit BlockInput).

Wenn dieser Befehl nicht verwendet wird, gilt standardmäßig eine Mausgeschwindigkeit von 2. Die interne Variable A_DefaultMouseSpeed enthält die aktuelle Einstellung.

Die Befehle MouseClick, MouseMove und MouseClickDrag haben Parameter, mit denen die voreingestellte Mausgeschwindigkeit überschrieben werden kann.

Solange Geschwindigkeit größer als 0 ist, wird neben diesem Befehl auch SetMouseDelay Einfluss auf die Geschwindigkeit haben, weil er jedesmal, wenn die Maus schrittweise Richtung Ziel bewegt wird, eine Verzögerung erzeugt.

Jeder neu gestartete Thread (z. B. ein Hotkey, ein benutzerdefinierter Menüpunkt oder eine zeitlich festgelegte Subroutine) beginnt mit der Standardeinstellung des Befehls. Diese Standardeinstellung kann man ändern, wenn man diesen Befehl bereits im automatischen Ausführungsbereich (am Anfang des Scripts) definiert.

Verwandte Befehle

SetMouseDelay, SendMode, Click, MouseClick, MouseMove, MouseClickDrag, SetWinDelay, SetControlDelay, SetKeyDelay, SetKeyDelay

Beispiel

SetDefaultMouseSpeed, 0 ; Bewegt die Maus sofort.
