SetDefaultMouseSpeed

Bestimmt die zu verwendende Mausgeschwindigkeit, wenn sie noch nicht in Click und MouseMove/Click/Drag definiert wurde.

SetDefaultMouseSpeed, Geschwindigkeit

Parameter

Geschwindigkeit Die Bewegungsgeschwindigkeit der Maus von 0 (schnell) bis 100 (langsam). Hinweis: mit einer Geschwindigkeit von 0 wird die Maus sofort bewegt. Dieser Parameter kann ein Ausdruck sein.

Bemerkungen

SetDefaultMouseSpeed hat keinen Einfluss auf SendInput/SendPlay: diese bewegen die Maus sofort (allerdings hat SetMouseDelay einen Modus, der für SendPlay gilt). Um die Maus sichtbar langsamer zu bewegen (z. B. für ein Script, dass eine Demonstration für das Publikum vorführt), kann SendEvent {Click 100, 200} oder SendMode Event verwendet werden (optional in Verbindung mit BlockInput).

Wenn dieser Befehl nicht verwendet wird, gilt für die Mausgeschwindigkeit standardmäßig 2. Die interne Variable A_DefaultMouseSpeed enthält die aktuelle Einstellung.

Die Befehle MouseClick, MouseMove und MouseClickDrag haben alle einen Parameter, um die Standard-Mausgeschwindigkeit zu überschreiben.

Sobald Geschwindigkeit größer als 0 ist, erzeugt SetMouseDelay eine Verzögerung nach jeder schrittweisen Mausbewegung Richtung Ziel.

Jeder neu gestartete Thread (z. B. ein Hotkey, ein benutzerdefinierter Menüpunkt oder eine zeitlich festgelegte Subroutine) beginnt mit der Standardeinstellung des Befehls. Diese Standardeinstellung kann geändert werden, indem dieser Befehl im automatischen Ausführungsbereich (am Anfang des Scripts) verwendet wird.

Verwandte Befehle

SetMouseDelay, SendMode, Click, MouseClick, MouseMove, MouseClickDrag, SetWinDelay, SetControlDelay, SetKeyDelay, SetKeyDelay

Beispiel

SetDefaultMouseSpeed, 0 ; Bewegt sofort die Maus.
