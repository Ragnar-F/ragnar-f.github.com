StringReplace

Ersetzt den angegebenen Teilstring mit einem neuen String.

StringReplace, AusgabeVar, EingabeVar, Suchtext [, Ersatztext, AllesErsetzen?]

Parameter

AusgabeVar Name der Variable, in der das Ergebnis vom Ersetzen gespeichert werden soll.
EingabeVar Name der Variable, deren Inhalt eingelesen werden soll. Der Name sollte nicht mit Prozentzeichen umschlossen werden, sofern nicht die Absicht besteht, den Inhalt der Variable als Namen zu verwenden.
Suchtext Der zu suchende String. Die Übereinstimmung unterscheidet nicht zwischen Groß- und Kleinschreibung, sofern StringCaseSense inaktiv ist.
Ersatztext Suchtext wird mit diesem Text ersetzt. Ist dieser Parameter leer oder nicht vorhanden, wird Suchtext mit einem leeren String ersetzt. Das heißt, dass der gesuchte Text in AusgabeVar wegfällt.
AllesErsetzen?

Ist dieser Parameter nicht vorhanden, wird nur das erste Vorkommen von Suchtext ersetzt. Wenn dieser allerdings eine 1, A oder All ist, werden alle Vorkommen ersetzt.

Es ist möglich, UseErrorLevel anzugeben, um in ErrorLevel die Anzahl der ersetzten Vorkommen zu speichern (0, falls keine). UseErrorLevel bewirkt, dass alle Vorkommen ersetzt werden.

ErrorLevel

Falls der letzte Parameter UseErrorLevel ist, wird ErrorLevel die Anzahl der ersetzten Vorkommen erhalten (0, falls keine). Ansonsten wird ErrorLevel auf 1 gesetzt, wenn Suchtext in EingabeVar nicht gefunden wurde, oder auf 0, falls doch.

Bemerkungen

Bei diesen und allen anderen Befehlen kann AusgabeVar auch EingabeVar sein, und umgekehrt.

Die internen Variablen A_Space und A_Tab enthalten jeweils ein einzelnes Leerzeichen und ein einzelnes Tabulatorzeichen. Sie sind nützlich, um nach Leer- und Tabulatorzeichen zu suchen, die entweder einzeln oder am Anfang oder Ende von Suchtext vorkommen.

Seit v1.0.45 ist die AllSlow-Option aufgrund von Performanceverbesserungen und Speichernutzung veraltet. Sie kann weiterhin angegeben werden, allerdings funktioniert sie nicht mehr.

Verwandte Befehle

RegExReplace(), IfInString, StringCaseSense, StringLeft, StringRight, StringMid, StringTrimLeft, StringTrimRight, StringLen, StringLower, StringUpper, StringGetPos, If Var is Typ

Beispiele

; Entfernt alle CR+LF-Paare in der Zwischenablage:
StringReplace, Clipboard, Clipboard, `r`n, , All

; Ersetzt alle Leerzeichen mit Pluszeichen:
StringReplace, NeuerStr, AlterStr, %A_SPACE%, +, All

; Entfernt alle leeren Zeilen im Text einer Variable:
Loop
{
    StringReplace, EigenerStr, EigenerStr, `r`n`r`n, `r`n, UseErrorLevel
    If ErrorLevel = 0  ; Ersetzen nicht mehr notwendig.
        break
}
