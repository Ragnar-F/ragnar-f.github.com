ToolTip

Erstellt ein Fenster irgendwo auf dem Bildschirm, das immer im Vordergrund ist.

ToolTip [, Text, X, Y, WelcherToolTip]

Parameter

Text

Ist dieser Parameter leer oder nicht vorhanden, wird der Tooltip versteckt, falls er bereits existiert. Ansonsten ist dieser Parameter der Text, der im Tooltip angezeigt werden soll. Wenn der Tooltip mehrzeilig sein soll, verwende zwischen jeder Zeile das Zeilenumbruchszeichen (`n), zum Beispiel Zeile1`nZeile2.

Falls der Text zu lang ist, kann er in mehreren kurzen Zeilen aufgeteilt werden (Fortsetzungsbereich), um ihn besser lesbar und verwaltbar zu machen.
X, Y Die X-Y-Koordinate der oberen linken Ecke des Tooltips, bezogen auf das aktive Fenster (verwende CoordMode, ToolTip, wenn die Koordinate relativ zum Bildschirm sein soll). Sind beide Parameter leer oder nicht vorhanden, wird der Tooltip in der Nähe des Mauszeigers angezeigt. X und Y können Ausdrücke sein.
WelcherToolTip

Dieser Parameter dient dazu, mehrere Tooltips gleichzeitig anzeigen zu lassen. Eine Nummer zwischen 1 und 20 kennzeichnet, auf welchen Tooltip sich dieser Befehl beziehen soll. Ist dieser Parameter nicht vorhanden, gilt standardmäßig die Nummer 1 (der erste ToolTip). Dieser Parameter kann ein Ausdruck sein.

Bemerkungen

Befindet sich die angegebene X-Y-Koordinate außerhalb des Bildschirms, wird der ToolTip neu positioniert, um vollständig sichtbar zu bleiben.

Der Tooltip wird solange angezeigt, bis eine der folgenden Bedingungen eintritt:

  • Das Script wird beendet.
  • Der ToolTip-Befehl wird mit leerem Text-Parameter erneut ausgeführt.
  • Der Benutzer hat den Tooltip angeklickt (dieses Verhalten variiert je nach Betriebssystem).

Per Gui +OwnDialogs kann ein ToolTip an ein GUI-Fenster gebunden werden. So ein Tooltip wird automatisch zerstört, sobald das dazugehörige GUI-Fenster zerstört wird.

Verwandte Befehle

CoordMode, TrayTip, GUI, Progress, SplashTextOn, MsgBox, InputBox, FileSelectFile, FileSelectFolder

Beispiel

ToolTip, Mehrzeiliger`nTooltip, 100, 150

; Ein Tooltip nach einer bestimmten Zeit verschwinden lassen, 
; ohne Sleep zu verwenden (das den aktuellen Thread anhalten würde):
#Persistent
ToolTip, Zeitlich festgelegter Tooltip`nDieser wird für 5 Sekunden angezeigt.
SetTimer, ToolTipEntfernen, 5000
Return

ToolTipEntfernen:
SetTimer, ToolTipEntfernen, Off
ToolTip
Return
