ToolTip

Erstellt ein Fenster irgendwo auf dem Bildschirm, das immer im Vordergrund ist.

ToolTip [, Text, X, Y, WelcherToolTip]

Parameter

Text

Ist dieser Parameter leer oder nicht vorhanden, wird der vorhandene Tooltip versteckt (falls vorhanden). Ansonsten ist dieser Parameter der Text, der im Tooltip angezeigt werden soll. Mithilfe des LF-Zeilenumbruchszeichens (`n) zwischen jeder Zeile kann ein mehrzeiliger ToolTip erstellt werden, zum Beispiel Zeile1`nZeile2.

Falls der Text zu lang ist, kann er in mehreren kurzen Zeilen aufgeteilt werden (Fortsetzungsbereich), um die Lesbarkeit und Wartbarkeit zu verbessern.
X, Y die X- und Y-Position des Tooltips, bezogen auf das aktive Fenster (mit CoordMode, ToolTip können diese relativ zum Bildschirm gemacht werden). Werden die Koordinaten weggelassen, wird der Tooltip nah am Mauszeiger angezeigt. X und Y können Ausdrücke sein.
WelcherToolTip

Dieser Parameter dient dazu, mehrere Tooltips gleichzeitig anzeigen zu lassen. Eine Nummer zwischen 1 und 20 kennzeichnet, auf welches Tooltip-Fenster sich dieser Befehl beziehen wird. Ist dieser Parameter nicht vorhanden, gilt standardmäßig die Nummer 1 (das erste Fenster). Dieser Parameter kann ein Ausdruck sein.

Bemerkungen

Wenn sich die X und Y-Koordinaten des Tooltips außerhalb des Bildschirms befinden, wird dieser neu positioniert, um vollständig sichtbar zu bleiben.

Der Tooltip wird solange angezeigt, bis eine der folgenden Bedingungen eintritt:

  • Das Script wird beendet.
  • Der ToolTip-Befehl wird erneut ausgeführt, aber mit leerem Text.
  • Der Benutzer klickt auf den Tooltip (dieses Verhalten variiert je nach Betriebssystem).

Ein GUI-Fenster kann mithilfe von Gui +OwnDialogs dazu gebracht werden, ein Tooltip an sich zu binden. So ein Tooltip wird automatisch zerstört, sobald das dazugehörige GUI-Fenster zerstört wird.

Verwandte Befehle

CoordMode, TrayTip, GUI, Progress, SplashTextOn, MsgBox, InputBox, FileSelectFile, FileSelectFolder

Beispiel

ToolTip, Mehrzeiliger`nTooltip, 100, 150

; Damit ein Tooltip nach einer bestimmten Zeit verschwindet, 
; ohne Sleep dabei zu verwenden (das den aktuelle Thread stoppen würde):
#Persistent
ToolTip, Zeitlich festgelegter Tooltip`nDieser wird für 5 Sekunden angezeigt.
SetTimer, ToolTipEntfernen, 5000
    Return

    ToolTipEntfernen:
SetTimer, ToolTipEntfernen, Off
ToolTip
Return
