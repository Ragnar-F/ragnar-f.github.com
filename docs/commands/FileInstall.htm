FileInstall

Fügt die angegebene Datei in die kompilierte Version des Scripts ein.

FileInstall, Quelle, Ziel [, Flag]

Parameter

Quelle

Der Name der Datei, die zur kompilierten EXE-Datei hinzugefügt wird. Es wird davon ausgegangen, dass die Datei sich im Ordner des Scripts befindet, solange kein absoluter Pfad angegeben ist.

Der Dateiname darf keine Anführungszeichen, Variablenverweise (z. B.%A_ProgramFiles%) oder Platzhalter enthalten. Darüber hinaus müssen Sonderzeichen wie direkt vorhandene Prozentzeichen und Kommas geschützt werden (wie in jedem anderen Befehlsparameter auch). Schließlich muss dieser Parameter rechts vom FileInstall-Befehl erfolgen (d. h. nicht auf eine Fortsetzungszeile darunter).
Ziel Wenn die Quelle von einer EXE-Datei extrahiert wird, dann ist dieser Parameter der Name der Datei, die erstellt wird. Sie befindet sich im A_WorkingDir-Verzeichnis, solange kein absoluter Pfad angegeben ist. Das Zielverzeichnis muss bereits vorhanden sein. Im Gegensatz zur Quelle können hier Variablenverweise eingesetzt werden.
Flag

(Optional) Mit diesem Parameter wird bestimmt, ob Dateien überschrieben werden, wenn sie bereits vorhanden sind:

0 = (Standard) vorhandene Dateien nicht überschreiben
1 = vorhandene Dateien überschreiben

Dieser Parameter kann ein Ausdruck oder True bzw. False sein (intern auch als 1 bzw. 0 gespeichert).

ErrorLevel

[v1.1.04+] Dieser Befehl ist in der Lage, bei Misserfolg eine Ausnahme auszulösen. Für mehr Informationen, siehe Laufzeitfehler.

Bei Problemen wird ErrorLevel auf 1 gesetzt, ansonsten enthält ErrorLevel eine 0.

Bemerkungen

Dieser Befehl ist eine Anweisung für den Ahk2Exe-Compiler, um Dateien zusätzlich zum kompilierten Script hinzuzufügen. Wird das kompilierte Script danach ausgeführt, werden die Dateien wieder auf das Laufwerk extrahiert.

Die Datei wird während der Script-Kompilierung hinzugefügt.  Wenn das ausgeführte kompilierte Script den FileInstall-Befehl erreicht, dann wird die Datei ans Ziel extrahiert.

Die hinzugefügten Dateien sind komprimiert und verschlüsselt.

Wenn dieser Befehl in einem normalen (nicht kompilierten) Script verwendet wird, dann erfolgt ein einfaches Kopieren der Datei (zum Testen des Scripts, wenn es kompiliert werden soll).

Verwandte Befehle

FileCopy, #Include

Beispiel

FileInstall, C:\Eigene Dokumente\Eigene Datei.txt, %A_ProgramFiles%\Eigene Anwendung\Readme.txt, 1
