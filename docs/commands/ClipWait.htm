ClipWait

Wartet, bis die Zwischenablage Text oder Dateien enthält.

ClipWait [, Wartezeit, 1]

Parameter

Wartezeit Ist dieser Parameter nicht vorhanden, wartet der Befehl unendlich lang. Ansonsten wartet der Befehl die angegebenen Sekunden (kann einen Dezimalpunkt enthalten oder ein Ausdruck sein). Der Wert 0 entspricht 0.5.
1
 Ist dieser Parameter nicht vorhanden, wartet der Befehl gezielt auf Texte oder Dateien, die in der Zwischenablage erscheinen. Falls dieser Parameter eine 1 enthält (kann auch ein Ausdruck sein), wartet der Befehl auf Daten beliebiger Art, die in der Zwischenablage erscheinen.

ErrorLevel

Sobald die Wartezeit abgelaufen ist, wird ErrorLevel auf 1 gesetzt. Ansonsten wird ErrorLevel auf 0 gesetzt, wenn die Zwischenablage Daten enthält.

Bemerkungen

Es ist besser, diesen Befehl zu verwenden, anstatt mit einer selbstgemachten Schleife zu überprüfen, ob die Zwischenablage leer ist. Der Grund ist, dass die Zwischenablage nie durch diesen Befehl geöffnet wird, wodurch eine höhere Performance erzielt und jede mögliche Störung bei anderen Anwendungen, die auch die Zwischenablage verwenden, verhindert wird.

Dieser Befehl berücksichtigt alle Daten, die in Text umgewandelt werden können (z. B. HTML). Es werden auch Dateien berücksichtigt, die per STRG+C in einem Explorer-Fenster kopiert wurden. Solche Dateien werden automatisch in ihre Dateinamen umgewandelt (mit vollständigem Pfad), sobald die Clipboard-Variable (%clipboard%) im Script verwendet wird. Siehe Clipboard für weitere Details.

Enthält der letzte Parameter eine 1, akzeptiert der Befehl alle möglichen Daten, die in der Zwischenablage erscheinen. Diese Option kann in Verbindung mit ClipboardAll benutzt werden, um Nicht-Text-Elemente wie Bilder zu speichern.

Während der Befehl in einem Wartezustand ist, können neue Threads mittels Hotkey, benutzerdefinierte Menüeinträge oder Timer gestartet werden.

Verwandte Befehle

Clipboard, WinWait, KeyWait

Beispiel

clipboard = ; Zwischenablage leeren
Send, ^c
ClipWait, 2
If ErrorLevel
{
    MsgBox, Kopieren des Textes in die Zwischenablage fehlgeschlagen.
    Return
}
MsgBox, clipboard = %clipboard%
Return
