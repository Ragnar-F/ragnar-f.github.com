FileGetVersion

Ermittelt die Version einer Datei.

FileGetVersion, Ausgabevariable [, Dateiname]

Parameter

Ausgabevariable Der Name der Variable, in der der String bzw. die Versionsnummer gespeichert wird.
Dateiname Der Name der Datei, die sich im A_WorkingDir-Verzeichnis befindet, solange kein absoluter Pfad angegeben ist. Wenn nicht vorhanden, dann wird die aktuelle Datei in der innersten dateilesende Schleife verwendet.

ErrorLevel

[v1.1.04+] Dieser Befehl ist in der Lage, bei Misserfolg eine Ausnahme auszulösen. Für mehr Informationen, siehe Laufzeitfehler.

Bei Problemen wird ErrorLevel auf 1 gesetzt, ansonsten enthält ErrorLevel eine 0.

Die interne Variable A_LastError enthält den Rückgabewert der Systemfunktion GetLastError().

Bemerkungen

Die meisten nicht-ausführbare Dateien (auch einige EXE-Dateien) haben keine Version, daher ist die Ausgabevariable in diesem Fall leer.

Verwandte Befehle

FileGetAttrib, FileSetAttrib, FileGetTime, FileSetTime, FileGetSize, dateilesende Schleife

Beispiel

FileGetVersion, version, C:\Eigene Anwendung.exe
FileGetVersion, version, %A_ProgramFiles%\AutoHotkey\AutoHotkey.exe
