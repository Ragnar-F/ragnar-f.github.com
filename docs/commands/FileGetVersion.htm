FileGetVersion

Ermittelt die Version einer Datei.

FileGetVersion, AusgabeVar [, Dateiname]

Parameter

AusgabeVar

Name der Variable, in der der String bzw. die Versionsnummer gespeichert werden soll.

Dateiname

Name der Datei, die sich im A_WorkingDir-Verzeichnis befindet, solange kein absoluter Pfad angegeben ist. Ist dieser Parameter nicht vorhanden, wird stattdessen die aktuelle Datei von der innersten umschließenden Loop (Dateien & Ordner) verwendet.

ErrorLevel

[v1.1.04+] Dieser Befehl ist in der Lage, bei Misserfolg eine Ausnahme auszulösen. Für mehr Informationen, siehe Laufzeitfehler.

ErrorLevel wird bei Misserfolg auf 1 und bei Erfolg auf 0 gesetzt.

Die Variable A_LastError enthält den Rückgabewert der Systemfunktion GetLastError().

Bemerkungen

Die meisten nicht ausführbaren Dateien (auch einige EXE-Dateien) werden keine Version haben, wodurch AusgabeVar leer sein wird.

Siehe auch

FileGetAttrib, FileSetAttrib, FileGetTime, FileSetTime, FileGetSize, Loop (Dateien & Ordner)

Beispiel

FileGetVersion, version, C:\Meine Anwendung.exe
FileGetVersion, version, %A_ProgramFiles%\AutoHotkey\AutoHotkey.exe
