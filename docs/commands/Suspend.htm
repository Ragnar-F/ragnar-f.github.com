Suspend

Deaktiviert oder aktiviert alle oder bestimmte Hotkeys und Hotstrings.

Suspend [, Modus]

Parameter

Modus

On: Sperrt alle Hotkeys und Hotstrings (siehe Bemerkungen für Ausnahmen).

Off: Aktiviert die zuvor deaktivierten Hotkeys und Hotstrings.

Toggle (Standard): Setzt den Zustand jeweils auf die entgegengesetzte Seite (On oder Off).

Permit: Markiert die aktuelle Subroutine, um diese von der Sperrung auszuschließen.

Bemerkungen

Jede Hotkey/Hotstring-Subroutine, die auf der ersten Zeile Suspend (außer Suspend On) stehen hat, wird von der Sperrung ausgeschlossen. Das heißt, dass der Hotkey weiterhin aktiviert bleibt, auch dann, wenn die Sperrung aktiv ist. Dadurch ist es möglich, die Sperrung mit solch einem Hotkey aufzuheben.

Mit #IfWinActive/Exist können bestimmte Hotkeys oder Hotstrings auf Basis des vorhandenen Fenstertyps automatisch deaktiviert werden.

Threads, die bereits laufen, können von Suspend nicht gestoppt werden - dazu ist nur der Pause-Befehl in der Lage.

Sobald Hotkeys mit dem Suspend-Befehl gesperrt werden, wird ein S im Tray-Icon angezeigt. Um das zu verhindern, kann das Icon "eingefroren" werden, indem eine 1 im letzten Parameter des Menu-Befehls angegeben wird. Zum Beispiel:

Menu, Tray, Icon, C:\Eigenes Icon.ico, , 1

Die interne Variable A_IsSuspended enthält eine 1, wenn das Script mit Suspend gesperrt wurde, ansonsten 0.

Verwandte Befehle

#IfWinActive/Exist, Pause, Menu, ExitApp

Beispiel

^!s::Suspend  ; Weist einem Hotkey die Umschaltfunktion von Suspend zu.
; Sendet einen Suspend-Befehl zu einem anderen Script.
DetectHiddenWindows, On
WM_COMMAND := 0x111
ID_FILE_SUSPEND := 65404
PostMessage, WM_COMMAND, ID_FILE_SUSPEND,,, C:\DeinScript.ahk ahk_class AutoHotkey
