SendLevel [v1.1.06+]

Bestimmt, welche künstliche Tastatur- oder Mausereignisse von Hotkeys und Hotstrings ignoriert werden.

SendLevel

Parameter

Level Ein Integer zwischen 0 und 100, das ein Ausdruck sein kann.

Bemerkungen

Tastatur- und Mausereignisse mit angegebenem Sendelevel lösen keine Hotkeys oder Hotstrings auf gleichem oder niedrigerem Eingabelevel aus. Dieser Befehl bestimmt den Sendelevel für alle Tastatur- und Mausereignisse, die vom Script erzeugt werden, außer durch ControlSend (das nie Hotkeys oder Hotstrings auslöst) und einzeiligen Hotstrings (die immer mit Sendelevel 0 arbeiten).

Level-0-Hotkeys können mit der Reg-Methode implementiert werden. In diesem Fall werden sie möglicherweise durch künstliche Eingaben jeden Levels ausgelöst (selbst Level 0).

Jeder neu gestartete Hotkey- oder Hotstring-Thread beginnt mit einem Sendelevel, das mit dem Eingabelevel vom Hotkey oder Hotstring übereinstimmt. Jeder andere neu gestartete Thread (z. B. ein Hotkey, ein benutzerdefinierter Menüpunkt oder eine zeitlich festgelegte Subroutine) beginnt mit der Standardeinstellung des Befehls (in diesem Fall eine 0). Diese Standardeinstellung kann geändert werden, indem dieser Befehl im Automatischen Ausführungsbereich (am Anfang des Scripts) verwendet wird.

Falls SendLevel im Automatischen Ausführungsbereich verwendet wird, beeinflusst diese Direktive zudem die Neubelegung von Tasten.

AutoHotkey-Versionen älter als v1.1.06 verhalten sich so, als wäre #InputLevel 0 und SendLevel 0 wirksam.

Verwandte Befehle

#InputLevel, Send, Click, MouseClick, MouseClickDrag

Beispiele

SendLevel 1
Send bzw{Space} ; Erzeugt "beziehungsweise ".
