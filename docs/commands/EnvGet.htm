EnvGet [v1.0.43.08+]

Ermittelt eine Umgebungsvariable.

EnvGet, AusgabeVar, UmgebungsVar

Parameter

AusgabeVar Name der Variable, in der der String gespeichert werden soll.
UmgebungsVar Name der Umgebungsvariable, die ermittelt werden soll. Zum Beispiel: EnvGet, AusgabeVar, Pfad.

Bemerkungen

Falls die angegebene Umgebungsvariable leer oder nicht vorhanden ist, wird AusgabeVar leer gemacht.

Das Betriebssystem begrenzt die Textgröße jeder Umgebungsvariable auf 32 KB.

Verwandte Befehle

EnvSet, #NoEnv, Umgebungsvariablen, EnvUpdate, SetEnv, Run, RunWait

Beispiel

EnvGet, AusgabeVar, LogonServer
