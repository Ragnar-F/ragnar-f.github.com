EnvGet [v1.0.43.08+]

Ermittelt eine Umgebungsvariable.

EnvGet, Ausgabevariable, Umgebungsvariable

Parameter

Ausgabevariable

Der Name der Variable, in der der String gespeichert wird.

Umgebungsvariable

Der Name der Umgebungsvariable, die ermittelt werden soll. Zum Beispiel: EnvGet, Ausgabevariable, Pfad

Bemerkungen

Wenn die angegebene Umgebungsvariable leer oder nicht vorhanden ist, dann wird die Ausgabevariable leer gemacht.

Eine Umgebungsvariable ist vom Betriebssystem auf 32 KB begrenzt.

Dieser Befehl existiert, weil normale Script-Variablen nicht in die Systemumgebung gespeichert werden. Das liegt zum einen daran, dass die Performance darunter leiden würde, und zum anderen, dass Umgebungsvariablen auf 32 KB begrenzt sind.

Verwandte Befehle

EnvSet, Umgebungsvariablen, EnvUpdate, Run, RunWait

Beispiel

EnvGet, Ausgabevariable, LogonServer
