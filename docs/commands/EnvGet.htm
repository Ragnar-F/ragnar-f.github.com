EnvGet [v1.0.43.08+]

Ermittelt eine Umgebungsvariable.

EnvGet, Ausgabevariable, Umgebungsvariable

Parameter

Ausgabevariable Der Name der Variable, in der der String gespeichert wird.
Umgebungsvariable Der Name der Umgebungsvariable, die ermittelt werden soll. Zum Beispiel: EnvGet, Ausgabevariable, Pfad

Bemerkungen

Wenn die angegebene Umgebungsvariable leer oder nicht vorhanden ist, dann wird die Ausgabevariable leer gemacht.

Eine Umgebungsvariable ist vom Betriebssystem auf 32 KB begrenzt.

Verwandte Befehle

EnvSet, #NoEnv, Umgebungsvariablen, EnvUpdate, SetEnv, Run, RunWait

Beispiel

EnvGet, Ausgabevariable, LogonServer
