Exit

Beendet den aktuellen Thread.

Exit [, ExitCode]

Parameter

ExitCode

Ein Integer-Wert zwischen -2147483648 und 2147483647 (kann ein Ausdruck sein), welcher zurückgegeben wird, wenn das Script beendet wird. Dieser Code ist für jedes Programm zugänglich, welches das Script startet, wie beispielsweise durch ein anderes Script (mittels RunWait) oder einer Batch-Datei (.bat). Sobald dieser Parameter weggelassen wird, enthält er standardmäßig eine 0. Eine Null wird normalerweise verwendet, um Erfolg zu zeigen.

Bemerkungen

Wenn das Script persistent und das der letzte Thread ist, wird das ganze Script beendet.

Ansonsten wird der aktuelle Thread beendet. Das heißt, dass die Subroutinen, die direkt oder indirekt durch ein Menü, Timer oder Hotkey aufgerufen wurden, sofort mit einem Return beendet werden. Sobald dieser Befehl direkt in so einer Subroutine verwendet wird (anstatt in einer Subroutine, die indirekt von ihr aufgerufen wurde), hat Exit die gleiche Bedeutung wie Return.

Mit ExitApp kann ein Script vollständig beendet werden, das persistent ist.

Verwandte Befehle

ExitApp, OnExit, Funktionen, Gosub, Return, Threads

Beispiel

#z::
Gosub, Sub2
MsgBox, Diese MsgBox wird wegen Exit nicht angezeigt.
Return

Sub2:
Exit  ; Beendet diese Subroutine, sowie auch die aufrufende Subroutine.
