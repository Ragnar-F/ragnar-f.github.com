Deref [v2]

Erweitert Variablenverweise und Schutzsequenzen, die sich in anderen Variablen befinden.

Deref, Ausgabevariable, String

Parameter

Ausgabevariable

Der Name der Variable, in der das Ergebnis gespeichert wird.

String

Ein String, der Variablenverweise und/oder Schutzsequenzen enthält.

Bemerkung

Jeder falsch formatierte Variablenverweis wird vom erweiterten Ergebnis ausgeschlossen. Das Gleiche gilt auch, wenn sich die Ausgabevariable in sich selbst erweitert. Das heißt, dass Verweise auf die Ausgabevariable innerhalb der String-Variablen von der Erweiterung ausgeschlossen werden (beachtet allerdings, dass der String selbst die %Ausgabevariable% sein kann).

Innerhalb einer Funktion kann jeder Variablenverweis im String in eine lokale oder globale Variable aufgelöst werden. Hierbei gelten die Standardregeln für die Gültigkeitsbereiche der Variablen.

Beispiele

Variable1 := "Test"
Variable2 := Chr(37) "Variable1"  ; Chr(37) ist %
Deref Ausgabevariable, %Variable2%
MsgBox %Ausgabevariable%  ; Zeigt "Test"
