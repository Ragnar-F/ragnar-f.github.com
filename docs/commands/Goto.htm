Goto

Springt zu einem bestimmten Label und setzt die Ausführung fort.

Goto, Label

Parameter

Label Name des Labels, das angesprungen werden soll.

Bemerkungen

Verwendet man ein dynamisches Label wie %EigenesLabel%, wird eine Fehlermeldung angezeigt, falls das Label nicht existieren sollte. Um das zu verhindert, kann man vorher IsLabel() aufrufen. Zum Beispiel:

If IsLabel(VarMitLabelname)
    Goto %VarMitLabelname%

Goto sollte nur in Ausnahmefällen verwendet werden, weil es Scripts schlechter lesbar macht und dadurch schwerer zu verwalten ist. Verwende stattdessen Else, Blocks, Break und Continue.

Verwandte Befehle

Gosub, Return, IsLabel(), Else, Block, Break, Continue

Beispiel

Goto, EigenesLabel
...
EigenesLabel:
Sleep, 100
