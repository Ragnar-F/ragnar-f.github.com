Goto

Springt zu einem bestimmten Label und setzt die Ausführung fort.

Goto, Label

Parameter

Label

Name des Labels, das angesprungen werden soll.

Bemerkungen

Verwendet man ein dynamisches Label wie %MeinLabel%, wird eine Fehlermeldung angezeigt, falls das Label nicht existieren sollte. Um das zu verhindern, kann man vorher IsLabel() aufrufen. Zum Beispiel:

if IsLabel(VarMitLabelname)
    Goto %VarMitLabelname%

Goto sollte nur in Ausnahmefällen verwendet werden, weil es Scripts schlechter lesbar macht und dadurch schwerer zu verwalten ist. Verwende stattdessen Else, Blöcke, Break und Continue.
