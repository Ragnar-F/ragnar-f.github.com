Goto

Springt zu einem bestimmten Label und setzt die Ausführung fort.

Goto, Label

Parameter

Label Der Name des Labels, zu dem gesprungen wird.

Bemerkungen

Beim Verwenden eines dynamischen Labels wie %EigenesLabel% erscheint eine Fehlermeldung, wenn das Label nicht existiert. Zur Vermeidung dieses Problems kann IsLabel() verwendet werden. Zum Beispiel:

If IsLabel(VarMitLabelname)
    Goto %VarMitLabelname%

Es wird davon abgeraten, Goto zu verwenden, weil Scripts damit weniger lesbar und schwerer zu pflegen sind. Als Ersatz sollten Else, Blocks, Break und Continue in Betracht gezogen werden.

Verwandte Befehle

Gosub, Return, IsLabel(), Else, Block, Break, Continue
