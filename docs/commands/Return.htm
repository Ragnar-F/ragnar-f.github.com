Return

Bewirkt die Rückkehr aus einer Subroutine, deren Ausführung vorher mittels Funktionsaufruf, Gosub, Hotkey-Aktivierung, GroupActivate oder Ähnlichem angesprungen worden ist.

Return [, Ausdruck]

Parameter

Ausdruck

Dieser Parameter sollte nur verwendet werden, wenn Return in einer Funktion verwendet wird.

Da dieser Parameter ein Ausdruck ist, sind alle folgenden Beispiele gültig:

Return 3
Return "direkt vorhandener String"
Return MeineVar
Return i + 1
Return true  ; Gibt 1 für "wahr" zurück.
Return Elementanzahl < Elementmaximum  ; Gibt einen wahren oder falschen Wert zurück.
Return FindeFarbe(Zielfarbe)

Bekannte Einschränkung: Aus Gründen der Abwärtskompatibilität und Benutzerfreundlichkeit sind die folgenden zwei Beispiele funktionsgemäß identisch:

Return MeineVar
Return %MeineVar%

Das heißt, dass eine Variable, die von Prozentzeichen umschlossen ist, nicht als Ausdruck behandelt wird. Um diese Einschränkung zu umgehen, sollte es zusätzlich mit Klammern umschlossen werden, damit es unmissverständlich ein Ausdruck ist. Zum Beispiel: Return (%MeineVar%).

Bemerkungen

Wenn kein Aufrufer vorhanden ist, zu dem zurückgekehrt werden kann, führt Return stattdessen ein Exit aus.

Es gibt verschiedene Wege, mehrere Werte von einer Funktion an ihren Aufrufer zurückzugeben, siehe dazu Rückgabewerte an den Aufrufer.

Beispiel

Gosub MeineSubroutine
Return
