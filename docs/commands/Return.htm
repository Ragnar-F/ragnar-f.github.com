Return

Bewirkt die Rückkehr aus einer Subroutine, deren Ausführung vorher mittels Funktionsaufruf, Gosub, Hotkey-Aktivierung, GroupActivate oder Ähnlichem angesprungen wurden ist.

Return [, Ausdruck]

Parameter

Ausdruck

Dieser Parameter sollte nur verwendet werden, wenn Return in einer Funktion verwendet wird.

Da dieser Parameter ein Ausdruck ist, sind alle folgenden Beispiele gültig:

Return 3
Return "direkt vorhandener String"
Return EigeneVar
Return i + 1
Return true  ; Gibt 1 zurück, dass "wahr" bedeutet.
Return Elementzählung < Elementmaximum  ; Gibt einen wahren oder falschen Wert zurück.
Return FindColor(TargetColor)

Bekannte Einschränkung: Aus Gründen der Abwärtskompatibilität und Benutzerfreundlichkeit sind die folgenden zwei Beispiele funktionsgemäß identisch:

Return EigeneVar
Return %EigeneVar%

Das heißt, dass eine Variable, die von Prozentzeichen umschlossen ist, nicht als Ausdruck behandelt wird. Um diese Einschränkung zu umgehen, umschließt es zusätzlich mit Klammern, damit es unmissverständlich ein Ausdruck ist. Zum Beispiel: Return (%EigeneVar%)

Bemerkungen

Wenn von nichts zurückgekehrt werden kann, dann führt Return stattdessen ein Exit aus.

Verwandte Befehle

Funktionen, Gosub, Exit, ExitApp, GroupActivate

Beispiel

#z::
MsgBox Der Hotkey Win+Z wurde gedrückt.
Gosub EigeneSubroutine
Return

EigeneSubroutine:
Sleep 1000
Return
