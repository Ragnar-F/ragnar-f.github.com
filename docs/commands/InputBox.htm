InputBox

Zeigt ein Fenster zur Eingabe eines Strings an.

InputBox, Ausgabevariable [, Titel, Anzeigetext, Optionen, Standardwert]

Parameter

Ausgabevariable

Der Name der Variable, in der der eingegebene Text des Benutzers gespeichert wird.

Titel

Der Titel der InputBox. Wenn leer oder nicht vorhanden, dann wird standardmäßig der aktuelle Name des Scripts verwendet.

Anzeigetext

Text des Eingabefensters, der darauf hinweist, was eingegeben werden soll. Wenn der Anzeigetext lang ist, dann kann die Liste in mehreren kurzen Zeilen aufgeteilt werden (Fortsetzungsbereich), die die Lesbarkeit und Wartbarkeit verbessern könnten.

Optionen

Ein String mit Optionen, die durch Leer- oder Tabulatorzeichen getrennt werden müssen und unabhängig von der Groß- und Kleinschreibung sind:

Xn Yn: Die X- und Y-Koordinaten des Dialogfensters. Zum Beispiel wird mit X0 Y0 das Fenster in die linken oberen Ecke des Desktops verschoben. Wenn einer der Koordinaten weggelassen wird, wird das Fenster in dieser Abmessung zentriert. Jede Koordinate kann negativ sein, um das Fenster teilweise oder vollständig außerhalb des Bildschirms zu positionieren (oder auf einem zweiten Monitor, falls mehrere Monitore aktiv sind).

Wn Hn: Die Breite und Höhe des Dialogfensters. Zum Beispiel ist W375 H189 die Standardgröße.

T: Die Zeitüberschreitung in Sekunden. Zum Beispiel wird sie mit T10.0 auf 10 Sekunden gesetzt. Wenn dieser Wert 2147483 (24.8 Tage) überschreitet, wird dieser auf 2147483 gesetzt. Wenn die Zeit abgelaufen ist, wird das Dialogfenster automatisch geschlossen und ErrorLevel auf 2 gesetzt. Die Ausgabevariable enthält weiterhin die Eingabe des Benutzers.

Password: Maskiert die Benutzereingabe. Mit dem folgenden Beispiel kann das Zeichen dafür bestimmt werden: Password*

Standardwert

Ein String, der anfänglich im Edit-Feld des Eingabefensters erscheint. Der Benutzer kann den String z. B. mit der Rücktaste ändern.

ErrorLevel

[v1.1.04+] Dieser Befehl ist in der Lage, bei Misserfolg eine Ausnahme auszulösen. Für mehr Informationen, siehe Laufzeitfehler.

Siehe unten.

Bemerkungen

Das Fenster erlaubt dem Benutzer, einen Text einzugeben und auf die Schaltflächen OK oder ABBRECHEN zu drücken. Der Benutzer kann das Fenster durch das Ziehen der Rahmen skalieren.

Sobald der Benutzer die Schaltfläche ABBRECHEN drückt, wird ErrorLevel auf 1 gesetzt (oder auf 0 bei der Schaltfläche OK) und in der Ausgabevariable der Eingabewert gespeichert. Dadurch kann die ABBRECHEN-Schaltfläche nebenbei eine andere Funktion ausführen, wenn es gewünscht ist.

Wenn die Wartezeit des Fensters abläuft, wird ErrorLevel auf 2 gesetzt.

Ein GUI-Fenster kann eine modale InputBox mittels Gui, +OwnDialogs anzeigen. Eine modale InputBox hindert den Benutzer solange daran, mit dem GUI-Fenster zu interagieren, bis die InputBox wieder geschlossen wird.

Verwandte Befehle

GUI, Input, MsgBox, FileSelectFile, FileSelectFolder, ToolTip

Beispiel

InputBox, Passwort, Passwort eingeben, (die Eingabe wird versteckt), password
InputBox, Benutzereingabe, Telefonnummer, Bitte geben sie eine Telefonnummer ein., w640 h480
If ErrorLevel
    MsgBox, ABBRECHEN wurde gedrückt.
Else
    MsgBox, Sie haben "%Benutzereingabe%" eingegeben.
