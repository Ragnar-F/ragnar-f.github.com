DetectHiddenText

Bestimmt, ob versteckte Texte in einem Fenster entdeckt werden können. Das beeinflusst Befehle wie IfWinExist und WinActivate.

DetectHiddenText, On|Off

Parameter

On|Off

On: Standardeinstellung für Nicht-AutoIt-v2-Scripts (z. B. .ahk, .ini). Versteckter Text wird erkannt.

Off: Standardeinstellung für AutoIt-v2-Scripts. Versteckter Text wird nicht erkannt.

Bemerkungen

"Versteckter Text" ist ein Begriff, der sich auf Steuerelemente eines Fenster bezieht, die unsichtbar ist. Deren Texte sind somit auch "versteckt". Das Ausschalten von DetectHiddenText kann nützlich sein, falls es darum geht, die Unterschiede zwischen einem Multipane- und Multitab-Fenster zu erkennen. Mithilfe von Windows Spy kann ermittelt werden, welcher Text vom aktiven Fenster versteckt ist. Alle Befehle, die einen Fenstertext-Parameter akzeptieren, werden durch diese Einstellung sowie auch durch WinActivate, IfWinActive, WinWait und IfWinExist beeinflusst.

Die interne Variable A_DetectHiddenText enthält die aktuelle Einstellung (On oder Off).

Jeder neu gestartete Thread (z. B. ein Hotkey, ein benutzerdefinierter Menüpunkt oder eine zeitlich festgelegte Subroutine) beginnt mit der Standardeinstellung des Befehls. Diese Standardeinstellung kann geändert werden, indem dieser Befehl im automatischen Ausführungsbereich (am Anfang des Scripts) verwendet wird.

Verwandte Befehle

DetectHiddenWindows

Beispiel

DetectHiddenText, off
