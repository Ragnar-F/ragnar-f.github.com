ExitApp

Beendet das Script bedingungslos.

ExitApp [, ExitCode]

Parameter

ExitCode Ein Integer zwischen -2147483648 und 2147483647 (kann seit v1.0.48.01 ein Ausdruck sein), der zurückgegeben werden soll, sobald das Script beendet wird. Dieser Code ist für jedes Programm zugänglich, das das Script gestartet hat, wie beispielsweise durch ein anderes Script (mittels RunWait) oder einer Batch-Datei (.bat). Ist dieser Parameter nicht vorhanden, gilt standardmäßig 0. Eine Null wird normalerweise verwendet, um Erfolg zu kennzeichnen. Hinweis: Windows 95 ist dahingehend eingeschränkt, wie groß ExitCode sein kann.

Bemerkungen

Das Script wird sofort beendet, sofern es keine OnExit-Subroutine hat. Das ist das gleiche wie, als würde "Exit" im Tray-Menü des Scripts oder im Hauptmenü ausgewählt.

Exit und ExitApp verhalten sich identisch, wenn das Script keine Hotkeys enthält, nicht persistent ist, und die Num/Scroll/Capslock-Tasten nicht auf AlwaysOn oder AlwaysOff gesetzt hat.

Hat das Script eine OnExist-Subroutine, wird es als Reaktion auf ExitApp ausgeführt.

Verwandte Befehle

Exit, OnExit, #Persistent

Beispiel

#x::ExitApp  ; Mit diesem Hotkey kann das Script beendet werden.
