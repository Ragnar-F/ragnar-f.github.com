ExitApp

Beendet das Script bedingungslos.

ExitApp [, ExitCode]

Parameter

ExitCode Ein Integer-Wert zwischen -2147483648 und 2147483647 (kann seit v1.0.48.01 ein Ausdruck sein), welcher zurückgegeben wird, wenn das Script beendet wird. Dieser Code ist für jedes Programm zugänglich, welches das Script startet, wie beispielsweise durch ein anderes Script (mittels RunWait) oder einer Batch-Datei (.bat). Wenn nicht vorhanden, dann enthält ExitCode standardmäßig eine Null. Eine Null wird normalerweise verwendet, um Erfolg zu zeigen. Hinweis: In Windows 95 kann es sein, dass die Größe von ExitCode eingeschränkt ist.

Bemerkungen

Das Script wird sofort beendet, sofern keine OnExit-Subroutine vorhanden ist. Das ist gleichbedeutend mit der Auswahl von "Exit" im Tray-Menü des Scripts oder im Hauptmenü.

Exit und ExitApp verhalten sich identisch, solange keine Hotkeys aktiv sind oder die Befehle #Persistent, SetCapsLockState, SetNumLockState oder SetScrollLockState nicht verwendet werden.

Eine OnExist-Subroutine wird in Verbindung mit ExitApp ausgeführt.

Verwandte Befehle

Exit, OnExit, #Persistent

Beispiel

#x::ExitApp  ; Mit diesem Hotkey kann das Script beendet werden.
