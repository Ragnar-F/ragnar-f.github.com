ExitApp

Beendet das Script bedingungslos.

ExitApp [, ExitCode]

Parameter

ExitCode

Ein Integer-Wert zwischen -2147483648 und 2147483647, welcher zurückgegeben wird, wenn das Script beendet wird. Dieser Code ist für jedes Programm zugänglich, welches das Script startet, wie beispielsweise durch ein anderes Script (mittels RunWait) oder einer Batch-Datei (.bat). Wenn nicht vorhanden, dann enthält ExitCode standardmäßig eine Null. Eine Null wird normalerweise verwendet, um Erfolg zu zeigen.

Bemerkungen

Das Script wird sofort beendet, sofern keine OnExit-Subroutine vorhanden ist. Das ist gleichbedeutend mit der Auswahl von "Exit" im Tray-Menü des Scripts oder im Hauptmenü.

ExitApp ist oft in Scripts unnötig, die persistent sind.

Eine OnExist-Subroutine wird in Verbindung mit ExitApp ausgeführt.

Verwandte Befehle

Exit, OnExit

Beispiel

#x::ExitApp  ; Mit diesem Hotkey kann das Script beendet werden.
