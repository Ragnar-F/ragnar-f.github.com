ExitApp

Beendet das Script bedingungslos.

ExitApp [, ExitCode]

Parameter

ExitCode

Ein Integer zwischen -2147483648 und 2147483647 (kann seit v1.0.48.01 ein Ausdruck sein), der beim Beenden des Scripts an den Aufrufer zurückgegeben werden soll. Dieser Code kann von jedem Programm abgefangen werden, der das Script gestartet hat, wie z. B. ein anderes Script (per RunWait) oder eine Batch-Datei (.bat). Fehlt dieser Parameter, gilt als ExitCode standardmäßig Null. Eine Null wird traditionell verwendet, um Erfolg zu melden.

Bemerkungen

Das Script wird sofort beendet, sofern es keine OnExit-Subroutine hat. Das ist das gleiche wie, als würde man "Exit" im Tray- oder Hauptmenü des Scripts auswählen.

Exit und ExitApp verhalten sich gleich, wenn das Script weder Hotkeys hat, noch persistent ist, noch die NUM/ROLLEN/FESTSTELL-Taste(n) auf AlwaysOn oder AlwaysOff gesetzt hat.

Hat das Script eine OnExist-Subroutine, wird es als Reaktion auf ExitApp ausgeführt.

Siehe auch

Exit, OnExit, #Persistent

Beispiel

#x::ExitApp  ; Weise einen Hotkey zu, um diesen Script zu beenden.
