IfInString / IfNotInString

Überprüft, ob eine Variable einen bestimmten String enthält.

IfInString, Variable, Suchstring
IfNotInString, Variable, Suchstring
Position := InStr(Heuhaufen, Nadel [, Zeichengenauigkeit, Startposition]]) ; Siehe InStr()-Funktion für Details.

Parameter

Variable Name der Variable, dessen Inhalt nach einer Übereinstimmung durchsucht wird.
Suchstring Text, der gesucht wird. Die Übereinstimmung ist erst von der Groß- bzw. Kleinschreibung abhängig, wenn StringCaseSense aktiv ist.

Bemerkungen

Die internen Variablen %A_Space% und %A_Tab% enthalten jeweils ein Leer- und Tabulatorzeichen, die dann nützlich sind, wenn nur nach diesen Zeichen gesucht wird.

Auf der gleichen Zeile kann ein weiterer Befehl stehen. Mit anderen Worten haben die beiden nachfolgenden Zeilen die gleiche Wirkung:

IfInString, EigeneVar, abc, Gosub, Process1
IfInString, EigeneVar, abc
    Gosub, Process1

Im Gegensatz zu Befehlen mit Namen werden andere Elemente nicht auf derselben Zeile unterstützt. Zum Beispiel:

IfInString, EigeneVar, abc, gefunden := true  ; Ungültig.

Verwandte Befehle

InStr(), RegExMatch(), StringGetPos, StringCaseSense, IfEqual, If Variable in/contains Übereinstimmungsliste, If Variable between, If Variable is Typ, Blocks, Else

Beispiel

Heuhaufen = abcdefghijklmnopqrs
Nadel = abc
IfInString, Heuhaufen, %Nadel%
{
    MsgBox, Der Text wurde gefunden.
    Return
}
Else
    Sleep, 1
