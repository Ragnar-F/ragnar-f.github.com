IfInString / IfNotInString

Überprüft, ob eine Variable einen bestimmten String enthält.

IfInString, Variable, Suchstring
IfNotInString, Variable, Suchstring
Position := InStr(Heuhaufen, Nadel [, Zeichengenauigkeit, Startposition]]) ; Siehe InStr()-Funktion für Details.

Parameter

Var

Name der Variable, deren Inhalt nach einer Übereinstimmung durchsucht werden soll.

Suchstring

Der zu suchende Text. Die Übereinstimmung ist erst von der Groß- und Kleinschreibung abhängig, wenn StringCaseSense aktiv ist.

Bemerkungen

Die internen Variablen %A_Space% und %A_Tab% enthalten jeweils ein Leer- und Tabulatorzeichen, die bei der Suche nach solchen Zeichen nützlich sein können.

Neben diesem Befehl kann ein weiterer Befehl folgen. Das heißt, dass beide folgenden Angaben die gleiche Wirkung erzielen:

IfInString, MeineVar, abc, Gosub, Process1
IfInString, MeineVar, abc
    Gosub, Process1

Elemente, die keine namentlichen Befehle sind, werden nicht auf derselben Zeile unterstützt. Zum Beispiel:

IfInString, MeineVar, abc, gefunden := true  ; Ungültig.

Siehe auch

InStr(), RegExMatch(), StringGetPos, StringCaseSense, IfEqual, if Var in/contains Übereinstimmungsliste, if Var between, if Var is Typ, Blöcke, Else

Beispiel

Heuhaufen = abcdefghijklmnopqrs
Nadel = abc
IfInString, Heuhaufen, %Nadel%
{
    MsgBox, Der Text wurde gefunden.
    Return
}
Else
    Sleep, 1
