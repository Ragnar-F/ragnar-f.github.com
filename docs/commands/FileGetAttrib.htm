FileGetAttrib

Kann feststellen, ob eine Datei schreibgeschützt, versteckt und so weiter ist.

FileGetAttrib, Ausgabevariable [, Dateiname]
Attributsstring := FileExist(Dateimuster)

Parameter

Ausgabevariable Der Name der Variable, die den ermittelten Text enthält.
Dateiname Der Name der Datei, die sich im A_WorkingDir-Verzeichnis befindet, solange kein absoluter Pfad angegeben ist. Wenn nicht vorhanden, dann wird die aktuelle Datei in der innersten dateilesende Schleife verwendet.

ErrorLevel

[v1.1.04+] Dieser Befehl ist in der Lage, bei Misserfolg eine Ausnahme auszulösen. Für mehr Informationen, siehe Laufzeitfehler.

Bei Problemen wird ErrorLevel auf 1 gesetzt, ansonsten enthält ErrorLevel eine 0.

Die interne Variable A_LastError enthält den Rückgabewert der Systemfunktion GetLastError().

Bemerkungen

Der zurückgegebene String enthält Buchstaben aus dem String "RASHNDOCT":

R = READONLY (Schreibgeschützt)
A = ARCHIVE (Archiviert)
S = SYSTEM (Systemdatei)
H = HIDDEN (Versteckt)
N = NORMAL
D = DIRECTORY (Verzeichnis)
O = OFFLINE
C = COMPRESSED (Komprimiert)
T = TEMPORARY (Temporär)


Mit dem folgenden Beispiel kann festgestellt werden, ob ein bestimmtes Attribut im String vorhanden ist:

FileGetAttrib, Attribute, C:\Eigene Datei.txt
IfInString, Attribute, H
    MsgBox Die Datei ist versteckt.

Mit dem folgenden Beispiel kann man den 8.3-Kurznamen einer Datei abfragen:

Loop, C:\Eigene Dokumente\Adressenliste.txt
    Kurzpfadname = %A_LoopFileShortPath%  ; Ergebnis: C:\EIGENE~1\ADRESS~1.TXT

Mit einem ähnlichen Verfahren kann der lange Name eines 8.3-Kurznamen ermittelt werden.

Verwandte Befehle

FileExist(), FileSetAttrib, FileGetTime, FileSetTime, FileGetSize, FileGetVersion, dateilesende Schleife

Beispiel

FileGetAttrib, Ausgabevariable, C:\Neuer Ordner
