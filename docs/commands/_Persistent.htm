#Persistent

Hält ein Script dauerhaft aufrecht (bis der Benutzer das Script schließt oder ein ExitApp vorkommt).

#Persistent

Wenn diese Direktive irgendwo im Script vorkommt, dann bleibt das Script aufrecht, nachdem der automatische Ausführungsbereich (oberster Bereich des Scripts) durchgeführt wurde. Das ist besonders nützlich, wenn ein Script Timer und/oder benutzerdefinierte Menüpunkte enthält, aber nicht Hotkeys, Hotstrings, OnMessage() oder Gui.

Wird diese Direktive in einem vorhandenen Script hinzugefügt, sollten einige oder alle Vorkommen von Exit mit ExitApp ersetzt werden. Denn Exit beendet nicht ein persistentes Script, nur den aktuellen Thread.

Seit v1.0.16 wird ein Script mit dieser Direktive zur Einzelinstanz. Um dieses Verhalten zu überschreiben, verwendet #SingleInstance.

Verwandte Befehle

#SingleInstance, SetTimer, Menu, Exit, ExitApp
