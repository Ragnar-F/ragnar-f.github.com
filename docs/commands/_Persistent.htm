#Persistent

Hält ein Script dauerhaft aufrecht (bis der Benutzer das Script schließt oder ein ExitApp vorkommt).

#Persistent

Diese Direktive (egal wo) wird das Script weiterlaufen lassen, wenn es das Ende des automatischen Ausführungsbereichs (oberster Bereich des Scripts) erreicht hat. Dies ist nützlich, wenn ein Script Timer und/oder benutzerdefinierte Menüpunkte enthält, aber nicht Hotkeys, Hotstrings, OnMessage() oder Gui.

Sobald diese Direktive in ein vorhandenes Script eingefügt wird, empfiehlt es sich, einige oder alle Vorkommen von Exit mit ExitApp zu ersetzen. Das Exit würde hierbei nur den aktuellen Thread beenden, nicht das ganze Script.

Seit v1.0.16 macht diese Direktive ein Script zu einer Einzelinstanz. Um dies zu überschreiben oder das Verhalten einer Einzelinstanz zu ändern, siehe #SingleInstance.

Verwandte Befehle

#SingleInstance, SetTimer, Menu, Exit, ExitApp
