EnvSub

Bestimmt eine Variable, die sich selbst mit einem vorgegebenen Wert subtrahiert (kann auch Datum-Zeit-Werte vergleichen). Gleichbedeutend mit: Var -= Wert

EnvSub, Var, Wert [, Zeiteinheit]
Var -= Wert [, Zeiteinheit]
Var--

Parameter

Var Der Name der Variable, die bearbeitet wird.
Wert Beliebiger Integer-Wert, Fließkommazahl oder Ausdruck (Ausdrücke werden nicht unterstützt, wenn Zeiteinheit vorhanden ist).
Zeiteinheit

Falls vorhanden, wird Wert von Var subtrahiert, wobei beide als Datum-Zeit-Stempel im YYYYMMDDHH24MISS-Format behandelt werden. Die Zeiteinheit enthält entweder Seconds, Minutes, Hours oder Days (oder jeweils nur den ersten Buchstaben). Wenn Var leer ist, wird die aktuelle Zeit verwendet. Ist Wert leer, wird ebenfalls die aktuelle Zeit verwendet.

Das Ergebnis wird immer auf die ganze Zahl abgerundet. Wenn beispielsweise die tatsächliche Differenz zwischen zwei Zeitstempeln 1.999 Tage ist, wird die Ausgabe auf 1 Tag gesetzt. Für eine höhere Genauigkeit muss Seconds in der Zeiteinheit verwendet und das Ergebnis durch 60.0, 3600.0 oder 86400.0 geteilt werden.

Wenn Var oder Wert einen ungültigen Zeitstempel oder ein Jahr vor 1601 enthält, wird Var leer gemacht, um auf ein Problem hinzuweisen.

Die interne Variable A_Now enthält die aktuelle Ortszeit im YYYYMMDDHH24MISS-Format.

Zur genauen Bestimmung der verstrichenen Zeit zwischen zwei Ereignissen wird die A_TickCount-Methode verwendet, weil A_TickCount eine Präzision im Millisekundenbereich unterstützt.

Für das Addieren oder Subtrahieren von Sekunden, Minuten, Stunden oder Tagen einer bestimmten Zahl mit einem Zeitstempel wird EnvAdd verwendet (mit einer negativen Zahl wird eine Subtraktion erreicht).

Bemerkungen

Dieser Befehl ist gleichbedeutend mit der Kurzfassung: Var -= Wert

Der Wert einer Variable kann um 1 erhöht oder verringert werden, wenn Var++, Var--, ++Var oder --Var verwendet wird.

Wenn Var oder Wert leer ist oder nicht mit einer Zahl beginnt, wird eine 0 als Berechnungsgrundlage verwendet (außer bei interner Verwendung in Ausdrücken oder bei Verwendung des Zeiteinheit-Parameters).

Wenn Var oder Wert einen Dezimalpunkt enthält, ist das Endergebnis eine Fließkommazahl, abhängig vom eingestellten Format (SetFormat).

Verwandte Befehle

EnvAdd, EnvMult, EnvDiv, SetFormat, Ausdrücke, If Var is [not] Typ, SetEnv, FileGetTime

Beispiel

EnvSub, EigeneZählung, 2
EigeneZählung -= 2  ; das Gleiche wie oben

Var1 = 20050126
Var2 = 20040126
EnvSub, Var1, %Var2%, days
MsgBox, %Var1%  ; Das Ergebnis ist 366, da 2004 ein Schaltjahr ist.
