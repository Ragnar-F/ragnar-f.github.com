EnvSub

Bestimmt eine Variable, die sich selbst mit einem vorgegebenen Wert subtrahiert (kann auch Datum-Zeit-Werte vergleichen). Gleichbedeutend mit: Var -= Wert.

EnvSub, Var, Wert [, Zeiteinheit]
Var -= Wert [, Zeiteinheit]
Var--

Parameter

Var

Name der Variable, die bearbeitet werden soll.

Value

Integer, Fließkommazahl oder Ausdruck (Ausdrücke werden nicht unterstützt, wenn Zeiteinheit vorhanden ist).

Zeiteinheit

Ist dieser Parameter vorhanden, wird der Befehl, während er Wert von Var subtrahiert, beide als Datum-Zeit-Stempel im YYYYMMDDHH24MISS-Format behandeln. Zeiteinheit kann entweder Seconds, Minutes, Hours oder Days sein (oder jeweils nur den ersten Buchstaben). Wenn Wert leer ist, wird die die aktuelle Zeit verwendet. Wenn Var eine leere Variable ist, wird ebenfalls die aktuelle Zeit verwendet.

Das Ergebnis wird immer auf die nächste ganze Zahl abgerundet. Beträgt die Differenz zwischen zwei Zeitstempeln beispielsweise 1.999 Tage, wäre das gemeldete Ergebnis 1 Tag. Benötigt man eine höhere Genauigkeit, sollte man Seconds in Zeiteinheit angeben und das Ergebnis mit 60.0, 3600.0 oder 86400.0 dividieren.

Enthält Var oder Wert einen ungültigen Zeitstempel oder ein Jahr vor 1601, wird Var leer gemacht, um das Problem zu kennzeichnen.

Die integrierte Variable A_Now enthält die aktuelle Ortszeit im YYYYMMDDHH24MISS-Format.

Mit der A_TickCount-Methode kann man genau bestimmen, wieviel Zeit zwischen zwei Ereignissen verstrichen ist, weil sie eine Präzision im Millisekundenbereich ermöglicht.

Mit EnvAdd können Sekunden, Minuten, Stunden oder Tage mit einem Zeitstempel addiert oder subtrahiert werden (mit einer negativen Zahl wird eine Subtraktion erreicht).

Bemerkungen

Dieser Befehl ist gleichbedeutend mit der Kurzfassung: Var -= Wert.

Mit Var++, Var--, ++Var oder --Var können Variablen um 1 erhöht oder verringert werden.

Wenn Var oder Wert leer oder keine Zahl ist, wird es bei der Berechnung als 0 angesehen (außer innerhalb eines Audrucks und beim Verwenden des Zeiteinheit-Parameters).

Enthält Var oder Wert einen Dezimalpunkt, wird das Endergebnis eine Fließkommazahl sein, abhängig vom gesetzten Format per SetFormat.

Siehe auch

EnvAdd, EnvMult, EnvDiv, SetFormat, Ausdrücke, If Var is [not] Typ, SetEnv, FileGetTime

Beispiel

EnvSub, MeineZählung, 2
MeineZählung -= 2  ; das Gleiche wie oben

Var1 = 20050126
Var2 = 20040126
EnvSub, Var1, %Var2%, days
MsgBox, %Var1%  ; Das Ergebnis ist 366, da 2004 ein Schaltjahr ist.
