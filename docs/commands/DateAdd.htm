DateAdd

Addiert oder subtrahiert Zeit von einem Datum-Zeit-Wert.

Ausgabevariable := DateAdd(DatumZeit, Zeit, Zeiteinheit)

Parameter

DatumZeit

Ein Datum-Zeit-Stempel im YYYYMMDDHH24MISS-Format.

Zeit

Die Zeitmenge als Integer oder Fließkommazahl, die addiert wird. Mit einer negativen Zahl wird eine Subtraktion erreicht.

Zeiteinheit

Die Bedeutung des Zeit-Parameters. Die Zeiteinheit enthält entweder Seconds, Minutes, Hours oder Days (oder jeweils nur den ersten Buchstaben).

Rückgabewert

Sobald DatumZeit einen ungültigen Zeitstempel oder ein Jahr vor 1601 enthält, oder Zeit nicht numerisch ist, wird ein leerer String zurückgegeben, um den Fehler zu kennzeichnen.

Ansonsten wird ein Zeitstempel im YYYYMMDDHH24MISS-Format zurückgegeben.

Bemerkungen

Die interne Variable A_Now enthält die aktuelle Ortszeit im YYYYMMDDHH24MISS-Format.

Zur Berechnung der Differenz zwischen zwei Zeitstempeln kann DateDiff verwendet werden.

Verwandte Befehle

DateDiff, FileGetTime, FormatTime

Beispiel

; Das Datum nach genau 31 Tagen berechnen.
Später := DateAdd(A_Now, 31, "days")
MsgBox % FormatTime(Später)
