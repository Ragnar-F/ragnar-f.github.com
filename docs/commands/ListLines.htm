ListLines

Zeigt die aktuell ausgeführten Script-Zeilen an.

ListLines [, On|Off]

Parameter

On|Off

Ist dieser Parameter leer oder nicht vorhanden, wird der Verlauf der aktuell ausgeführten Zeilen angezeigt. Seit v1.0.48.01 ist es möglich, wahlweise On oder Off im ersten Parameter anzugeben. Dies wirkt sich wie folgt nur auf das Verhalten des aktuellen Threads aus:

On: Alle danach ausgeführten Zeilen werden im Verlauf eingefügt. Das ist der Standardwert für alle Scripts.

Off: Alle danach ausgeführten Zeilen werden im Verlauf weggelassen.

Bemerkungen

ListLines (ohne Parameter) ist auch über den Menüpunkt "View->Lines most recently executed" im Hauptfenster aufrufbar. Dieser Befehl kann dabei helfen, ein Script zu debuggen.

Mit ListLines Off/On können bestimmte Zeilen im Verlauf weggelassen werden, um zu verhindern, dass sich der Verlauf zu schnell füllt (z. B. bei einer Schleife mit schnellen Iterationen). Mit ListLines, Off kann die Performance zudem um einige Prozente verbessert werden.

Jeder neu gestartete Thread (z. B. ein Hotkey, ein benutzerdefinierter Menüpunkt oder eine zeitlich festgelegte Subroutine) beginnt mit der Standardeinstellung des Befehls. Diese Standardeinstellung kann geändert werden, indem dieser Befehl im automatischen Ausführungsbereich (am Anfang des Scripts) verwendet wird.

Obwohl die interne Variable "A_ListLines" nicht existiert, kann dennoch eine ähnliche Funktionalität erreicht werden, sobald das folgende Beispiel im Script eingefügt wird:

ListLines(PassTrueToTurnOnOrFalseToTurnOff)  ; Gibt die vorherige Einstellung von ListLines zurück (vor diesem Aufruf).
{
    static sListLines := true  ; Die Standardeinstellung für alle Scripts ist "ListLines On".
    ListLines % PassTrueToTurnOnOrFalseToTurnOff ? "On" : "Off"  ; Führt ListLines bedingungslos aus, um danach ausgeführte Zeilen im Verlauf wegzulassen.
    ListLines_prev := sListLines
    sListLines := PassTrueToTurnOnOrFalseToTurnOff
    Return ListLines_prev
}

; Um die obere Funktion anzuwenden:
prev_ListLines := ListLines(false)  ; ListLines wird vorübergehend deaktiviert.
; ...
ListLines(prev_ListLines)  ; Die vorherige Einstellung von ListLines wird wiederhergestellt.

Beachtet zudem, dass die internen Variablen A_LineNumber und A_LineFile die derzeit ausgeführte Zeilennummer und den dazugehörigen Dateinamen enthalten.

Verwandte Befehle

KeyHistory, ListHotkeys, ListVars

Beispiel

ListLines
ListLines Off
