If Var [not] in/contains Wert1,Wert2,...

Überprüft, ob der Inhalt der Variable mit einem Element aus der Liste übereinstimmt.

If Var in Übereinstimmungsliste
If Var not in Übereinstimmungsliste

If Var contains Übereinstimmungsliste
If Var not contains Übereinstimmungsliste

Parameter

Var

Name der Variable, deren Inhalt überprüft werden soll. Bei dem Operator "in" ist eine exakte Übereinstimmung mit einem Element aus der Liste notwendig. Bei dem Operator "contains" verhält sich die Übereinstimmung toleranter: das heißt, sobald Var ein Element aus der Liste als Teilstring enthält.

Übereinstimmungsliste

Eine kommagetrennte Liste mit Strings, die mit dem Inhalt von Var verglichen werden. Alle Leer- und Tabulatorzeichen an den Trennkommas sind von Bedeutung - das heißt, sie gehören zum übereinstimmenden String. Wenn die Übereinstimmungsliste beispielsweise ABC , XYZ enthält, muss Var entweder ABC mit einem Leerzeichen danach oder XYZ mit einem Leerzeichen davor enthalten, damit eine Übereinstimmung erfolgen kann.

Zwei aufeinanderfolgende Kommas stellen ein literales Komma dar. Im folgenden Beispiel wird ein literales Komma nach dem Wort string1 erzeugt: If Var In string1,,,string2. Ebenso enthält die folgende Liste nur ein einzelnes Element, in dem sich ein literales Komma befindet: If Var In einzelnes,,Element. Um in der Liste ein leeres Element einzufügen, muss das erste Zeichen ein Komma sein: If Var In ,string1,string2 (beim Verwenden des Operators "contains" ist die Übereinstimmung mit einem leeren Element immer erfolgreich, da ein leerer String in jedem String gefunden werden kann).

Da die Elemente in Übereinstimmungsliste nicht als einzelne Parameter behandelt werden, kann sich die komplette Liste in einer Variable befinden. Tatsächlich muss sich alles oder ein Teil davon in der Variable befinden, wenn eine Länge von 16383 Zeichen überschritten wird (die maximale Länge einer Script-Zeile). Zum Beispiel könnte Übereinstimmungsliste aus %Liste1%,%Liste2%,%Liste3% bestehen, von der jede Unterliste eine lange Liste mit Elementen enthält.

Jedes einzelne Element in der Liste, das länger als 16384 Zeichen ist, behandelt die zusätzlichen Zeichen als ein neues Listenelement. Daher sollten solche Elemente am besten vermieden werden.

Bemerkungen

Der Vergleich erfolgt immer alphabetisch, nicht numerisch. Zum Beispiel würde der String "11" nicht mit dem Listenelement "11.0" übereinstimmen.

Der Operator "contains" verhält sich wie IfInString/IfNotInString, aber mit dem Unterschied, dass mehrere Suchstrings unterstützt werden (jeder von denen bewirkt eine Übereinstimmung).

Es kann StringCaseSense On verwendet werden, um den Vergleich abhängig von der Groß- und Kleinschreibung zu machen.

Falls die Übereinstimmungsliste zu lang ist, kann diese in mehreren kurzen Zeilen aufgeteilt werden (Fortsetzungsbereich), um die Lesbarkeit und Wartbarkeit zu verbessern.

Die Operatoren "between", "is", "in" und "contains" werden in Ausdrücken nicht unterstützt.

Beispiele

If Var in exe,bat,com
    MsgBox Die Datei ist ausführbar.

If Var in 2,3,5,7,11 ; Leerzeichen vermeiden.
    MsgBox %Var% ist eine kleine Primzahl.

If Var contains 1,3  ; Die Werte werden als Strings anstatt als Zahlen verglichen.
    MsgBox Var enthält die Zahl 1 oder 3 (Var kann eine 1, 3, 10, 21, 23, etc. sein)

If Var in %Übereinstimmungsliste%
    MsgBox %Var% ist in der Liste.
