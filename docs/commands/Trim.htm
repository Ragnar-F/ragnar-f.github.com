Trim [AHK_L 31+]

Entfernt Zeichen vom Anfang und/oder Ende eines Strings.

Ergebnis :=  Trim(String, ZeichenAuslassen = " `t")
Ergebnis := LTrim(String, ZeichenAuslassen = " `t")
Ergebnis := RTrim(String, ZeichenAuslassen = " `t")

Parameter

String Beliebiger String-Wert oder Variable. Zahlen werden nicht unterstützt.
ZeichenAuslassen Eine optionale Liste von Zeichen (Unterscheidung zwischen Groß- und Kleinschreibung), die am Anfang und Ende im String ignoriert werden sollen. Ist dieser Parameter nicht vorhanden, werden Leer- und Tabulatorzeichen entfernt.

Beispiele

text := "  Text  "
MsgBox % "Ohne Trim:`t '" Text "'"
    . "`nTrim:`t '" Trim(Text) "'"
    . "`nLTrim:`t '" LTrim(Text) "'"
    . "`nRTrim:`t '" RTrim(Text) "'"
MsgBox % LTrim("00000123","0")
