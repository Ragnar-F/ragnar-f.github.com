SoundSetWaveVolume

Ändert die Wave-Wiedergabelautstärke eines Audio-Geräts.

SoundSetWaveVolume, Prozentwert [, Gerätenummer]

Parameter

Prozentwert Eine Prozentzahl zwischen -100 und 100 (kann auch eine Fließkommazahl oder ein Ausdruck sein). Wenn die Zahl mit einem Plus- oder Minuszeichen beginnt, dann wird die aktuelle Lautstärke mit der angegebenen Menge nach oben oder nach unten reguliert. Andernfalls wird die Lautstärke direkt auf dem angegebenen Prozentwert gesetzt.
Gerätenummer Wenn dieser Parameter weggelassen wird, dann standardmäßig eine 1 (das erste Audiogerät), welches in der Regel das Standardgerät für die Aufnahme und Wiedergabe ist. Verwendet eine Zahl größer als 1, um ein anderes Audiogerät zu bearbeiten.

ErrorLevel

[v1.1.04+] Dieser Befehl ist in der Lage, bei Misserfolg eine Ausnahme auszulösen. Für mehr Informationen, siehe Laufzeitfehler.

ErrorLevel wird bei Misserfolg auf 1 und bei Erfolg auf 0 gesetzt.

Bemerkungen

SoundSetWaveVolume wird zurzeit nur von Windows Vista und höher unterstützt, wenn AutoHotkey.exe im XP-Kompatibilitätsmodus läuft (dass wird möglicherweise in einer zukünftigen Version behoben). Eine Möglichkeit, die Gesamtlautstärke des Systems zu ändern:

Send {Volume_Up}  ; Erhöht die Gesamtlautstärke um 1 Stufe (normalerweise 5%).
Send {Volume_Down 3}  ; Verringert die Gesamtlautstärke um 3 Stufen.
Send {Volume_Mute}  ; Gesamtlautstärke stumm schalten bzw. Stummschaltung aufheben.

Die derzeitige Wave-Wiedergabelautstärke kann mit SoundGetWaveVolume ermittelt werden. Einstellungen wie Gesamtlautstärke, Synthesizer, Mikrofon, Stummschaltung, Höhen und Tiefen können mithilfe von SoundSet und SoundGet gesetzt oder abgerufen werden.

Im Gegensatz zu SoundSet versucht dieser Befehl, die Balance zwischen den Kanälen (z. B. links und rechts) zu bewahren, wenn die Lautstärke geändert wird.

Verwandte Befehle

SoundGetWaveVolume, SoundSet, SoundGet, SoundPlay

Beispiele

SoundSetWaveVolume, 50  ; Setzt auf halbe Lautstärke.
SoundSetWaveVolume, -10  ; Verringert die Lautstärke um 10 (z. B. 80 wird zu 70).
SoundSetWaveVolume, +20  ; Erhöht die Lautstärke um 20.
