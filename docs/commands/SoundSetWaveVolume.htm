SoundSetWaveVolume

Ändert die Wave-Wiedergabelautstärke eines Audio-Geräts.

SoundSetWaveVolume, Prozentwert [, Gerätenummer]

Parameter

Prozentwert Eine Prozentzahl zwischen -100 und 100 (kann eine Fließkommazahl oder ein Ausdruck sein). Wenn die Zahl mit einem Plus- oder Minuszeichen beginnt, wird die aktuelle Einstellung mit der angegebenen Menge nach oben oder nach unten reguliert. Andernfalls wird der Prozentwert direkt gesetzt.
Gerätenummer Ist dieser Parameter nicht vorhanden, gilt standardmäßig eine 1 (das erste Audiogerät), das für gewöhnlich das Standardgerät des Systems für Aufnahme und Wiedergabe ist. Mit einer Zahl größer als 1 kann ein anderes Audiogerät bearbeitet werden.

ErrorLevel

[v1.1.04+] Dieser Befehl ist in der Lage, bei Misserfolg eine Ausnahme auszulösen. Für mehr Informationen, siehe Laufzeitfehler.

ErrorLevel wird bei Misserfolg auf 1 und bei Erfolg auf 0 gesetzt.

Bemerkungen

SoundSetWaveVolume funktioniert nur in Windows Vista und höher, wenn AutoHotkey.exe im XP-Kompatibilitätsmodus ausgeführt wird (das wird möglicherweise in einer zukünftigen Version behoben). Die Gesamtlautstärke des Systems kann zum Beispiel wie folgt geändert werden:

Send {Volume_Up}  ; Erhöht die Gesamtlautstärke um 1 Stufe (normalerweise 5%).
Send {Volume_Down 3}  ; Verringert die Gesamtlautstärke um 3 Stufen.
Send {Volume_Mute}  ; Gesamtlautstärke stumm schalten bzw. Stummschaltung aufheben.

Die derzeitige Wave-Wiedergabelautstärke kann mit SoundGetWaveVolume abgerufen werden. Einstellungen wie Gesamtlautstärke, Synthesizer, Mikrofon, Stummschaltung, Höhe und Bass können mithilfe von SoundSet und SoundGet gesetzt oder abgerufen werden.

Im Gegensatz zu SoundSet versucht dieser Befehl, die vorhandene Balance zwischen den Kanälen (z. B. links und rechts) beim Ändern der Lautstärke beizubehalten.

Verwandte Befehle

SoundGetWaveVolume, SoundSet, SoundGet, SoundPlay

Beispiele

SoundSetWaveVolume, 50  ; Setzt auf halbe Lautstärke.
SoundSetWaveVolume, -10  ; Verringert die Lautstärke um 10 (z. B. 80 wird zu 70).
SoundSetWaveVolume, +20  ; Erhöht die Lautstärke um 20.
