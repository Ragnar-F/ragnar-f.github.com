#MaxHotkeysPerInterval

Zusammen mit #HotkeyInterval kann die Frequenz der Hotkey-Aktivierungen angegeben werden, bei dessen Überschreitung eine Warnmeldung angezeigt wird.

#MaxHotkeysPerInterval Wert

Parameter

Wert Die maximale Anzahl an Hotkeys, die im Intervall (durch #HotkeyInterval angegeben) gedrückt werden können, ohne dabei eine Fehlermeldung auszulösen.

Bemerkungen

Es sollte darauf geachtet werden, nicht zu nachsichtig mit dem oben genannten Wert zu sein: Solltest du aus Versehen eine Endlosschleife mit Tastatureingaben erzeugen (per Send-Befehl, der versehentlich andere Hotkeys auslöst), kann dein Computer aufgrund der Flut von Tastaturereignissen möglicherweise nicht mehr reagieren.

Zum Beispiel würde der Hotkey ^c::Send ^c eine Endlosschleife mit Tastatureingaben erzeugen. Um das zu verhindern, kann das $-Präfix zur Hotkey-Definition hinzugefügt werden (z. B. $^c::), so dass der Hotkey nicht durch den Send-Befehl ausgelöst werden kann.

Wenn diese Direktive nicht angegeben ist, gilt standardmäßig 70.

Verwandte Befehle

#HotkeyInterval

Beispiel

#MaxHotkeysPerInterval 200
