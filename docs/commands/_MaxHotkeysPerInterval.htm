#MaxHotkeysPerInterval

Zusammen mit #HotkeyInterval kann die Frequenz der Hotkey-Aktivierungen angegeben werden, bei dessen Überschreitung eine Warnmeldung angezeigt wird.

#MaxHotkeysPerInterval Wert

Parameter

Wert Die maximale Anzahl an Hotkeys, die im Intervall (bestimmt durch #HotkeyInterval) gedrückt werden können, ohne dabei eine Fehlermeldung auszulösen.

Bemerkungen

Achtet darauf, nicht zu nachsichtig mit dem Wert zu sein, weil dadurch versehentlich eine Endlosschleife von Tastatureingaben entstehen kann (durch einen Send-Befehl, der versehentlich andere Hotkeys auslöst) und folglich das System nicht mehr durch die hohe Anzahl an Tastaturereignissen reagieren kann.

Zum Beispiel erzeugt der Hotkey ^c::Send ^c eine Endlosschleife von Tastatureingaben. Um das zu vermeiden, fügt das $-Präfix zur Hotkey-Definition hinzu (z. B. $^c::), damit der Hotkey nicht vom Send-Befehl ausgelöst wird.

Wenn diese Direktive nicht angegeben ist, dann wird standardmäßig 70 verwendet.

Verwandte Befehle

#HotkeyInterval

Beispiel

#MaxHotkeysPerInterval 200
