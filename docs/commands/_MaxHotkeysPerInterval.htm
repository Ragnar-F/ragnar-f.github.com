#MaxHotkeysPerInterval

Zusammen mit #HotkeyInterval kann festgelegt werden, wie viele Hotkeys innerhalb eines Zeitrahmens aktiviert werden können, bevor eine Warnmeldung erscheint.

#MaxHotkeysPerInterval Wert

Parameter

Wert Die maximale Anzahl an Hotkeys, die im Intervall (durch #HotkeyInterval angegeben) gedrückt werden können, ohne dabei eine Fehlermeldung auszulösen.

Bemerkungen

Man sollte darauf achten, nicht zu nachsichtig mit dem oben genannten Wert zu sein: Erzeugt man aus Versehen eine Endlosschleife mit Tastatureingaben (per Send-Befehl, der versehentlich andere Hotkeys auslöst), könnte es passieren, dass der Computer aufgrund von vielen schnellen Tastaturereignissen nicht mehr reagieren kann.

Der Hotkey ^c::Send ^c würde beispielsweise eine Endlosschleife von Tastatureingaben erzeugen. Dieses Verhalten kann unterdrückt werden, wenn man das $-Präfix in die Hotkey-Definition einfügt (z. B. $^c::), so dass der Hotkey nicht mehr durch den Send-Befehl ausgelöst werden kann.

Ist diese Direktive im Script nicht angegeben, gilt standardmäßig 70.

Verwandte Befehle

#HotkeyInterval

Beispiel

#MaxHotkeysPerInterval 200
