If/IfEqual/IfNotEqual/IfLess/IfLessOrEqual/IfGreater/IfGreaterOrEqual

Befehl(e), die ausgeführt werden, wenn der Vergleich einer Variable mit einem Wert WAHR ist. Sobald mehrere Befehle vorhanden sind, müssen sie von einem Block (geschweifte Klammern) umschlossen werden.

IfEqual, Variable, Wert (If Variable = Wert)
IfNotEqual, Variable, Wert (If Variable <> Wert) (!= kann statt <> verwendet werden)
IfGreater, Variable, Wert (If Variable > Wert)
IfGreaterOrEqual, Variable, Wert (If Variable >= Wert)
IfLess, Variable, Wert (If Variable < Wert)
IfLessOrEqual, Variable, Wert (If Variable <= Wert)
If Variable ; Wenn der Inhalt von Variable leer oder 0 ist, dann wird es als falsch angesehen. Ansonsten ist es wahr.

Siehe auch: IfInString

Parameter

Variable Der Name der Variable.
Wert Direkter String, direkte Zahl oder Variablenverweis (z. B. %Variable2%). Der Wert kann weggelassen werden, wenn die Variable mit einem leeren String verglichen werden soll.

Bemerkungen

Falls sowohl die Variable als auch der Wert numerisch sind, werden sie als Zahlen anstatt als Strings verglichen. Ansonsten werden sie als Strings alphabetisch verglichen (d. h. die alphabetische Reihenfolge bestimmt, ob die Variable größer, gleich oder kleiner als der Wert ist).

Sobald IF oder ELSE mehrere Zeilen enthält, müssen diese Zeilen mit Klammern umschlossen werden. Zum Beispiel:

If count <= 0
{
    WinClose Unbenannt - Editor
    MsgBox Keine Elemente vorhanden.
}

Wenn allerdings nur eine Zeile vorhanden ist, sind die Klammern optional.

Ein anderer Befehl kann nur auf derselben Zeile der IF-Anweisung stehen, wenn der Befehlsnamen-Style verwendet wird. Mit anderen Worten sind diese Beispiele gültig:

IfEqual, x, 1, Sleep, 1
IfGreater, x, 1, EnvAdd, x, 2

Aber diese ungültig:

If x = 1 Sleep 1
IfGreater, x, 1, x += 2

Der "One True Brace Style" kann bei solchen If-Anweisungen nicht verwendet werden. Dieser ist nur bei If-Ausdrücken gültig.

Auf ähnliche Weise kann mit der Anweisung If Variable [not] between Untergrenze and Obergrenze überprüft werden, ob eine Variable zwischen zwei Werten liegt, sowie mit If Variable [not] in Wert1,Wert2 überprüft werden, ob der Variableninhalt in einer Werteliste vorkommt.

Verwandte Befehle

IF (Ausdruck), StringCaseSense, Zuweisungsausdruck (:=), If Variable in/contains Übereinstimmungsliste, If Variable between, IfInString, Blocks, Else

Beispiel

If counter >= 1
    Sleep, 10
