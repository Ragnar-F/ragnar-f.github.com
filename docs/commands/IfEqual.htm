If/IfEqual/IfNotEqual/IfLess/IfLessOrEqual/IfGreater/IfGreaterOrEqual

Befehl(e), die ausgeführt werden, wenn der Vergleich einer Variable mit einem Wert WAHR ist. Wenn mehrere Befehle vorhanden sind, dann müssen sie von einem Block (geschweifte Klammern) umschlossen werden.

IfEqual, Variable, Wert (If Variable = Wert)
IfNotEqual, Variable, Wert (If Variable <> Wert) (!= kann statt <> verwendet werden)
IfGreater, Variable, Wert (If Variable > Wert)
IfGreaterOrEqual, Variable, Wert (If Variable >= Wert)
IfLess, Variable, Wert (If Variable < Wert)
IfLessOrEqual, Variable, Wert (If Variable <= Wert)
If Variable ; Wenn der Inhalt von Variable leer oder 0 ist, dann wird es als falsch angesehen. Ansonsten ist es wahr.

Siehe auch: IfInString

Parameter

Variable Der Name der Variable.
Wert Direkt vorhandener String, Zahl oder Variablenreferenz (z. B. %Variable2%). Der Wert kann weggelassen werden, wenn die Variable mit einem leeren String verglichen werden soll.

Bemerkungen

Wenn sowohl die Variable als auch der Wert numerisch sind, dann werden sie als Zahlen anstatt als Strings verglichen. Ansonsten werden sie alphabetisch als Strings verglichen (d. h. die alphabetische Reihenfolge bestimmt, ob die Variable größer, gleich oder kleiner als der Wert ist).

Wenn IF oder ELSE mehrere Zeilen enthält, dann müssen diese Zeilen mit Klammern umschlossen werden. Zum Beispiel:

If count <= 0
{
    WinClose Unbenannt - Editor
    MsgBox Keine Elemente vorhanden.
}

Wenn allerdings nur eine Zeile vorhanden ist, dann sind die Klammern optional.

Ein anderer Befehl kann nur auf derselben Zeile der IF-Anweisung stehen, wenn der Befehlsnamen-Style verwendet wird. Mit anderen Worten sind diese gültig:

IfEqual, x, 1, Sleep, 1
IfGreater, x, 1, EnvAdd, x, 2

Aber diese sind nicht gültig:

If x = 1 Sleep 1
IfGreater, x, 1, x += 2

Der One True Brace (OTB)-Style kann nicht mit dieser Art von If-Anweisung verwendet werden. OTB kann nur mit If-Ausdrücken verwendet werden.

Ein weiterer Befehl If Variable [not] between Untergrenze and Obergrenze überprüft, ob eine Variable zwischen zwei Werten liegt. If Variable [not] in Wert1,Wert2 überprüft, ob der Inhalt einer Variable in einer Liste von Werten vorhanden ist.

Verwandte Befehle

IF (Ausdruck), StringCaseSense, Zuweisungsausdruck (:=), If Variable in/contains Übereinstimmungsliste, If Variable between, IfInString, Blocks, Else

Beispiel

If counter >= 1
    Sleep, 10
