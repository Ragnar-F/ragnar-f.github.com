If/IfEqual/IfNotEqual/IfLess/IfLessOrEqual/IfGreater/IfGreaterOrEqual

Ein oder mehrere Befehle, die ausgeführt werden, wenn der Vergleich einer Variable mit einem Wert WAHR ist. Sobald mehrere Befehle vorhanden sind, müssen sie von einem Block (geschweifte Klammern) umschlossen werden.

IfEqual, Var, Wert (If Var = Wert)
IfNotEqual, Var, Wert (If Var <> Wert) (!= kann statt <> verwendet werden)
IfGreater, Var, Wert (If Var > Wert)
IfGreaterOrEqual, Var, Wert (If Var >= Wert)
IfLess, Var, Wert (If Var < Wert)
IfLessOrEqual, Var, Wert (If Var <= Wert)
If Var ; Wenn der Inhalt von Var leer oder 0 ist, wird dieser als falsch angesehen. Ansonsten ist dieser wahr.

Siehe auch: IfInString

Parameter

Var Name der Variable.
Wert Direkter String, direkte Zahl oder Variablenverweis (z. B. %Variable2%). Der Wert kann weggelassen werden, wenn Var mit einem leeren String verglichen werden soll.

Bemerkungen

Sobald Var und Wert numerisch sind, werden sie als Zahlen verglichen. Ansonsten werden sie als Strings alphabetisch verglichen (das heißt, dass die alphabetische Reihenfolge bestimmt, ob Var größer, gleich oder kleiner als Wert ist).

Sobald IF oder ELSE mehrere Zeilen enthält, müssen diese Zeilen mit Klammern umschlossen werden. Zum Beispiel:

If Anzahl <= 0
{
    WinClose Unbenannt - Editor
    MsgBox Keine Elemente vorhanden.
}

Wenn allerdings nur eine Zeile vorhanden ist, sind die Klammern optional.

Ein anderer Befehl kann nur auf derselben Zeile der IF-Anweisung stehen, wenn der Befehlsnamen-Style verwendet wird. Mit anderen Worten sind folgende Beispiele gültig:

IfEqual, x, 1, Sleep, 1
IfGreater, x, 1, EnvAdd, x, 2

Aber diese ungültig:

If x = 1 Sleep 1
IfGreater, x, 1, x += 2

Der "One True Brace Style" kann bei solchen If-Anweisungen nicht verwendet werden. Dieser ist nur bei If-Ausdrücken gültig.

Auf ähnliche Weise kann mit der Anweisung If Var [not] between Untergrenze and Obergrenze überprüft werden, ob eine Variable zwischen zwei Werten liegt, sowie mit If Var [not] in Wert1,Wert2 überprüft werden, ob der Variableninhalt in einer Werteliste vorkommt.

Verwandte Befehle

IF (Ausdruck), StringCaseSense, Zuweisungsausdruck (:=), If Var in/contains Übereinstimmungsliste, If Var between, IfInString, Blocks, Else

Beispiel

If Zähler >= 1
    Sleep, 10
