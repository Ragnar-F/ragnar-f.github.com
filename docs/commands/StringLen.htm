StrLen / StringLen

Ermittelt die Anzahl an Zeichen, die sich in einem String befinden.

AusgabeVar := StrLen(EingabeVar)
StringLen, AusgabeVar, EingabeVar

Parameter

AusgabeVar

Name der Variable, in der die Länge gespeichert werden soll.

EingabeVar

Name der Variable, deren Inhalt gemessen werden soll. Umschließe den Namen nicht mit Prozentzeichen, es sei denn, du willst den Inhalt der Variable als Namen verwenden.

Bemerkungen

Ist EingabeVar eine Variable, der ClipboardAll zugewiesen wurde, wird StringLen deren Gesamtgröße melden.

Siehe auch

IfInString, StringGetPos, StringMid, StringTrimLeft, StringTrimRight, StringLeft, StringRight, StringLower, StringUpper, StringReplace

Beispiel

EingabeVar := "Franz jagt im komplett verwahrlosten Taxi quer durch Bayern."
MsgBox % "EingabeVar ist " StrLen(EingabeVar) " Zeichen lang." ; Ergebnis: 60
StringLen, Länge, EingabeVar
MsgBox, EingabeVar ist %Länge% Zeichen lang.
