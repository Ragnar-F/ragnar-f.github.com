StrLen / StringLen

Ermittelt die Anzahl an Zeichen, die sich in einem String befinden.

Ausgabevariable := StrLen(Eingabevariable)
StringLen, Ausgabevariable, Eingabevariable

Parameter

Ausgabevariable Name der Variable, in der die Länge gespeichert werden soll.
Eingabevariable Name der Variable, deren Inhalt gemessen werden soll. Der Name sollte nicht mit Prozentzeichen umschlossen werden, sofern nicht die Absicht besteht, den Inhalt der Variable als Namen zu verwenden.

Bemerkungen

Ist Eingabevariable eine Variable, der ClipboardAll zugewiesen wurde, wird StringLen deren Gesamtgröße melden.

Verwandte Befehle

IfInString, StringGetPos, StringMid, StringTrimLeft, StringTrimRight, StringLeft, StringRight, StringLower, StringUpper, StringReplace

Beispiel

Eingabevariable := "Franz jagt im komplett verwahrlosten Taxi quer durch Bayern."
MsgBox % "Eingabevariable ist " StrLen(Eingabevariable) " Zeichen lang." ; Ergebnis: 60
StringLen, Länge, Eingabevariable
MsgBox, Eingabevariable ist %Länge% Zeichen lang.
