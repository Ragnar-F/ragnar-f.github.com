StringLen

Ermittelt die Anzahl an Zeichen, die sich in einem String befinden.

StringLen, Ausgabevariable, Eingabevariable
Länge := StrLen(String)

Parameter

Ausgabevariable Name der Variable, in der die Länge gespeichert wird.
Eingabevariable Name der Variable, deren Inhalt gemessen wird. Der Name sollte nicht mit Prozentzeichen umschlossen werden, sofern nicht die Absicht besteht, den Inhalt der Variable als Namen zu verwenden.

Bemerkungen

Ist Eingabevariable eine Variable, der ClipboardAll zugewiesen wurde, wird StringLen deren Gesamtgröße melden.

Verwandte Befehle

StrLen(), IfInString, StringGetPos, StringMid, StringTrimLeft, StringTrimRight, StringLeft, StringRight, StringLower, StringUpper, StringReplace

Beispiel

StringLen, Länge, Eingabevariable
MsgBox, Die Länge der Eingabevariable ist %Länge%.

MsgBox % "Die Länge von EigeneVar ist " . StrLen(EigeneVar)
