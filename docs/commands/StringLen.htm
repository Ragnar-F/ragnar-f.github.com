StringLen

Ermittelt die Anzahl an Zeichen, die sich im String befinden.

StringLen, Ausgabevariable, Eingabevariable
Länge := StrLen(String)

Parameter

Ausgabevariable Der Name der Variable, in der die Länge gespeichert wird.
Eingabevariable Name der Variable, dessen Inhalt abgemessen wird. Umschließt den Namen erst mit Prozentzeichen, wenn der Inhalt der Variable als Name verwendet werden soll.

Bemerkungen

Wenn die Eingabevariable eine Variable ist, der vorher ClipboardAll zugewiesen wurden ist, dann meldet StringLen deren gesamte Größe.

Verwandte Befehle

StrLen(), IfInString, StringGetPos, StringMid, StringTrimLeft, StringTrimRight, StringLeft, StringRight, StringLower, StringUpper, StringReplace

Beispiel

StringLen, Länge, Eingabevariable
MsgBox, Die Länge der Eingabevariable ist %Länge%.

MsgBox % "Die Länge von EigeneVar ist " . StrLen(EigeneVar)
