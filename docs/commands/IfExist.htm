IfExist / IfNotExist

Überprüft, ob eine Datei oder ein Ordner vorhanden ist.

IfExist, Dateimuster
IfNotExist, Dateimuster
AttributeString := FileExist(Dateimuster)

Parameter

Dateimuster

Der Pfad, Dateiname oder Dateimuster. Dateimuster befindet sich im A_WorkingDir-Verzeichnis, solange kein absoluter Pfad angegeben ist.

Siehe auch

FileExist(), Blöcke, Else, Loop (Dateien & Ordner)

Beispiele

IfExist, D:\
    MsgBox, Das Laufwerk existiert.
IfExist, D:\Dokumente\*.txt
    MsgBox, Mindestens eine TXT-Datei existiert.
IfNotExist, C:\Temp\FlagFile.txt
    MsgBox, Die Zieldatei existiert nicht.
