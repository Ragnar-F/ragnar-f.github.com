#EscapeChar (und die Erläuterung zu Escapesequenzen)

Ändert das Escapezeichen des Scripts (z. B. Akzent zu Backslash).

#EscapeChar NeuesZeichen

Parameter

NeuesZeichen Ein beliebiges Zeichen.

Bemerkungen

Mit dem Escapezeichen wird kenntlich gemacht, dass das nachfolgende Zeichen anders als normal interpretiert werden soll.

Das Standard-Escapezeichen ist ein Akzent/Backtick (`).

Escapesequenzen (wenn ein Akzent das Escapezeichen ist)

Schreibe das um das zu erhalten
`, , (literales Komma). Hinweis: Kommas, die im letzten Parameter eines erscheinen, müssen nicht mit Escapezeichen versehen werden, weil das Programm weiß, dass diese Kommas literal behandelt werden sollen. Das gleiche gilt auch für alle Parameter der MsgBox, weil es eine intelligente Komma-Handhabung hat.
`% % (literales Prozentzeichen)
`` ` (literales Akzent; d. h. zwei aufeinanderfolgende Escapezeichen ergeben ein einzelnes literales Zeichen)
`;  ; (literales Semikolon). Hinweis: Das ist nur erforderlich, wenn auf der linken Seite des Semikolons ein Leer- oder Tabulatorzeichen vorkommt. Wenn dies nicht der Fall ist, wird das Semikolon auch ohne Escapezeichen richtig erkannt.
`:: :: (literales Doppelpunkt-Paar). Seit v1.0.40 muss dieses Paar nicht länger mit einem Escapezeichen versehen werden.
`n Neuzeile (Zeilenvorschub/LF)
`r Zeilenumschaltung (CR)
`b Rücktaste
`t Tabulator (die horizontale Variante)
`v vertikaler Tabulator - entspricht dem ASCII-Wert 11. Kann auch bei einigen Anwendungen mit STRG+K erzeugt werden.
`a Alarm (Glocke) - entspricht dem ASCII-Wert 7. Kann auch bei einigen Anwendungen mit STRG+G erzeugt werden.
`f Seitenvorschub - entspricht dem ASCII-Wert 12. Kann auch bei einigen Anwendungen mit STRG+L erzeugt werden.
Send Wenn Send-Befehle oder Hotstrings in ihrem normalen (nicht-raw) Modus verwendet werden, haben Zeichen wie {}^!+# eine besondere Bedeutung. Wenn solche Zeichen daher literal sein sollen, müssen sie in Klammern gesetzt werden. Zum Beispiel: Send {^}{!}{{}.
"" Um innerhalb eines literalen Strings, der sich in einem Ausdruck befindet, ein Anführungszeichen darstellen zu können, müssen zwei aufeinanderfolgende Anführungszeichen angegeben werden. Zum Beispiel: Var := "Die Farbe ""Rot"" wurde gefunden.".

Verwandte Befehle

Die folgenden Direktiven können auch verwendet werden:

#DerefChar #  ; Ändert das Standard-Derefenzierungszeichen %.
#Delimiter /  ; Ändert das Standard-Trennungszeichen Komma.

Beispiel

#EscapeChar \  ; Ändert das standardmäßig verwendete Akzent (`) in ein Backslash.
