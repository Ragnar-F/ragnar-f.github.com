FileCreateShortcut

Erstellt eine Verknüpfung (.lnk).

FileCreateShortcut, Ziel, Verknüpfung [, Arbeitsverzeichnis, Parameter, Beschreibung, Icondatei, Tastenkürzel, Iconnummer, Ausführungsstatus]

Parameter

Ziel Name der zu verknüpfenden Datei, die nur einen absoluten Pfad beinhalten sollte, wenn sie im System integriert ist (z. B. Notepad.exe). Die Datei muss nicht beim Erstellen der Verknüpfung vorhanden sein. Das heißt, dass Verknüpfungen mit ungültigem Ziel erstellt werden können.
Verknüpfung Name der Verknüpfung, die sich im A_WorkingDir-Verzeichnis befindet, solange kein absoluter Pfad angegeben ist. Die Erweiterung .lnk muss mit angegeben werden. Falls die Datei bereits vorhanden ist, wird sie überschrieben.
Arbeitsverzeichnis Verzeichnis, dass das Arbeitsverzeichnis von Ziel beim Ausführen der Verknüpfung sein wird. Ist dieser Parameter leer oder nicht vorhanden, wird die Verknüpfung ein leeres "Ausführen in"-Feld haben und der Verknüpfung ein Standard-Arbeitsverzeichnis vom System bereitgestellt, sobald sie ausgeführt wird.
Parameter Parameter, die beim Ausführen der Verknüpfung ans Ziel übergeben werden. Parameter werden mit Leerzeichen getrennt. Wenn ein Parameter Leerzeichen enthält, umschließt diesen mit Anführungszeichen.
Beschreibung Kommentare, die die Verknüpfung beschreiben (wird beispielsweise zum Anzeigen des ToolTips verwendet).
Icondatei Der vollständige Pfad und Name des Icons, das für die Verknüpfung angezeigt wird. Entweder eine ICO-Datei oder das erste Icon einer EXE- oder DLL-Datei.
Tastenkürzel

Buchstabe, Zahl oder Name einer einzelnen Taste aus der Tastenliste (Maustasten und andere untypische Tasten werden möglicherweise nicht unterstützt). Modifikatorzeichen können nicht verwendet werden. Zurzeit werden alle Tastenkürzel mit Strg+Alt kombiniert. Wenn zum Beispiel der Buchstabe B für diesen Parameter verwendet wird, gilt die Tastenkombination Strg+Alt+B.

Für Windows 9x: Möglicherweise ist ein Neustart notwendig, um solche Tastenkombinationen wirksam zu machen. Alternativ kann das Eigenschaftsfenster der Verknüpfung geöffnet und das Tastenkürzel neu erstellt werden, um es sofort wirksam zu machen.
Iconnummer Um anstelle des ersten Icons ein beliebiges Icon in Icondatei zu verwenden, kann hier die Nummer angegeben werden (kann ein Ausdruck sein). Zum Beispiel ist 2 das zweite Icon.
Ausführungsstatus

Damit das Ziel minimiert oder maximiert ausgeführt wird, kann eine der folgenden Zahlen verwendet werden:

1 - Normal (Standard)
3 - Maximiert
7 - Minimiert

ErrorLevel

[v1.1.04+] Dieser Befehl ist in der Lage, bei Misserfolg eine Ausnahme auszulösen. Für mehr Informationen, siehe Laufzeitfehler.

ErrorLevel wird bei Misserfolg auf 1 und bei Erfolg auf 0 gesetzt.

Bemerkungen

Das Ziel muss nicht unbedingt ein Pfad beinhalten, wenn sich die Zieldatei in einem Ordner aus der PATH-Umgebungsvariable befindet.

Das Tastenkürzel einer neu erstellten Verknüpfung ist nur wirksam, wenn die Verknüpfung auf dem Desktop oder irgendwo im Startmenü vorhanden ist. Falls das ausgewählte Tastenkürzel bereits verwendet wird, hat die Verknüpfung des Benutzers Vorrang.

Eine URL-Verknüpfung kann auch wie folgt erstellt werden. Passt die ersten beiden Parameter je nach Bedarf an:

IniWrite, http://www.google.com, C:\Eigene Verknüpfung.url, InternetShortcut, URL

Mit dem folgenden Beispiel kann noch ein Icon zugewiesen werden:

IniWrite, <Icondatei>, C:\Eigene Verknüpfung.url, InternetShortcut, IconFile
IniWrite, 0, C:\Eigene Verknüpfung.url, InternetShortcut, IconIndex

Ersetzt die 0 mit dem Index des gewünschten Icons (0 ist das erste Icon) und ersetzt <Icondatei> mit einer URL-, EXE-, DLL- oder ICO-Datei. Beispiele: C:\Icons.dll, C:\App.exe, http://www.somedomain.com/ShortcutIcon.ico

Das Betriebssystem behandelt eine auf diese Weise erstellte URL-Datei wie eine richtige Verknüpfung, selbst wenn es eine Klartextdatei anstelle einer .LNK-Datei ist.

Verwandte Befehle

FileGetShortcut, FileAppend

Beispiel

; Der Buchstabe "i" im letzten Parameter ermöglicht die Tastenkombination Strg+Alt+I:
FileCreateShortcut, Notepad.exe, %A_Desktop%\Eigene Verknüpfung.lnk, C:\, "%A_ScriptFullPath%", Eigene Beschreibung, C:\Eigenes Icon.ico, i
