FileCreateShortcut

Erstellt eine Verknüpfungsdatei (.lnk).

FileCreateShortcut, Ziel, Verknüpfungsdatei [, Arbeitsverzeichnis, Parameter, Beschreibung, Icondatei, Tastenkürzel, Iconnummer, Ausführungsstatus]

Parameter

Ziel Name der zu verknüpfenden Datei, die nur einen absoluten Pfad beinhalten sollte, wenn sie im System integriert ist (z. B. Notepad.exe). Die Datei muss nicht vorhanden sein, um eine Verknüpfung zu erstellen. Mit anderen Worten können Verknüpfungen mit einem ungültigen Ziel erstellt werden.
Verknüpfungsdatei Name der Verknüpfung, die sich im A_WorkingDir-Verzeichnis befindet, solange kein absoluter Pfad angegeben ist. Die .lnk-Erweiterung muss mit angegeben werden. Wenn die Datei bereits vorhanden ist, dann wird sie überschrieben.
Arbeitsverzeichnis Verzeichnis, dass zum Arbeitsverzeichnis vom Ziel wird, wenn die Verknüpfung ausgeführt wird. Wenn leer oder nicht vorhanden, dann enthält die Verknüpfung ein leeres "Ausführen in"-Feld und das System verweist auf ein Standard-Arbeitsverzeichnis, wenn die Verknüpfung ausgeführt wird.
Parameter Parameter, die ans Ziel übergeben werden, wenn die Verknüpfung ausgeführt wird. Parameter werden mit Leerzeichen getrennt. Wenn ein Parameter Leerzeichen enthält, umschließt diesen mit Anführungszeichen.
Beschreibung Kommentare, die die Verknüpfung beschreibt (wird vom Betriebssystem verwendet, um einen ToolTip darzustellen, usw.)
Icondatei Der vollständige Pfad und Name des Icons für die Verknüpfungsdatei. Kann entweder eine ICO-Datei oder das erste Icon einer EXE- oder DLL-Datei enthalten.
Tastenkürzel

Ein Buchstabe, eine Zahl oder der Name einer Taste aus der Tastenliste (Maustasten und andere Nicht-Standard-Tasten werden möglicherweise nicht unterstützt). Modifikatorsymbole werden nicht unterstützt. Zurzeit werden alle Tastenkombinationen mit STRG+ALT erstellt. Wenn zum Beispiel der Buchstabe B für diesen Parameter angegeben ist, dann lautet die Tastenkombination STRG+ALT+B.

Für Windows 9x: Ein Neustart ist vielleicht notwendig, damit solche Tastenkombinationen in Kraft treten. Alternativ kann man die Tastenkombination auch über das Eigenschaftsfenster erneuern, damit diese sofort in Kraft tritt.
Iconnummer Für die Verwendung eines beliebigen Icons in der Icondatei muss die Positionsnummer hier eingetragen werden (kann ein Ausdruck sein). Zum Beispiel eine 2 für das zweite Icon.
Ausführungsstatus

Um das Ziel minimiert oder maximiert zu starten, fügt eine folgende Zahl ein:

1 - Normal (Standard)
3 - Maximiert
7 - Minimiert

ErrorLevel

[v1.1.04+] Dieser Befehl ist in der Lage, bei Misserfolg eine Ausnahme auszulösen. Für mehr Informationen, siehe Laufzeitfehler.

Bei Problemen wird ErrorLevel auf 1 gesetzt, ansonsten enthält ErrorLevel eine 0.

Bemerkungen

Das Ziel muss nicht unbedingt ein Pfad beinhalten, wenn die Zieldatei sich in einem der Ordner in der Umgebungsvariable PATH befindet.

Das Tastenkürzel einer neu erstellten Verknüpfung ist nur wirksam, wenn die Verknüpfungsdatei sich auf dem Desktop oder irgendwo im Startmenü befindet. Wenn das ausgewählte Tastenkürzel bereits verwendet wird, dann übernimmt die neue Verknüpfung des Benutzers Vorrang.

Alternativ kann eine URL-Verknüpfung auch wie im folgenden Beispiel erstellt werden. Die ersten beiden Parameter je nach Belieben anpassen:
IniWrite, http://www.google.com, C:\Eigene Verknüpfung.url, InternetShortcut, URL

Mit dem folgenden Beispiel kann noch ein Icon zugewiesen werden:
IniWrite, <Icondatei>, C:\Eigene Verknüpfung.url, InternetShortcut, IconFile
IniWrite, 0, C:\Eigene Verknüpfung.url, InternetShortcut, IconIndex

Ersetzt die 0 mit dem Index des gewünschten Icons (0 ist das erste Icon) und ersetzt <Icondatei> mit einer URL-, EXE-, DLL- oder ICO-Datei. Beispiele: C:\Icons.dll, C:\App.exe, http://www.somedomain.com/ShortcutIcon.ico

Das Betriebssystem behandelt eine auf diese Weise erstellte URL-Datei wie eine richtige Verknüpfung, selbst wenn es eine Klartextdatei anstelle einer .LNK-Datei ist.

Verwandte Befehle

FileGetShortcut, FileAppend

Beispiel

; Der Buchstabe "i" im letzten Parameter ermöglicht die Tastenkombination Strg+Alt+I:
FileCreateShortcut, Notepad.exe, %A_Desktop%\Eigene Verknüpfung.lnk, C:\, "%A_ScriptFullPath%", Eigene Beschreibung, C:\Eigenes Icon.ico, i
