#ErrorStdOut

Sendet alle Syntaxfehler, die den Start eines Scripts verhindern, zur Standardausgabe, anstatt ein Dialogfenster anzuzeigen.

#ErrorStdOut

Damit können originelle Texteditoren wie Textpad, SciTE, Crimson und EditPlus zur fehlerhaften Zeile springen, wenn ein Syntaxfehler aufgetreten ist. Es ist meist besser, den Editor so einzustellen, dass er die Befehlszeilenoption / ErrorStdOut beim jedem Start eines AutoHotkey-Skripts verwendet (siehe weiter unten für Setup-Anweisungen).

Obwohl Syntaxfehler zur Standardausgabe (stdout) gesendet werden, werden sie nicht direkt in der Eingabeaufforderung angezeigt. Stattdessen können solche Ausgaben abgefangen werden, wenn sie weiter- oder umgeleitet werden. Zum Beispiel:

"C:\Programme\AutoHotkey\AutoHotkey.exe" /ErrorStdOut "Eigenes Script.ahk" |more
"C:\Programme\AutoHotkey\AutoHotkey.exe" /ErrorStdOut "Eigenes Script.ahk" >"Syntax-Fehler Log.txt"

Die Ausgabe kann auch direkt zur Zwischenablage umgeleitet werden, wenn cb.zip (4 KB) heruntergeladen und das folgende Beispiel verwendet wird:

"C:\Programme\AutoHotkey\AutoHotkey.exe" /ErrorStdOut "Eigenes Script.ahk" |cb.exe

Anweisungen für bestimmte Editoren:

EditPlus:
Wählt in der Menüleiste "Tools > Configure User Tools" aus.
Drückt die Schaltfläche: Add Tool > Program
Menu Text: Beliebig
Command: C:\Programme\AutoHotkey\AutoHotkey.exe
Argument: /ErrorStdOut "$(FilePath)"
Initial directory: $(FileDir)
Capture output: Ja

TextPad:
Wählt in der Menüleiste "Configure > Preferences" aus.
Klappt den Eintrag "Tools" auf.
Drückt die Schaltfläche "Add" und wählt "Program" aus.
Kopieren und Einfügen (Pfad anpassen): C:\Windows\System32\cmd.exe -- OK drücken.
Dreifachklick auf dem neu hinzugefügten Eintrag (cmd.exe) in der ListBox und nach Wunsch umbenennen (z. B. Script starten).
Apply drücken.
Wählt das neue Element in der Baumstruktur links aus und tragt folgende Informationen ein:
Command (sollte bereits ausgefüllt sein): cmd.exe (oder der vollständige Pfad)
Parameters (an eigenen Pfad anpassen, falls notwendig): /c ""C:\Programme\AutoHotkey\AutoHotkey.exe" /ErrorStdOut "$File""
Initial folder: $FileDir
Folgende Häkchen setzen: 1) Run minimized; 2) Capture output.
OK drücken. Das neu hinzugefügte Element sollte nun im Menü "Tools" vorhanden sein.

Verwandte Befehle

FileAppend (weil es den Text auch zur Standardausgabe senden kann)
