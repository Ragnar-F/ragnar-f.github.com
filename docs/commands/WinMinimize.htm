WinMinimize

Verkleinert das angegebene Fenster zur einer Schaltfläche auf der Taskleiste.

WinMinimize [, Fenstertitel, Fenstertext, Titelausnahme, Textausnahme]

Parameter

Fenstertitel Der Titel oder Teiltitel des Fensters (das Übereinstimmungsverhalten wird mit SetTitleMatchMode bestimmt). Wenn dieser und die nächsten 3 Parameter nicht vorhanden sind, dann wird das zuletzt gefundene Fenster verwendet. Wenn dieser den Buchstaben A enthält und die nächsten 3 Parameter nicht vorhanden sind, dann wird das aktive Fenster verwendet. Bei Verwendung einer Fensterklasse muss ahk_class ExakterKlassenname angegeben werden (wird von Windows Spy angezeigt). Bei Verwendung einer Prozess-Kennung (PID) muss ahk_pid %VariableMitPID% angegeben werden. Um eine Fenstergruppe zu minimieren, verwendet ahk_group EigenerGruppenname (in diesem Fall müssen Fenstertext, Titelausnahme und Textausnahme leer sein). Bei Verwendung einer eindeutigen ID-Nummer eines Fensters muss ahk_id %VariableMitID% angegeben werden. Die Suche kann durch Angabe mehrerer Kriterien eingeschränkt werden. Zum Beispiel: Eigene Datei.txt ahk_class Notepad
Fenstertext Wenn vorhanden, dann muss dieser Parameter einen Teil der einzelnen Textelemente des Fensters enthalten (wird von Windows Spy angezeigt). Versteckte Textelemente werden erfasst, wenn DetectHiddenText aktiv ist.
Titelausnahme Fenstertitel mit diesem Wert werden nicht berücksichtigt.
Textausnahme Fenstertexte mit diesem Wert werden nicht berücksichtigt.

Bemerkungen

Verwendet WinRestore, um die Minimierung des Fensters aufzuheben und WinMaximize, um es zu maximieren.

Wenn bestimmte Fensterarten nicht richtig auf WinMinimize reagieren, versucht stattdessen Folgendes:
PostMessage, 0x112, 0xF020,,, Fenstertitel, Fenstertext ; 0x112 = WM_SYSCOMMAND, 0xF020 = SC_MINIMIZE

Dieser Befehl arbeitet immer nur mit dem ersten gefundenen Fenster, es sein denn, es befindet sich ahk_group Groupname in Fenstertitel. In dem Fall sind alle Fenster der Gruppe betroffen.

Fenstertitel und -texte sind von der Groß- und Kleinschreibung abhängig. Solange DetectHiddenWindows deaktiviert ist, werden versteckte Fenster nicht wahrgenommen.

Verwandte Befehle

WinRestore, WinMaximize, WinMinimizeAll

Beispiel

Run, notepad.exe
WinWait, Unbenannt - Editor
WinMinimize ; verwendet das zuletzt gefundene Fenster

^Down::WinMinimize, A  ; Hotkey zuweisen, um das aktive Fenster zu minimieren.
