WinKill

Zwingt das angegebene Fenster sich zu schließen.

WinKill [, Fenstertitel, Fenstertext, Wartezeit, Titelausnahme, Textausnahme]

Parameter

Fenstertitel Ein Fenstertitel oder sonstige Kriterien, die das Zielfenster ermitteln sollen. Siehe Fenstertitel.
Fenstertext In diesem Parameter kann ein Teilstring angegeben werden, der in einem Textelement des Zielfensters vorkommt (werden mit Windows Spy angezeigt). Versteckte Textelemente werden auch erkannt, sofern DetectHiddenText aktiv ist.
Wartezeit Ist dieser Parameter leer oder nicht vorhanden, wartet der Befehl überhaupt nicht. Wenn 0 angegeben ist, wartet der Befehl 500 ms. Ansonsten wartet der Befehl die angegebene Anzahl an Sekunden (kann einen Dezimalpunkt oder einen Ausdruck enthalten), bis das Fenster geschlossen wird. Wenn sich das Fenster innerhalb der angegebenen Zeit nicht schließt, wird das Script fortgesetzt. Da ErrorLevel nicht von diesem Befehl gesetzt wird, muss man mit IfWinExist oder WinWaitClose herausfinden, ob ein Fenster geschlossen worden ist.
Titelausnahme Ignoriert Fenster, deren Titel diesen Wert enthalten.
Textausnahme Ignoriert Fenster, deren Textelemente diesen Wert enthalten.

Bemerkungen

Zuerst versucht dieser Befehl, das Fenster auf normale Weise zu schließen. Funktioniert das nicht, wird der Befehl versuchen, das Fenster durch Beenden seines Prozesses zu schließen.

Wenn ein übereinstimmendes Fenster aktiv ist, wird vorrangig dieses Fenster geschlossen, anstatt jedes andere übereinstimmende Fenster darunter. Im Allgemeinen wird, wenn mehrere Fenster übereinstimmen, das oberste (aktuell verwendete) Fenster geschlossen.

Dieser Befehl bezieht sich immer auf das oberste übereinstimmende Fenster. Enthält Fenstertitel allerdings ahk_group Gruppenname, sind alle Fenster von der Gruppe betroffen.

Fenstertitel und -texte unterscheiden zwischen Groß- und Kleinschreibung. Solange DetectHiddenWindows deaktiviert ist, werden versteckte Fenster nicht wahrgenommen.

Verwandte Befehle

WinClose, WinWaitClose, Process, WinActivate, SetTitleMatchMode, DetectHiddenWindows, Zuletzt gefundenes Fenster, IfWinExist, IfWinActive, WinWaitActive, WinWait, GroupActivate

Beispiel

IfWinExist, Unbenannt - Editor
    WinKill ; verwendet das oben gefundene Fenster
Else
    WinKill, Rechner
