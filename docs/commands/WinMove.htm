WinMove

Verändert die Position und/oder Größe des angegebenen Fensters.

WinMove, X, Y
WinMove, Fenstertitel, Fenstertext, X, Y [, Breite, Höhe, Titelausnahme, Textausnahme]

Parameter

X, Y

Die Koordinaten X und Y der oberen linken Ecke der neuen Position vom Zielfenster (in Pixel), die Ausdrücke sein können. Das obere linke Pixel des Bildschirms befindet sich auf 0, 0.

Sind nur diese Parameter angegeben, wird das zuletzt gefundene Fenster als Zielfenster verwendet.

Ansonsten kann X und/oder Y weggelassen werden, um die aktuelle Position zu verwenden.
Fenstertitel Ein Fenstertitel oder sonstige Kriterien, die das Zielfenster ermitteln sollen. Siehe Fenstertitel.
Fenstertext In diesem Parameter kann ein Teilstring angegeben werden, der in einem Textelement des Zielfensters vorkommt (werden mit Windows Spy angezeigt). Versteckte Textelemente werden auch erkannt, sofern DetectHiddenText aktiv ist.
Breite, Höhe Die neue Breite und Höhe des Fensters (in Pixel), die Ausdrücke sein können. Wenn einer der Parameter leer ist, weggelassen wird oder das Wort DEFAULT enthält, dann wird die Größe in dieser Abmessung nicht verändert.
Titelausnahme Ignoriert Fenster, deren Titel diesen Wert enthalten.
Textausnahme Ignoriert Fenster, deren Textelemente diesen Wert enthalten.

Bemerkungen

Die meisten Fenster, deren Breite und Höhe verringert (oder negativ gemacht) wird, werden in der Regel nicht kleiner als 112 x 27 Pixel (allerdings kann es sein, dass einige Fensterarten eine andere Mindestgröße haben). Die meisten Fenster, deren Breite und Höhe erhöht wird, werden nicht größer als ungefähr 12 Pixel über den Abmessungen des Desktops.

Negative Werte können bei den Koordinaten X und Y angegeben werden, um Systeme mit mehreren Monitoren zu unterstützen und die Möglichkeit zu bieten, ein Fenster über den Bildschirmrand hinaus zu verschieben.

WinMove kann minimierte Fenster nicht verschieben. Allerdings kann es versteckte Fenster verschieben, sofern DetectHiddenWindows aktiviert ist.

Die Geschwindigkeit von WinMove wird von SetWinDelay beeinflusst.

Fenstertitel und -texte unterscheiden zwischen Groß- und Kleinschreibung. Solange DetectHiddenWindows deaktiviert ist, werden versteckte Fenster nicht wahrgenommen.

Verwandte Befehle

ControlMove, WinGetPos, WinHide, WinMinimize, WinMaximize, WinSet

Beispiel

Run, calc.exe
WinWait, Rechner
WinMove, 0, 0 ; Verschiebt das gefundene Fenster von WinWait in die obere linke Ecke des Bildschirms.

SplashTextOn, 400, 300, Zwischenablage, Die Zwischenablage enthält:`n%clipboard%
WinMove, Zwischenablage, , 0, 0 ; Verschiebt das Splash-Fenster in die linke obere Ecke.
Msgbox, Drücke auf OK`, um das Popup-Fenster zu schließen.
SplashTextOff

; Die folgende Funktion zentriert das angegebene Fenster auf den Bildschirm:
FensterZentrieren(Fenstertitel)
{
    WinGetPos,,, Breite, Höhe, %Fenstertitel%
    WinMove, %Fenstertitel%,, (A_ScreenWidth/2)-(Breite/2), (A_ScreenHeight/2)-(Höhe/2)
}
