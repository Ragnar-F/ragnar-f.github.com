FileRemoveDir

Entfernt einen Ordner.

FileRemoveDir, Verzeichnisname [, Rekursiv?]

Parameter

Verzeichnisname Der Name des Verzeichnisses, das sich im A_WorkingDir-Verzeichnis befindet, solange kein absoluter Pfad angegeben ist.
Rekursiv?

0 (Standard): Entfernt nicht Dateien und Unterverzeichnisse, die im Verzeichnisnamen enthalten sind. In diesem Fall passiert nichts und ErrorLevel wird auf 1 gesetzt, wenn der Verzeichnisname nicht leer ist.

1: Entfernt alle Dateien und Unterverzeichnisse (ähnlich dem Windows-Befehl "rmdir /S").

Dieser Parameter kann ein Ausdruck oder True bzw. False sein (intern auch als 1 bzw. 0 gespeichert).

ErrorLevel

[v1.1.04+] Dieser Befehl ist in der Lage, bei Misserfolg eine Ausnahme auszulösen. Für mehr Informationen, siehe Laufzeitfehler.

ErrorLevel wird bei Misserfolg auf 1 und bei Erfolg auf 0 gesetzt.

Bemerkungen

Keine.

Verwandte Befehle

FileCreateDir, FileDelete

Beispiel

FileRemoveDir, C:\Download Temp
FileRemoveDir, C:\Download Temp, 1
