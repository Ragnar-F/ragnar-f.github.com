#KeyHistory

Bestimmt die maximale Anzahl an Tastatur- und Mausereignisse, die im KeyHistory-Fenster angezeigt werden. Mit 0 kann der Tastenverlauf deaktiviert werden.

#KeyHistory Maximalereignisse

Parameter

Maximalereignisse Die maximale Anzahl an Tastatur- und Mausereignisse, die im KeyHistory-Fenster angezeigt werden (Standard 40, Limit 500). Mit 0 kann der Tastenverlauf deaktiviert werden.

Bemerkungen

Diese Anweisung muss nur einmal irgendwo angegeben werden, da sie bereits vor dem Ausführen des Scripts aktiv wird.

Da jede Tastatureingabe oder jeder Mausklick aus einem Down- und Up-Event besteht, zeigt KeyHistory nur halb so viele "vollständige Ereignisse" an. Wenn das Script zum Beispiel "#KeyHistory 50" enthält, dann werden bis zu 25 Tastatureingaben und Mausklicks angezeigt.

Verwandte Befehle

KeyHistory, #NoTrayIcon

Beispiel

#KeyHistory 0      ; Deaktiviert den Tastenverlauf.
#KeyHistory 100  ; Speichert bis zu 100 Ereignisse.
