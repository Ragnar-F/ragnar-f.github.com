#KeyHistory

Bestimmt die maximale Anzahl an Tastatur- und Mausereignissen, die im KeyHistory-Fenster angezeigt werden sollen. Mit 0 kann der Tastenverlauf deaktiviert werden.

#KeyHistory MaxEreignisse

Parameter

MaxEreignisse Die maximale Anzahl an Tastatur- und Mausereignissen, die im KeyHistory-Fenster angezeigt werden sollen (Standard 40, Limit 500). Mit 0 kann der Tastenverlauf deaktiviert werden.

Bemerkungen

Da diese Einstellung bereits aktiviert wird, bevor das Script startet, muss die Direktive nur einmal angegeben werden (egal wo).

Da jede Tastatur- oder Mauseingabe aus einem Down- und Up-Ereignis (Drücken und Loslassen) besteht, zeigt KeyHistory nur halb so viele "absolvierte Ereignisse" an. Enthält das Script beispielsweise "#KeyHistory 50", werden bis zu 25 Tastatureingaben und Mausklicks angezeigt.

Verwandte Befehle

KeyHistory, #NoTrayIcon, ListHotkeys

Beispiel

#KeyHistory 0      ; Deaktiviert den Tastenverlauf.
#KeyHistory 100  ; Speichert bis zu 100 Ereignisse.
