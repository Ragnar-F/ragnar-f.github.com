#CommentFlag

Ändert das Kommentarsymbol des Scripts zu einem anderen String.

#CommentFlag NeuerString

Parameter

NeuerString Ein oder mehrere Zeichen, die als neues Kommentarkennzeichen verwendet werden sollen. Es können bis zu 15 Zeichen angegeben werden.

Bemerkungen

Das Standard-Kommentarkennzeichen ist ein Semikolon (;).

Das Kommentarkennzeichen gibt an, dass nachfolgender Text nicht vom Script eingelesen werden soll (beim Ausführen des Scripts werden Kommentare nicht in den Speicher geladen, damit diese keinen Einfluss auf die Performance haben).

Ein Kommentarkennzeichen neben einem Befehl wird erst als Kommentar angesehen, wenn auf dessen linken Seite mindestens ein Leer- oder Tabulatorzeichen vorkommt. Zum Beispiel:

MsgBox, Test1 ; Das ist ein Kommentar.
MsgBox, Test2; Das ist kein Kommentar und wird von der MsgBox angezeigt.

Verwandte Befehle

#EscapeChar

Beispiel

#CommentFlag //  ; Wechselt auf die C++-Kommentierung.
