#CommentFlag

Wechselt das Kommentarsymbol des Scripts vom Semikolon zu einigen anderen Strings.

#CommentFlag NeuerString

Parameter

NeuerString Beliebig viele Zeichen, die als neue Kommentarkennzeichen verwendet werden. Bis zu 15 Zeichen können angegeben werden.

Bemerkungen

Das Standard-Kommentarkennzeichen ist ein Semikolon (;).

Das Kommentarkennzeichen wird verwendet, damit das Script den nachfolgenden Text nicht einliest (Kommentare werden nicht in den Speicher geladen, wenn das Script ausgeführt wird, um die Performance nicht zu beeinflussen).

Ein Kommentarkennzeichen, das auf derselben Zeile des Befehls steht, wird nur als Kommentar gewertet, wenn ein Leerzeichen oder Tabulator davor ist. Zum Beispiel:

MsgBox, Test1 ; Das ist ein Kommentar.
MsgBox, Test2; Das ist kein Kommentar und wird von der MsgBox angezeigt.

Verwandte Befehle

#EscapeChar

Beispiel

#CommentFlag //  ; Wechselt auf die C++-Kommentierung.
