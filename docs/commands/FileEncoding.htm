FileEncoding [AHK_L 42+]

Bestimmt die Standardcodierung für FileRead, FileReadLine, Loop Read, FileAppend und FileOpen.

FileEncoding [, Codierung]

Codierung kann einer der folgenden Werte sein:

  • UTF-8: Unicode UTF-8, entspricht CP65001.
  • UTF-16: Unicode UTF-16 mit Little-Endian-Bytereihenfolge, entspricht CP1200.
  • UTF-8-RAW oder UTF-16-RAW: Wie oben, aber ohne dass eine BOM (Bytereihenfolge-Markierung) beim Erstellen einer neuen Datei hinzugefügt wird.
  • CPnnn: Ein Zeichensatz mit numerischer Kennung nnn. Siehe Kennungen von Zeichensätzen.
  • Leer oder nicht vorhanden: Standard-ANSI-Zeichensatz des Systems, der auch die Standardeinstellung ist.

Bemerkungen

A_FileEncoding enthält die aktuelle Einstellung.
