FileEncoding [AHK_L 42+]

Bestimmt die Standardkodierung für FileRead, FileReadLine, dateilesende Schleife, FileAppend und FileOpen.

FileEncoding [, Kodierung]

Die Kodierung kann einer der folgenden Werte sein:

  • UTF-8: Unicode UTF-8, entspricht CP65001.
  • UTF-16: Unicode UTF-16, Little-Endian-Bytereihenfolge. Das ist gleichbedeutend mit CP1200.
  • UTF-8-RAW oder UTF-16-RAW: Wie oben, aber ohne BOM (Bytereihenfolge-Markierung), wenn eine neue Datei erstellt wird.
  • CPnnn: Ein Zeichensatz mit numerischer Kennung nnn. Siehe Kennungen von Zeichensätzen.
  • Leer oder nicht vorhanden: Standard-ANSI-Zeichensatz des Systems, der standardmäßig verwendet wird.

Bemerkungen

A_FileEncoding enthält die aktuelle Einstellung.

Verwandte Befehle

FileOpen, StrPut/StrGet, Script-Kompatibilität
