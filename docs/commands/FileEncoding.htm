FileEncoding [AHK_L 42+]

Bestimmt die Standardkodierung für FileRead, dateilesende Schleife, FileAppend und FileOpen.

FileEncoding [, Kodierung]

Die Kodierung enthält einen folgenden Wert:

  • UTF-8: Unicode UTF-8, entspricht CP65001.
  • UTF-16: Unicode UTF-16, Little-Endian-Bytereihenfolge. Das ist gleichbedeutend mit CP1200.
  • UTF-8-RAW oder UTF-16-RAW: Wie oben, jedoch wird keine Bytereihenfolge-Markierung geschrieben, sobald eine neue Datei erstellt wird.
  • CPnnn: Ein Zeichensatz mit numerischer Kennung nnn. Siehe Kennungen von Zeichensätzen.
  • Leer oder nicht vorhanden: der Standard-ANSI-Zeichensatz des Systems, was auch die Standardeinstellung ist.

Bemerkungen

A_FileEncoding enthält die aktuelle Einstellung.

Verwandte Befehle

FileOpen, StrPut/StrGet, Script-Kompatibilität
