ComObjGet() [AHK_L 53+]

Ermittelt den Verweis auf ein Objekt, das von einer COM-Komponente unterstützt wird.

ComObject := ComObjGet(Name)

Parameter

Name Der Anzeigename des Objekts, das abgerufen werden soll. Siehe MkParseDisplayName (MSDN) für mehr Informationen.

Verwandte Befehle

ComObjCreate, ComObjActive, ComObjConnect, ComObjError, CoGetObject (MSDN)

Beispiele

; Beispiel: Drückt Umschalt+Escape, um die zuletzt verwendete Befehlszeile anzuzeigen und
; den aktiven Prozess des Fensters zu starten. Benötigt XP oder höher.
+Esc::
    WinGet pid, PID, A
    ; WMI-Service-Objekt ermitteln.
    wmi := ComObjGet("winmgmts:")
    ; Abfrage ausführen, um den/die passenden Prozess(e) zu ermitteln.
    queryEnum := wmi.ExecQuery(""
        . "Select * from Win32_Process where ProcessId=" . pid)
        ._NewEnum()
    ; Den ersten passenden Prozess übernehmen.
    If queryEnum[process]
        MsgBox 0, Befehlszeile, % process.CommandLine
    Else
        MsgBox Prozess nicht gefunden!
    ; Alle globalen Objekte leeren (nicht notwendig, wenn lokale Variablen verwendet werden).
    wmi := queryEnum := process := ""
Return
; Win32_Process: http://msdn.microsoft.com/en-us/library/aa394372.aspx
