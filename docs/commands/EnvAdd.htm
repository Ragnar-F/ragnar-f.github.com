EnvAdd

Bestimmt eine Variable, die sich selbst mit einem vorgegebenen Wert addiert (kann auch Zeit von einem Datum-Zeit-Wert addieren oder subtrahieren). Gleichbedeutend mit: Var += Wert

EnvAdd, Var, Wert [, Zeiteinheit]
Var += Wert [, Zeiteinheit]
Var++

Parameter

Var Der Name der Variable, die bearbeitet wird.
Wert Integer, Fließkommazahl oder Ausdruck.
Zeiteinheit

Sobald dieser Parameter vorhanden ist, wird Var als Datum-Zeit-Stempel im YYYYMMDDHH24MISS-Format und Wert als Integer oder Fließkommazahl behandelt (mit einer negativen Zahl wird eine Subtraktion erreicht). Die Zeiteinheit enthält entweder Seconds, Minutes, Hours oder Days (oder jeweils nur den ersten Buchstaben).

Wenn Var leer ist, wird die aktuelle Zeit verwendet. Wenn Var einen ungültigen Zeitstempel oder ein Jahr vor 1601 enthält, oder Wert keine gültige Zahl ist, wird Var leer gemacht werden, um auf ein Problem hinzuweisen.

Die interne Variable A_Now enthält die aktuelle Ortszeit im YYYYMMDDHH24MISS-Format.

Zur Berechnung des Abstands zwischen zwei Zeitstempeln kann EnvSub verwendet werden.

Bemerkungen

Dieser Befehl ist gleichbedeutend mit der Kurzfassung: Var += Wert

Der Wert einer Variable kann um 1 erhöht oder verringert werden, wenn Var++, Var--, ++Var oder --Var verwendet wird.

Wenn Var oder Wert leer ist oder nicht mit einer Zahl beginnt, wird eine 0 als Berechnungsgrundlage verwendet (außer bei interner Verwendung in Ausdrücken oder bei Verwendung des Zeiteinheit-Parameters).

Wenn Var oder Wert einen Dezimalpunkt enthält, ist das Endergebnis eine Fließkommazahl, abhängig vom eingestellten Format (SetFormat).

Verwandte Befehle

EnvSub, EnvMult, EnvDiv, SetFormat, Ausdrücke, If Var is [not] Typ, SetEnv, FileGetTime

Beispiel

EnvAdd, EigeneZählung, 2
EigeneZählung += 2  ; das Gleiche wie oben

Var1 =  ; Variable leer machen, damit die aktuelle Zeit verwendet wird.
Var1 += 31, days
MsgBox, %Var1%  ; Das Ergebnis ist das Datum nach genau 31 Tagen.
