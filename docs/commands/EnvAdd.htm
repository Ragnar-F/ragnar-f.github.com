EnvAdd

Bestimmt eine Variable, die sich selbst mit einem vorgegebenen Wert addiert (kann auch Zeit von einem Datum-Zeit-Wert addieren oder subtrahieren). Gleichbedeutend mit: Var += Wert.

EnvAdd, Var, Wert [, Zeiteinheit]
Var += Wert [, Zeiteinheit]
Var++

Parameter

Var

Name der Variable, die bearbeitet werden soll.

Value

Integer, Fließkommazahl oder Ausdruck.

Zeiteinheit

Ist dieser Parameter vorhanden, wird der Befehl, während er Wert und Var addiert, Var als Datum-Zeit-Stempel im YYYYMMDDHH24MISS-Format und Wert als Integer oder Fließkommazahl behandeln (mit einer negativen Zahl kann eine Subtraktion durchgeführt werden). Zeiteinheit kann entweder Seconds, Minutes, Hours oder Days sein (oder jeweils nur den ersten Buchstaben).

Wenn Var eine leere Variable ist, wird die aktuelle Zeit verwendet. Enthält Var einen ungültigen Zeitstempel oder ein Jahr vor 1601, oder ist Wert nicht-numerisch, wird Var leer gemacht, um das Problem zu kennzeichnen.

Die integrierte Variable A_Now enthält die aktuelle Ortszeit im YYYYMMDDHH24MISS-Format.

Mit EnvSub kann die Zeitmenge zwischen zwei Zeitstempeln berechnet werden.

Bemerkungen

Dieser Befehl ist gleichbedeutend mit der Kurzfassung: Var += Wert.

Mit Var++, Var--, ++Var oder --Var können Variablen um 1 erhöht oder verringert werden.

Wenn Var oder Wert leer oder keine Zahl ist, wird es bei der Berechnung als 0 angesehen (außer innerhalb eines Audrucks und beim Verwenden des Zeiteinheit-Parameters).

Enthält Var oder Wert einen Dezimalpunkt, wird das Endergebnis eine Fließkommazahl sein, abhängig vom gesetzten Format per SetFormat.

Siehe auch

EnvSub, EnvMult, EnvDiv, SetFormat, Ausdrücke, If Var is [not] Typ, SetEnv, FileGetTime

Beispiel

EnvAdd, MeineZählung, 2
MeineZählung += 2  ; das Gleiche wie oben

Var1 =  ; Macht sie leer, so dass das untere stattdessen die aktuelle Zeit verwendet.
Var1 += 31, days
MsgBox, %Var1%  ; Das Ergebnis ist das heutige Datum, addiert mit 31 Tagen.
