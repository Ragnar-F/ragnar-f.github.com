StringLeft / StringRight

Ermittelt eine bestimmte Anzahl an Zeichen von der linken oder rechten Seite eines Strings.

StringLeft, Ausgabevariable, Eingabevariable, Anzahl
StringRight, Ausgabevariable, Eingabevariable, Anzahl
NeuerStr := SubStr(String, Startposition [, Länge]) ; Siehe SubStr()-Funktion für Details.

Parameter

Ausgabevariable Name der Variable, in die der extrahierte Teilstring aus Eingabevariable gespeichert wird.
Eingabevariable Name der Variable, deren Inhalt extrahiert wird. Der Name sollte nicht mit Prozentzeichen umschlossen werden, sofern nicht die Absicht besteht, den Inhalt der Variable als Namen zu verwenden.
Anzahl Die Anzahl der zu extrahierenden Zeichen, das ein Ausdruck sein kann. Ist Anzahl kleiner gleich 0, wird Ausgabevariable leer gemacht. Wenn Anzahl die Länge von Eingabevariable überschreitet, wird Ausgabevariable den gesamten Inhalt von Eingabevariable enthalten.

Bemerkungen

Bei diesen und allen anderen Befehlen kann die Ausgabevariable auch die Eingabevariable sein, und umgekehrt.

Verwandte Befehle

SubStr(), StringMid, StringTrimLeft, StringTrimRight, IfInString, StringGetPos, StringLen, StringLower, StringUpper, StringReplace

Beispiel

String = Das ist ein Test.
StringLeft, Ausgabevariable, String, 3  ; Speichert den String "Das" in die Ausgabevariable.
StringRight, Ausgabevariable, String, 5  ; Speichert den String "Test." in die Ausgabevariable.
