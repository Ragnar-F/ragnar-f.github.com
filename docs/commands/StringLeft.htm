StringLeft / StringRight

Ermittelt eine bestimmte Anzahl an Zeichen von der linken oder rechten Seite eines Strings.

StringLeft, Ausgabevariable, Eingabevariable, Anzahl
StringRight, Ausgabevariable, Eingabevariable, Anzahl
NeuerStr := SubStr(String, Startposition [, Länge]) ; Siehe SubStr()-Funktion für Details.

Parameter

Ausgabevariable Der Name der Variable, in der der extrahierte Teilstring aus der Eingabevariable gespeichert wird.
Eingabevariable Name der Variable, dessen Inhalt extrahiert wird. Umschließt den Namen erst mit Prozentzeichen, wenn der Inhalt der Variable als Name verwendet werden soll.
Anzahl Die Anzahl an Zeichen, die extrahiert werden, das auch ein Ausdruck sein kann. Wenn die Anzahl kleiner gleich 0 ist, wird dann wird die Ausgabevariable leer gemacht. Wenn die Anzahl die Länge der Eingabevariable überschreitet, dann wird die Ausgabevariable den gesamten Inhalt der Eingabevariable enthalten.

Bemerkungen

Wie bei allen anderen Befehlen darf die Ausgabevariable dieselbe Variable wie die Eingabevariable sein.

Verwandte Befehle

SubStr(), StringMid, StringTrimLeft, StringTrimRight, IfInString, StringGetPos, StringLen, StringLower, StringUpper, StringReplace

Beispiel

String = Das ist ein Test.
StringLeft, Ausgabevariable, String, 3  ; Speichert den String "Das" in die Ausgabevariable.
StringRight, Ausgabevariable, String, 5  ; Speichert den String "Test." in die Ausgabevariable.
