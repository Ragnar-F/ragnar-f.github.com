Download

Lädt eine Datei aus dem Internet herunter.

Download, URL, Dateiname

Parameter

URL

URL der Datei, die heruntergeladen werden soll. Mit http://someorg.org beispielsweise wird die Willkommensseite der Organisation abgerufen.

Dateiname

In eine Datei herunterladen: Der Name der lokal erstellten Datei, die sich im A_WorkingDir-Verzeichnis befindet, solange kein absoluter Pfad angegeben ist. Jede vorhandene Datei wird durch die neue Datei überschrieben.

In eine Variable herunterladen: Siehe unteres Beispiel. Für alternative Methoden, siehe folgenden Forumsbeitrag: www.autohotkey.com/forum/topic10466.html.

ErrorLevel

[v1.1.04+] Dieser Befehl ist in der Lage, bei Misserfolg eine Ausnahme auszulösen. Für mehr Informationen, siehe Laufzeitfehler.

ErrorLevel wird bei Misserfolg auf 1 und bei Erfolg auf 0 gesetzt.

Bemerkungen

Es kann sein, dass bei dem Versuch, eine nicht vorhandene Datei herunterzuladen, der Download scheinbar trotzdem erfolgreich war. Das liegt daran, dass die meisten Webserver für die fehlende Datei eine Fehlerseite senden. Daher wird stattdessen einfach die Fehlerseite in Dateiname gespeichert.

Firewalls oder mehrere Netzwerkkarten könnten bewirken, dass diese Funktion fehlschlägt. Außerdem können einige Webseiten solche Downloads blockieren.

Caching: Standardmäßig wird die URL direkt vom entfernt liegenden Server abgerufen (also nie vom Cache des Internet Explorers). Um das Caching zuzulassen, setzt *0 vor der URL, gefolgt von einem Leerzeichen; zum Beispiel: *0 http://someorg.org. Die Null nach dem Sternchen kann mit einer gültigen dwFlags-Nummer ersetzt werden; sucht für weitere Details in www.microsoft.com nach InternetOpenUrl.

Proxies: Falls ein Proxy-Server in den Einstellungen des Internet Explorers konfiguriert wurde, wird dieser verwendet.

FTP- und Gopher-URLs werden auch unterstützt. Zum Beispiel:

Download, ftp://example.com/home/Datei.zip, C:\Eigener Ordner\Datei.zip  ; Anonym anmelden.
Download, ftp://user:pass@example.com:21/home/Datei.zip, C:\Eigener Ordner\Datei.zip  ; Bestimmten Benutzer anmelden.
Download, ftp://user:pass@example.com/Verzeichnis, C:\Liste.html  ; Lädt eine Liste im HTML-Format runter.

Verwandte Befehle

FileRead, FileCopy

Beispiele

Download, http://l.autohotkey.net/version.txt, C:\AutoHotkey_L Aktuellste Version.txt
Download, http://someorg.org/archive.zip, C:\Someorg-Archiv.zip
; Beispiel: Text in eine Variable herunterladen:
whr := ComObjCreate("WinHttp.WinHttpRequest.5.1")
whr.Open("GET", "http://l.autohotkey.net/version.txt")
whr.Send()
version := whr.ResponseText
MsgBox % version
