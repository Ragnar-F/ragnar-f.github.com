EnvMult

Bestimmt eine Variable, die sich selbst mit einem vorgegebenen Wert multipliziert. Gleichbedeutend mit: Var *= Wert.

EnvMult, Var, Wert

Parameter

Var Name der Variable, die bearbeitet werden soll.
Wert Integer, Fließkommazahl oder Ausdruck.

Bemerkungen

Dieser Befehl ist gleichbedeutend mit der Kurzfassung: Var *= Wert.

Wenn Var oder Wert leer ist oder nicht mit einer Zahl beginnt, wird eine 0 als Berechnungsgrundlage verwendet (außer bei interner Verwendung in Ausdrücken wie z. B. Var := X *= Y).

Wenn Var oder Wert einen Dezimalpunkt enthält, ist das Endergebnis eine Fließkommazahl, abhängig vom eingestellten Format (SetFormat).

Verwandte Befehle

EnvAdd, EnvSub, EnvDiv, SetFormat, Ausdrücke, If Var is [not] Typ, SetEnv, bitweise Operationen (Transform)

Beispiel

EnvMult, EigeneZählung, 2
EigeneZählung *= 2  ; das Gleiche wie oben
