EnvMult

Bestimmt eine Variable, die sich selbst mit einem vorgegebenen Wert multipliziert. Gleichbedeutend mit: Var *= Wert.

EnvMult, Var, Wert

Parameter

Var

Name der Variable, die bearbeitet werden soll.

Wert

Integer, Fließkommazahl oder Ausdruck.

Bemerkungen

Dieser Befehl ist gleichbedeutend mit der Kurzfassung: Var *= Wert.

Wenn Var oder Wert leer oder keine Zahl ist, wird es bei der Berechnung als 0 angesehen (außer innerhalb eines Audrucks wie Var := X *= Y).

Enthält Var oder Wert einen Dezimalpunkt, wird das Endergebnis eine Fließkommazahl sein, abhängig vom gesetzten Format per SetFormat.

Siehe auch

EnvAdd, EnvSub, EnvDiv, SetFormat, Ausdrücke, If Var is [not] Typ, SetEnv, bitweise Operationen (Transform)

Beispiel

EnvMult, MeineZählung, 2
MeineZählung *= 2  ; das Gleiche wie oben
