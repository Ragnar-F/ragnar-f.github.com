EnvMult

Bestimmt eine Variable, die sich selbst mit einem vorgegebenen Wert multipliziert. Gleichbedeutend mit: Variable *= Wert

EnvMult, Variable, Wert

Parameter

Variable Der Name der Variable, die bearbeitet wird.
Wert Beliebiger Integer-Wert, Fließkommazahl oder Ausdruck.

Bemerkungen

Dieser Befehl ist gleichbedeutend mit der Kurzfassung: Variable *= Wert

Wenn entweder Variable oder Wert leer ist oder nicht mit einer Zahl beginnt, dann wird eine 0 als Berechnungsgrundlage verwendet (außer bei interner Verwendung in Ausdrücken wie z. B. Variable := X *= Y).

Wenn die Variable oder der Wert einen Dezimalpunkt enthält, dann ist das Endergebnis eine Fließkommazahl, abhängig vom eingestellten Format (SetFormat).

Verwandte Befehle

EnvAdd, EnvSub, EnvDiv, SetFormat, Ausdrücke, If Variable is [not] Typ, SetEnv, bitweise Operationen (Transform)

Beispiel

EnvMult, EigeneZählung, 2
EigeneZählung *= 2  ; das Gleiche wie oben
