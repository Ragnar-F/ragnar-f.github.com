IfWinActive / IfWinNotActive

Überprüft, ob das angegebene Fenster existiert und zurzeit aktiv ist.

IfWinActive [, Fenstertitel, Fenstertext, Titelausnahme, Textausnahme]
IfWinNotActive [, Fenstertitel, Fenstertext, Titelausnahme, Textausnahme]
UniqueID := WinActive("Fenstertitel", "Fenstertext", "Titelausnahme", "Textausnahme")

Parameter

Fenstertitel Der Titel oder Teiltitel des Fensters (das Übereinstimmungsverhalten wird mit SetTitleMatchMode bestimmt). Wenn dieser und die anderen 3 Fensterparameter nicht vorhanden sind, dann wird das zuletzt gefundene Fenster verwendet. Bei Verwendung einer Fensterklasse muss ahk_class ExakterKlassenname angegeben werden (wird von Windows Spy angezeigt). Bei Verwendung einer Prozess-Kennung (PID) muss ahk_pid %VariableMitPID% angegeben werden. Bei Verwendung einer Fenstergruppe muss ahk_group Gruppenname angegeben werden. Bei Verwendung einer eindeutigen ID-Nummer eines Fensters muss ahk_id %VariableMitID% angegeben werden. Die Suche kann durch Angabe mehrerer Kriterien eingeschränkt werden. Zum Beispiel: Eigene Datei.txt ahk_class Notepad
Fenstertext Wenn vorhanden, dann muss dieser Parameter einen Teil der einzelnen Textelemente des Fensters enthalten (wird von Windows Spy angezeigt). Versteckte Textelemente werden erfasst, wenn DetectHiddenText aktiv ist.
Titelausnahme Fenstertitel mit diesem Wert werden nicht berücksichtigt. Hinweis: Aufgrund der Abwärtskompatibilität mit .aut-Scripts wird dieser Parameter als Befehl interpretiert, wenn eine exakte Übereinstimmung mit dem Befehlsnamen vorhanden ist. Als Übergangslösung kann die Funktion WinActive() benutzt werden.
Textausnahme Fenstertexte mit diesem Wert werden nicht berücksichtigt.

Bemerkungen

Wenn dieser und die anderen 3 Fensterparameter nicht vorhanden sind, dann wird das zuletzt gefundene Fenster verwendet.

Wenn einer der Befehle das aktive Fenster als passende Übereinstimmung ansieht, dann wird das zuletzt gefundene Fenster zum aktiven Fenster aktualisiert. Wenn IfWinActive als "wahr" oder IfWinNotActive als "falsch" gewertet wird, dann wird das zuletzt gefundene Fenster aktualisiert.

Die Funktion WinActive() gibt die eindeutige ID (HWND) des aktiven Fensters zurück, wenn es die angegebenen Kriterien erfüllt. Ansonsten gibt die Funktion eine 0 zurück. Da alle Zahlen außer 0 als "Wahr" angesehen werden, ist die Anweisung If WinActive("Fenstertitel") immer wahr, sobald der Fenstertitel aktiv ist.

SetWinDelay hat keinen Einfluss auf IfWinExist/IfWinActive.

Fenstertitel und -texte sind von der Groß- und Kleinschreibung abhängig. Solange DetectHiddenWindows deaktiviert ist, werden versteckte Fenster nicht wahrgenommen.

Verwandte Befehle

IfWinExist, SetTitleMatchMode, DetectHiddenWindows, zuletzt gefundenes Fenster, WinActivate, WinWaitActive, WinWait, WinWaitClose, #IfWinActive/Exist

Beispiel

IfWinActive, Unbenannt - Editor
{
    WinMaximize  ; Maximiert das oben gefundene Fenster.
    Send, Dies ist ein Text.{Enter}
    Return
}

If WinActive("ahk_class Notepad") or WinActive("ahk_class" . Klassenname)  ; Nach "ahk_class" muss ein Leerzeichen sein.
    WinClose  ; Verwendet das zuletzt gefundene Fenster.
