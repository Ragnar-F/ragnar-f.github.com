WinGetClass

Ermittelt den Klassennamen des angegebenen Fensters.

WinGetClass, Ausgabevariable [, Fenstertitel, Fenstertext, Titelausnahme, Textausnahme]

Parameter

Ausgabevariable Der Name der Variable, in der der ermittelte Klassenname gespeichert wird.
Fenstertitel Der Titel oder Teiltitel des Fensters (das Übereinstimmungsverhalten wird mit SetTitleMatchMode bestimmt). Wenn dieser und die nächsten 3 Parameter weggelassen werden, wird das zuletzt gefundene Fenster verwendet. Sobald dieser den Buchstaben A enthält und die nächsten 3 Parameter weggelassen werden, wird das aktive Fenster verwendet. Bei Verwendung einer Prozess-Kennung (PID) muss ahk_pid %VariableMitPID% angegeben werden. Bei Verwendung einer Fenstergruppe muss ahk_group Gruppenname angegeben werden. Bei Verwendung einer eindeutigen ID-Nummer eines Fensters muss ahk_id %VariableMitID% angegeben werden. Die Suche kann durch Angabe mehrerer Kriterien eingeschränkt werden. Zum Beispiel: Eigene Datei.txt ahk_pid %VarMitPID%
Fenstertext In diesem Parameter kann ein Teilstring angegeben werden, der in einem Textelement des Zielfensters vorkommt (werden mit Windows Spy angezeigt). Versteckte Textelemente werden auch erkannt, sofern DetectHiddenText aktiv ist.
Titelausnahme Ignoriert Fenster, deren Titel diesen Wert enthalten.
Textausnahme Ignoriert Fenster, deren Textelemente diesen Wert enthalten.

Bemerkungen

Nur der Klassenname wird abgerufen (das Präfix "ahk_class" ist nicht in der Ausgabevariable enthalten).

Fenstertitel und -texte sind von der Groß- und Kleinschreibung abhängig. Solange DetectHiddenWindows deaktiviert ist, werden versteckte Fenster nicht wahrgenommen.

Verwandte Befehle

WinGet, WinGetTitle

Beispiel

WinGetClass, Klasse, A
MsgBox, Die Klasse des aktiven Fensters ist "%Klasse%".
