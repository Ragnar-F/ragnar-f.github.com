TrayTip

Erstellt eine Sprechblase in der Nähe des Tray-Icons. Benötigt Windows 2000/XP oder höher.

TrayTip [, Titel, Text, Sekunden, Optionen]

Parameter

Titel

Sind keine Parameter vorhanden, werden alle derzeit angezeigten TrayTip-Fenster entfernt.

Andernfalls ist dieser Parameter der Fenstertitel, der bis zu 73 Zeichen lang sein kann (alle Zeichen danach werden nicht angezeigt).

Falls Titel leer ist, wird die Titelzeile des TrayTip-Fensters komplett weggelassen, indem es vertikal gekürzt wird.
Text

Ist dieser Parameter leer oder nicht vorhanden, werden alle derzeit angezeigten TrayTip-Fenster entfernt.

Andernfalls kann eine Nachricht angegeben werden, die unter dem Titel angezeigt wird. Es werden nur die ersten 265 Zeichen vom Text angezeigt. Die Zeilenumbruchszeichen CR (`r) oder LF (` n) können verwendet werden, um mehrere Textzeilen zu erstellen. Zum Beispiel: Zeile1`nZeile2

Falls der Text zu lang ist, kann er in mehreren kurzen Zeilen aufgeteilt werden (Fortsetzungsbereich), um die Lesbarkeit und Wartbarkeit zu verbessern.
Sekunden

Die ungefähre Anzahl an Sekunden, wie lange das Fenster angezeigt werden soll, bevor es danach wieder vom Betriebssystem entfernt wird. Die Angabe einer Zahl kleiner als 10 oder größer als 30 bewirkt, dass stattdessen die minimale (10) oder maximale (30) Anzeigedauer verwendet wird. Ist dieser Parameter leer oder nicht vorhanden, wird üblicherweise die minimale Zeit verwendet. Dieser Parameter kann ein Ausdruck sein.

Die tatsächliche Zeitüberschreitung weicht von dem ab, was angegeben wurde. Eine Erklärung von Microsoft: "Wenn der Benutzer keine Aktionen auf dem Computer ausführt, berücksichtigt das System diese Zeit bei der Berechnung des Timeouts nicht." (technische Details hier). Zur genauen Bestimmung der Anzeigedauer des TrayTip-Fensters kann der Sleep-Befehl, gefolgt von TrayTip ohne Parameter, verwendet werden, oder SetTimer, wie im unteren Beispielabschnitt gezeigt.
Optionen

Es ist möglich, ein kleines Icon links neben dem Titel anzuzeigen, sobald eine der folgenden Zahlen angegeben wird:

1: Info-Icon
2: Warnung-Icon
3: Fehler-Icon

Ist dieser Parameter nicht vorhanden, gilt standardmäßig eine 0 (kein Icon). Dieser Parameter kann ein Ausdruck sein.

Bemerkungen

TrayTip benötigt Windows 2000/XP oder höher. In Windows 9x/NT funktioniert dieser Befehl nicht.

Besitzt das Script kein Tray-Icon (durch #NoTrayIcon oder Menu, Tray, NoIcon), kann das TrayTip-Fenster nicht angezeigt werden. Außerdem wird TrayTip nicht funktionieren, wenn der folgende REG_DWORD-Wert vorhanden und auf 0 gesetzt ist:
HKCU\Software\Microsoft\Windows\CurrentVersion\Explorer\Advanced >> EnableBalloonTips

Das Betriebssystem spielt üblicherweise einen Sound ab, sobald das TrayTip-Fenster angezeigt wird. Dieser Sound kann deaktiviert werden, indem eine 16 mit dem Optionen-Parameter addiert wird (z. B.: 16 - kein Sound, kein Icon; 17 - kein Sound, aber Info-Icon; und so weiter)

Beachte zudem, dass ein ToolTip angezeigt wird, sobald sich der Mauszeiger über das Tray-Icon des Scripts befindet. Der Inhalt des Tooltips kann wie folgt geändert werden: Menu, Tray, Tip, Neuer Text.

Verwandte Befehle

ToolTip, SetTimer, Menu, SplashTextOn, MsgBox, InputBox, FileSelectFile, FileSelectFolder

Beispiele

TrayTip, Eigener Titel, Mehrzeiliger`nText, 20, 17

; Bietet die Möglichkeit, die Anzeigedauer genauer zu bestimmen,
; ohne dabei Sleep zu verwenden (das den aktuellen Thread stoppen würde):
#Persistent
TrayTip, Zeitlich festgelegter TrayTip, Wird für 5 Sekunden angezeigt.
SetTimer, TrayTipEntfernen, 5000
Return

TrayTipEntfernen:
SetTimer, TrayTipEntfernen, Off
TrayTip
Return

; Damit ein TrayTip dauerhaft angezeigt wird, kann ein Timer verwendet werden, der diesen regelmäßig aktualisiert:
SetTimer, TrayTipAktualisieren, 1000
Gosub, TrayTipAktualisieren ; Einmal aufrufen, um es sofort zu starten.
Return

TrayTipAktualisieren:
TrayTip, Aktualisierter TrayTip, Das ist ein dauerhaft angezeigter TrayTip., , 16
Return
