EnvUpdate

Benachrichtigt das Betriebssystem und alle laufenden Anwendungen, dass die Umgebungsvariablen geändert wurden.

EnvUpdate

ErrorLevel

Bei Problemen wird ErrorLevel auf 1 gesetzt, ansonsten enthält ErrorLevel eine 0.

Bemerkungen

Aktualisiert die OS-Umgebung. Abmelden und wieder Anmelden hat eine ähnliche Wirkung.

Bei Änderung des folgenden Registrierungsschlüssels (nur ab Windows NT/2000/XP) kann mit EnvUpdate beispielsweise diese Änderung aktualisiert werden: HKEY_LOCAL_MACHINE\System\CurrentControlSet\Control\Session
Manager\Environment

Verwandte Befehle

EnvSet, RegWrite
