StringTrimLeft / StringTrimRight

Entfernt eine bestimmte Anzahl an Zeichen von der linken oder rechten Seite eines Strings.

StringTrimLeft, Ausgabevariable, Eingabevariable, Anzahl
StringTrimRight, Ausgabevariable, Eingabevariable, Anzahl
NeuerStr := SubStr(String, Startposition [, Länge]) ; Siehe SubStr()-Funktion für Details.

Parameter

Ausgabevariable Der Name der Variable, in der die gekürzte Version der Eingabevariable gespeichert wird.
Eingabevariable Name der Variable, dessen Inhalt eingelesen wird. Umschließt den Namen erst mit Prozentzeichen, wenn der Inhalt der Variable als Name verwendet werden soll.
Anzahl Die Anzahl an Zeichen, die entfernt werden, das auch ein Ausdruck sein kann. Wenn die Anzahl kleiner gleich 0 ist, wird dann wird die Ausgabevariable den gesamten Inhalt der Eingabevariable enthalten. Wenn die Anzahl die Länge der Eingabevariable überschreitet, dann wird die Ausgabevariable leer gemacht.

Bemerkungen

Wie bei allen anderen Befehlen darf die Ausgabevariable dieselbe Variable wie die Eingabevariable sein.

Verwandte Befehle

SubStr(), StringMid, StringLeft, StringRight, IfInString, StringGetPos, StringLen, StringLower, StringUpper, StringReplace

Beispiel

String = Das ist ein Test.
StringTrimLeft, Ausgabevariable, String, 4  ; Speichert den String "ist ein Test." in die Ausgabevariable.
StringTrimRight, Ausgabevariable, String, 6  ; Speichert den String "Das ist ein" in die Ausgabevariable.
