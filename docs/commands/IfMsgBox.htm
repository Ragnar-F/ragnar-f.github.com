IfMsgBox

Überprüft, welche Schaltfläche der Benutzer während eines MsgBox-Befehls betätigt hat.

IfMsgBox, Schaltfläche

Parameter

Schaltfläche

Ein folgender Text repräsentiert die Schaltfläche, die der Benutzer in der MsgBox gedrückt hat:

Yes (Ja)
No (Nein)
OK
Cancel (Abbrechen)
Abort (Abbrechen)
Ignore (Ignorieren)
Retry (Wiederholen)
Continue (Weiter)[v1.0.44.08+]
TryAgain (Wiederholen)[v1.0.44.08+]
Timeout (wenn MsgBox die Zeit überschreitet)

Verwandte Befehle

MsgBox

Beispiel

MsgBox, 4, , Weiter?, 5  ; 5-sekündliche Zeitüberschreitung.
IfMsgBox, No
    Return  ; Benutzer hat die "Nein"-Schaltfläche betätigt.
IfMsgBox, Timeout
    Return ; d. h. "Nein", wenn Zeitüberschreitung.
