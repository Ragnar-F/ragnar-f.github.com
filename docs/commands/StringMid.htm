StringMid

Ermittelt ein oder mehrere Zeichen von der angegebenen Position innerhalb eines Strings.

StringMid, AusgabeVar, EingabeVar, Startposition [, Anzahl, L]
NeuerStr := SubStr(String, Startposition [, Länge]) ; Siehe SubStr() für Details.

Parameter

AusgabeVar Name der Variable, in der der extrahierte Teilstring aus EingabeVar gespeichert werden soll.
EingabeVar Name der Variable, deren Inhalt extrahiert werden soll. Der Name sollte nicht mit Prozentzeichen umschlossen werden, sofern nicht die Absicht besteht, den Inhalt der Variable als Namen zu verwenden.
Startposition Die Position des ersten zu extrahierenden Zeichens, das ein Ausdruck sein kann. Im Gegensatz zu StringGetPos ist die 1 das erste Zeichen. Falls Startposition kleiner als 1 ist, wird diese als 1 angesehen. Wenn Startposition die Länge des Strings überschreitet, wird AusgabeVar leer gemacht.
Anzahl

Seit v1.0.43.10 kann dieser Parameter leer- oder weggelassen werden, um zu erreichen, dass alle Zeichen des Strings abgerufen werden.

Ansonsten kann die Anzahl der zu extrahierenden Zeichen angegeben werden, das ein Ausdruck sein kann. Ist Anzahl kleiner gleich 0, wird AusgabeVar leer gemacht. Wenn Anzahl die Länge von EingabeVar überschreitet, gemessen von Startposition, wird AusgabeVar den gesamten Inhalt von EingabeVar enthalten, beginnend von Startposition.
L

Der Buchstabe L kann dazu verwendet werden, Zeichen zu extrahieren, die sich auf der linken Seite von Startposition befinden, anstatt auf der rechten. Im folgenden Beispiel wird AusgabeVar auf Rote gesetzt:

EingabeVar = Der Rote Fuchs
StringMid, AusgabeVar, EingabeVar, 8, 4, L

Ist die L-Option vorhanden und Startposition kleiner als 1, wird AusgabeVar leer gemacht. Wenn Startposition die Länge von EingabeVar überschreitet, werden nur Zeichen extrahiert, die sich in Reichweite von Anzahl befinden. Im folgenden Beispiel wird AusgabeVar auf Fuchs gesetzt:

EingabeVar = Der Rote Fuchs
StringMid, AusgabeVar, EingabeVar, 18, 9, L

Bemerkungen

Bei diesen und allen anderen Befehlen kann AusgabeVar auch EingabeVar sein, und umgekehrt.

Verwandte Befehle

SubStr(), StringLeft, StringRight, StringTrimLeft, StringTrimRight, IfInString, StringGetPos, StringLen, StringLower, StringUpper, StringReplace

Beispiel

Quelle = Hallo das ist ein Test.
StringMid, Wort_das, Quelle, 7, 3
