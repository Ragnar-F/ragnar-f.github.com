StringMid

Ermittelt ein oder mehrere Zeichen von der angegebenen Position eines Strings.

StringMid, Ausgabevariable, Eingabevariable, Startposition [, Anzahl, L]
NeuerStr := SubStr(String, Startposition [, Länge]) ; Siehe SubStr()-Funktion für Details.

Parameter

Ausgabevariable Der Name der Variable, in der der extrahierte Teilstring aus der Eingabevariable gespeichert wird.
Eingabevariable Name der Variable, dessen Inhalt extrahiert wird. Umschließt den Namen erst mit Prozentzeichen, wenn der Inhalt der Variable als Name verwendet werden soll.
Startposition Die Position des ersten Zeichens, ab der extrahiert wird, welches auch ein Ausdruck sein kann. Im Gegensatz zu StringGetPos ist Position 1 das erste Zeichen. Wenn die Startposition kleiner als 1 ist, dann wird Position 1 angenommen. Wenn die Startposition das Ende des Strings überschreitet, dann wird die Ausgabevariable leer gemacht.
Anzahl

Seit v1.0.43.10 kann dieser Parameter leer- oder weggelassen werden, um alle Zeichen des Strings abzurufen.

Ansonsten die Anzahl an Zeichen verwenden, die extrahiert werden, welches auch ein Ausdruck sein kann. Wenn die Anzahl kleiner gleich 0 ist, wird dann wird die Ausgabevariable leer gemacht. Wenn die Anzahl die Länge der Eingabevariable überschreitet, gemessen von der Startposition, dann wird die Ausgabevariable den gesamten Inhalt der Eingabevariable enthalten, beginnend von der Startposition.
L

Der Buchstabe L kann verwendet werden, um Zeichen zu extrahieren, die sich auf der linken und nicht auf der rechten Seite der Startposition befinden. Im folgenden Beispiel wird die Ausgabevariable auf Rote gesetzt:
Eingabevariable = Der Rote Fuchs
StringMid, Ausgabevariable, Eingabevariable, 8, 4, L

Wenn die L-Option vorhanden und die Startposition kleiner als 1 ist, dann wird die Ausgabevariable leer gemacht. Wenn die Startposition die Länge der Eingabevariable überschreitet, dann werden nur Zeichen in Reichweite der Anzahl extrahiert. Im folgenden Beispiel wird die Ausgabevariable auf Fuchs gesetzt:
Eingabevariable = Der Rote Fuchs
StringMid, Ausgabevariable, Eingabevariable, 18, 9, L

Bemerkungen

Wie bei allen anderen Befehlen darf die Ausgabevariable dieselbe Variable wie die Eingabevariable sein.

Verwandte Befehle

SubStr(), StringLeft, StringRight, StringTrimLeft, StringTrimRight, IfInString, StringGetPos, StringLen, StringLower, StringUpper, StringReplace

Beispiel

Quelle = Hallo das ist ein Test.
StringMid, Wort_das, Quelle, 7, 3
