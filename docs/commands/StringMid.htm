StringMid

Ermittelt ein oder mehrere Zeichen von der angegebenen Position innerhalb eines Strings.

StringMid, Ausgabevariable, Eingabevariable, Startposition [, Anzahl, L]
NeuerStr := SubStr(String, Startposition [, Länge]) ; Siehe SubStr() für Details.

Parameter

Ausgabevariable Name der Variable, in der der extrahierte Teilstring aus Eingabevariable gespeichert wird.
Eingabevariable Name der Variable, deren Inhalt extrahiert wird. Der Name sollte nicht mit Prozentzeichen umschlossen werden, sofern nicht die Absicht besteht, den Inhalt der Variable als Namen zu verwenden.
Startposition Die Position des ersten zu extrahierenden Zeichens, das ein Ausdruck sein kann. Im Gegensatz zu StringGetPos ist die 1 das erste Zeichen. Falls Startposition kleiner als 1 ist, wird diese als 1 angesehen. Wenn Startposition die Länge des Strings überschreitet, wird Ausgabevariable leer gemacht.
Anzahl

Seit v1.0.43.10 kann dieser Parameter leer- oder weggelassen werden, um zu erreichen, dass alle Zeichen des Strings abgerufen werden.

Ansonsten kann die Anzahl der zu extrahierenden Zeichen angegeben werden, das ein Ausdruck sein kann. Ist Anzahl kleiner gleich 0, wird Ausgabevariable leer gemacht. Wenn Anzahl die Länge von Eingabevariable überschreitet, gemessen von Startposition, wird Ausgabevariable den gesamten Inhalt von Eingabevariable enthalten, beginnend von Startposition.
L

Der Buchstabe L kann dazu verwendet werden, Zeichen zu extrahieren, die sich auf der linken Seite von Startposition befinden, anstatt auf der rechten. Im folgenden Beispiel wird Ausgabevariable auf Rote gesetzt:

Eingabevariable = Der Rote Fuchs
StringMid, Ausgabevariable, Eingabevariable, 8, 4, L

Ist die L-Option vorhanden und Startposition kleiner als 1, wird Ausgabevariable leer gemacht. Wenn Startposition die Länge von Eingabevariable überschreitet, werden nur Zeichen extrahiert, die sich in Reichweite von Anzahl befinden. Im folgenden Beispiel wird Ausgabevariable auf Fuchs gesetzt:

Eingabevariable = Der Rote Fuchs
StringMid, Ausgabevariable, Eingabevariable, 18, 9, L

Bemerkungen

Bei diesen und allen anderen Befehlen kann die Ausgabevariable auch die Eingabevariable sein, und umgekehrt.

Verwandte Befehle

SubStr(), StringLeft, StringRight, StringTrimLeft, StringTrimRight, IfInString, StringGetPos, StringLen, StringLower, StringUpper, StringReplace

Beispiel

Quelle = Hallo das ist ein Test.
StringMid, Wort_das, Quelle, 7, 3
