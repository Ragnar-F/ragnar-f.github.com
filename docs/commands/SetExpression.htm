Var := Ausdruck

Wertet einen Ausdruck aus und speichert das Ergebnis in eine Variable.

Var := Ausdruck

Parameter

Var

Name der Variable, in der das Ergebnis von Ausdruck gespeichert werden soll.

Ausdruck

Siehe Ausdrücke und die Beispiele unten für Details.

Bemerkungen

Der Operator ":=" wurde optimiert, um bei einfachen Fällen genauso schnell zu sein wie der Operator "=", zum Beispiel:

x := y  ; Gleiche Performance wie x = %y%
x := 5  ; Gleiche Performance wie x = 5.
x := "literaler String"  ; Gleiche Performance wie "x = literaler String".

Die Wörter True und False sind integrierte Konstanten, die eine 1 und eine 0 enthalten. Sie können verwendet werden, um ein Script besser lesbar zu machen, z. B.:

Zeichengenauigkeit := false
Suchfortsetzung := true

Es ist möglich, einen Pseudo-Array mit diesem Befehl zu erstellen, sowie mit anderen Befehlen, die AusgabeVar akzeptieren. Dazu enthält AusgabeVar einen Verweis auf eine andere Variable (z. B. Array%i% := Var/100 + 5). Siehe Arrays für mehr Details.
