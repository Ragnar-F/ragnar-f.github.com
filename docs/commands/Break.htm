Break

Beendet bzw. verlässt eine Schleife. Nur in einer Schleife gültig.

Break [, LoopLabel]

[AHK_L 59+]: Wenn angegeben, kennzeichnet LoopLabel, für welche Schleife dieser Befehl gilt. Es kann entweder der Labelname oder die numerische Verschachtelungsebene angegeben werden. Ist dieser Parameter nicht vorhanden oder 1, gilt dieser Befehl für die innerste Schleife, von der er umschlossen ist. LoopLabel muss ein konstanter Wert sein, das heißt, dass Variablen und Ausdrücke nicht unterstützt werden. Wenn ein Label angegeben wurde, muss dieser direkt auf ein Loop-Befehl verweisen.

Break und Continue sollten gegenüber Goto immer vorgezogen werden, da sich Scripts damit einfacher lesen und pflegen lassen.

Verwandte Befehle

Continue, Loop, While-Loop, For-Loop, Block, Label

Beispiel

Loop
{
    ...
    If var > 25
        break
    ...
    If var <= 5
        continue
}
; Unterbricht die äußere Schleife.
outer:
Loop 3
{
    x := A_Index
    Loop 3
    {
        If (x*A_Index = 6)
            break outer  ; Gleichwertig mit break 2 oder goto break_outer.
        MsgBox %x%,%A_Index%
    }
}
break_outer: ; Für goto.
