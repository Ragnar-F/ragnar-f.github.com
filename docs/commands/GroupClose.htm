GroupClose

Schließt das aktive Fenster, das gerade mit GroupActivate oder GroupDeactivate aktiviert worden ist, und aktiviert das nächste Fenster in der Gruppe. Dieser Befehl ist auch in der Lage, alle Fenster in einer Gruppe zu schließen.

GroupClose, Gruppenname [, A|R]

Parameter

Gruppenname Name der Gruppe, die ursprünglich mit GroupAdd definiert worden ist.
A|R

Ist der Buchstabe A angegeben, werden alle Fenster von der Gruppe geschlossen. Das hat die gleiche Wirkung wie WinClose ahk_group Gruppenname.

Ansonsten: Schließt der Befehl das aktive Fenster, wird das nächste Fenster in der Reihe aktiviert. Dieser Parameter legt fest, ob das älteste oder das neueste Fenster aktiviert werden soll, wenn kein einziges Fenster aus der Gruppe aktiv ist. Ist dieser Parameter nicht vorhanden, wird immer das älteste Fenster aktiviert. Ist der Buchstabe R angegeben, wird das neueste (zuletzt aktive) Fenster aktiviert, aber nur, wenn beim Ausführen des Befehls kein Fenster der Gruppe aktiv ist. "R" ist beispielsweise nützlich, wenn der Benutzer vorübergehend eine andere Aufgabe durchführen will. Kehrt er dann mit GroupActivate, GroupDeactivate oder GroupClose zur Gruppe zurück, wird das am aktuellsten genutzte Fenster aktiviert.

Bemerkungen

Enthält der A|R-Parameter kein "A", wird das Verhalten des Befehls davon bestimmt, ob die vorherige Aktion auf Gruppenname GroupActivate oder GroupDeactivate war. Falls es GroupDeactivate war, wird das aktive Fenster nur geschlossen, wenn das Fenster nicht zur Gruppe gehört (ansonsten passiert nichts). Falls es GroupActivate oder keiner dieser Befehle war, wird das aktive Fenster nur geschlossen, wenn das Fenster zur Gruppe gehört (ansonsten passiert nichts). Durch dieses Verhalten ist es möglich, GroupClose einem Hotkey zuzuweisen, als Ergänzung zum GroupActivate- oder GroupDeactivate-Hotkey von Gruppenname.

Siehe GroupAdd für mehr Details über Fenstergruppen.

Verwandte Befehle

GroupAdd, GroupActivate, GroupDeactivate

Beispiel

GroupClose, EigeneGruppe, R
