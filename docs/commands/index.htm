Alphabetischer Befehls- und Funktionsindex

Klickt für mehr Details auf einen Befehls- oder Funktionsnamen. Häufig verwendete Einträge sind groß geschrieben.

Eintrag Beschreibung
{ ... } (Block) Zwei geschweifte Klammern kennzeichnen einen Block. Blöcke werden für gewöhnlich bei Funktionen, Else, Loop, While-Loop und If-Befehle benutzt.
{ ... } / Object() Erstellt ein scriptfähiges assoziatives Array.
[ ... ] / Array() Erstellt ein scriptfähiges assoziatives Array mit Integer-Schlüssel.
Abs() Gibt den absoluten Wert von Zahl zurück.
AutoTrim Bei Anweisungen wie Var1 = %Var2% werden Leer- und Tabulatorzeichen am Beginn und am Ende von Var2 entfernt.
ASin() Gibt den Arkussinus (die Zahl, deren Sinus Zahl ist) als Bogenmaß zurück.
ACos() Gibt den Arkuskosinus (die Zahl, deren Kosinus Zahl ist) als Bogenmaß zurück.
ATan() Gibt den Arkustangens (die Zahl, deren Tangens Zahl ist) als Bogenmaß zurück.
BlockInput Deaktiviert oder aktiviert die Fähigkeit des Benutzers, mit dem Computer mittels Tastatur und Maus zu interagieren.
Break Beendet bzw. verlässt eine Schleife. Nur in einer Schleife gültig.
Catch Bestimmt den Code, der ausgeführt werden soll, wenn eine Ausnahme während einer Try-Anweisung ausgelöst wird.
Ceil() Gibt Zahl zurück, welche auf den nächsten Integer aufgerundet ist (ohne .00-Suffix).
Click Klickt mit einer Maustaste auf bestimmte Koordinaten. Kann auch eine Maustaste gedrückt halten, das Mausrad drehen oder die Maus bewegen.
ClipWait Wartet, bis die Zwischenablage Text oder Dateien enthält.
ComObjActive() Ermittelt ein registriertes COM-Objekt.
ComObjArray() Erstellt einen SafeArray, das mit COM verwendbar ist.
ComObjConnect() Verbindet die Ereignisquellen eines COM-Objekts mit Funktionen, die einen vorgegeben Präfix enthalten.
ComObjCreate() Erstellt ein COM-Objekt.
ComObjEnwrap() / ComObjUnwrap() Packt ein COM-Objekt ein/aus.
ComObjError() Aktiviert oder deaktiviert COM-Fehlermeldungen.
ComObjFlags() Ermittelt oder ändert Flags, die das Verhalten eines COM-Wrapper-Objekts kontrollieren.
ComObjGet() Gibt den Verweis auf ein Objekt zurück, das von einer COM-Komponente bereitgestellt wird.
ComObjMissing() Erstellt ein Objekt für "fehlende Parameter", das an eine COM-Methode übergeben werden kann.
ComObjParameter()
 Erstellt ein Objekt als Eingabewert, der als Parameter oder Rückgabewert übergeben werden soll.
ComObjQuery() Fragt die Schnittstelle oder den Dienst eines COM-Objekts ab.
ComObjType() Ermittelt Informationen über den Typ eines COM-Objekts.
ComObjValue() Ermittelt den Wert oder Pointer, der in einem COM-Wrapper-Objekt enthalten ist.
Continue Überspringt den Rest der aktuellen Schleife und beginnt einen neuen Durchlauf. Nur in einer Schleife gültig.
Control Ermöglicht eine Vielzahl von Änderungen an einem Steuerelement.
ControlClick Sendet eine Maus- oder Mausradaktion an einem Steuerelement.
ControlFocus Setzt den Eingabefokus auf einem vorgegebenen Steuerelement im Fenster.
ControlGet Ruft verschiedene Informationen zu einem Steuerelement ab.
ControlGetFocus Ermittelt, welches Steuerelement im Zielfenster den Eingabefokus hat.
ControlGetPos Ermittelt die Position und Größe eines Steuerelements.
ControlGetText Ermittelt den Text eines Steuerelements.
ControlMove Verschiebt ein Steuerelement oder ändert dessen Größe.
ControlSend / ControlSendRaw Sendet künstliche Tastatureingaben an ein Fenster oder Steuerelement.
ControlSetText Ändert den Text eines Steuerelements.
CoordMode Bestimmt den Koordinaten-Modus für verschiedene Befehle bzgl. relativer oder absoluter Fenster-/Bildschirmposition.
Cos() Gibt den trigonometrischen Kosinus von Zahl zurück.
Critical Verhindert, dass der aktuelle Thread von anderen Threads unterbrochen wird.
DetectHiddenText Legt fest, ob unsichtbare Texte in einem Fenster entdeckt werden können. Das beeinflusst Befehle wie IfWinExist und WinActivate.
DetectHiddenWindows Legt fest, ob unsichtbare Fenster vom Script erkannt werden können.
DllCall() Ruft eine Funktion innerhalb einer DLL-Datei auf, wie z. B. eine API-Funktion von Windows.
Drive Öffnet/Schließt einen CD- oder DVD-Laufwerk oder ändert die Laufwerksbezeichnung.
DriveGet Ermittelt unterschiedliche Informationen über die Laufwerke eines Computers.
DriveSpaceFree Ermittelt den freien Speicher eines Laufwerks in MB.
Edit Öffnet das aktuelle Script mit dem Standard-Editor.
Else Verweist auf Befehle, welche durchzuführen sind, wenn eine If-Bedingung nicht erfüllt wird. Sobald mehrere Befehle vorhanden sind, müssen sie mit einem Block (geschweifte Klammern) umschlossen werden.
EnvAdd Bestimmt eine Variable, die sich selbst mit einem vorgegebenen Wert addiert (kann auch die Zeit in Datum-Zeit-Werte addieren oder subtrahieren). Gleichbedeutend mit: Variable += Wert.
EnvDiv Bestimmt eine Variable, die sich selbst mit einem vorgegebenen Wert dividiert. Gleichbedeutend mit: Variable /= Wert.
EnvGet Ermittelt eine Umgebungsvariable.
EnvMult Bestimmt eine Variable, die sich selbst mit einem vorgegebenen Wert multipliziert. Gleichbedeutend mit: Variable *= Wert.
EnvSet Schreibt einen Wert in eine Umgebungsvariable.
EnvSub Bestimmt eine Variable, die sich selbst mit einem vorgegebenen Wert subtrahiert (kann auch Datum-Zeit-Werte vergleichen). Gleichbedeutend mit: Variable -= Wert.
EnvUpdate Benachrichtigt das Betriebssystem und alle laufenden Anwendungen, dass die Umgebungsvariablen geändert wurden.
Exit Beendet den aktuellen Thread oder (wenn das Script nicht persistent ist und keine Hotkeys enthält) das gesamte Script.
ExitApp Beendet das Script bedingungslos.
Exp() Gibt e zurück (ungefähr 2.71828182845905), potenziert mit N.
FileAppend Schreibt Text an das Ende einer Datei (erstellt die Datei je nach Bedarf).
FileCopy Kopiert eine oder mehrere Dateien.
FileCopyDir Kopiert einen Ordner, einschließlich seine Unterordner und Dateien (ähnlich wie xcopy).
FileCreateDir Erstellt ein(en) Verzeichnis/Ordner.
FileCreateShortcut Erstellt eine Verknüpfung (.lnk).
FileDelete Löscht eine oder mehrere Dateien.
FileEncoding Bestimmt die Standardkodierung für FileRead, FileReadLine, dateilesende Schleife, FileAppend und FileOpen.
Siehe auch: Textkodierungen
FileInstall Fügt die angegebene Datei in die kompilierte Version des Scripts ein.
FileGetAttrib Ermittelt, ob eine Datei zum Beispiel schreibgeschützt oder versteckt ist.
FileGetShortcut Ermittelt Informationen von einer Verknüpfungsdatei (.lnk), sowie von ihrer Zieldatei.
FileGetSize Ermittelt die Größe einer Datei.
FileGetTime Ermittelt den Zeitstempel einer Datei oder eines Ordners.
FileGetVersion Ermittelt die Version einer Datei.
FileMove Verschiebt oder benennt eine oder mehrere Dateien um.
FileMoveDir Verschiebt oder benennt einen Ordner um, einschließlich seine Unterordner und Dateien.
FileOpen Ermöglicht eine objektorientierte Dateibearbeitung.
FileRead Liest den Inhalt einer Datei in eine Variable.
FileReadLine Liest eine bestimme Zeile aus einer Datei und speichert den Text in eine Variable.
FileRecycle Verschiebt eine Datei oder ein Verzeichnis in den Papierkorb, falls möglich.
FileRecycleEmpty Leert den Papierkorb.
FileRemoveDir Löscht einen Ordner.
FileSelectFile Zeigt ein Standard-Dialogfenster an, womit der Benutzer eine oder mehrere Dateien öffnen oder speichern kann.
FileSelectFolder Zeigt ein Standard-Dialogfenster an, womit der Benutzer einen Ordner auswählen kann.
FileSetAttrib Ändert die Attribute von einer oder mehreren Dateien (auch Ordner). Platzhalter werden unterstützt.
FileSetTime Ändert den Zeitstempel von einer oder mehreren Dateien (auch Ordner). Platzhalter werden unterstützt.
Floor() Gibt Zahl zurück, welche auf den nächsten Integer abgerundet ist (ohne .00-Suffix).
For Wiederholt mehrere Befehle für jedes Schlüssel-Wert-Paar in einem Objekt.
FormatTime Verwandelt einen YYYYMMDDHH24MISS-Zeitstempel in das angegebene Datum/Zeit-Format.
Func() Ermittelt einen Verweis auf eine Funktion.
GetKeyName()
GetKeyVK()
GetKeySC()		 Ermittelt den Namen oder Text, virtuellen Tastencode oder Scancode einer Taste.
GetKeyState Überprüft, ob eine Maus-/Joystick- oder Tastaturtaste gehalten wird oder nicht. Ruft auch den Joystickstatus ab.
Gosub Springt zu einem bestimmten Label und setzt die Ausführung bis zu einem Return fort.
Goto Springt zu einem bestimmten Label und setzt die Ausführung fort.
GroupActivate Aktiviert das nächste Fenster in einer Fenstergruppe, die mit GroupAdd definiert worden ist.
GroupAdd Fügt eine Fensterspezifikation zu einer Fenstergruppe hinzu, und erstellt die Gruppe, falls erforderlich.
GroupClose Schließt das aktive Fenster, das gerade mit GroupActivate oder GroupDeactivate aktiviert worden ist, und aktiviert das nächste Fenster in der Gruppe. Dieser Befehl ist auch in der Lage, alle Fenster in einer Gruppe zu schließen.
GroupDeactivate Ähnlich wie GroupActivate, außer dass das nächste Fenster aktiviert wird, welches nicht zur Gruppe gehört.
Gui Erstellt und verwaltet Fenster und deren Steuerelemente. Solche Fenster können zur Dateneingabe oder als individuelle Benutzerschnittstellen verwendet werden.
GuiControl Ermöglicht Änderungen eines Steuerelements in einem GUI-Fenster.
GuiControlGet Ermittelt unterschiedliche Informationen über ein Steuerelement in einem GUI-Fenster.
Hotkey Erstellt, bearbeitet, aktiviert oder deaktiviert ein Hotkey, während das Script läuft.
If Ein oder mehrere Befehle, die ausgeführt werden, wenn der Vergleich einer Variable mit einem Wert WAHR ist. Sobald mehrere Befehle vorhanden sind, müssen sie von einem Block (geschweifte Klammern) umschlossen werden.
If (Ausdruck) Ein oder mehrere Befehle, die ausgeführt werden, wenn ein Ausdruck als WAHR gewertet wird.
If Var [not] between Überprüft, ob der Inhalt einer Variable zwischen zwei Werten liegt (numerisch oder alphabetisch).
If Var [not] in/contains Übereinstimmungsliste Überprüft, ob der Inhalt der Variable mit einem Element aus der Liste übereinstimmt.
If Var is [not] Typ
 Überprüft, ob der Inhalt der Variable zum Beispiel numerisch oder großgeschrieben ist.
IfEqual / IfNotEqual
 Vergleicht eine Variable mit einem Wert. Gleichbedeutend mit: If Var = Wert | If Var <> Wert.
IfExist / FileExist()
Überprüft, ob eine Datei oder ein Ordner vorhanden ist.
IfGreater / IfGreaterOrEqual
 Vergleicht eine Variable mit einem Wert. Gleichbedeutend mit: If Var > Wert | If Var >= Wert.
IfInString / InStr()
Überprüft, ob eine Variable einen bestimmten String enthält.
IfLess / IfLessOrEqual
 Vergleicht eine Variable mit einem Wert. Gleichbedeutend mit: If Var < Wert | If Var <= Wert.
IfMsgBox
 Überprüft, welche Schaltfläche der Benutzer während eines MsgBox-Befehls betätigt hat.
IfWinActive / IfWinNotActive
 Überprüft, ob das angegebene Fenster existiert und zurzeit aktiv ist.
IfWinExist / IfWinNotExist
 Überprüft, ob das angegebene Fenster existiert.
ImageSearch Sucht in einem bestimmten Bildschirmbereich nach einem Bild.
IniDelete Entfernt einen Wert aus einer Standard-INI-Datei.
IniRead Liest einen Wert aus Standard-INI-Datei.
IniWrite Schreibt einen Wert in eine Standard-INI-Datei.
Input Wartet auf die Tastatureingabe des Benutzers (nicht in Windows 9x unterstützt).
InputBox Zeigt ein Fenster zur Eingabe eines Strings an.
InStr() Sucht einen bestimmten String, beginnend von links oder rechts.
IsByRef() Bestimmt, ob ein ByRef-Parameter mit einer Variable übergeben wurde.
IsObject() Erkennt, ob ein Wert ein Objekt ist.
KeyHistory Zeigt Script-Informationen und einen Verlauf der aktuellsten Tastatureingaben und Mausklicks.
KeyWait Wartet, bis eine Tastatur-, Maus- oder Joystick-Taste losgelassen oder gedrückt wird.
ListHotkeys Zeigt an, welche Hotkeys vom aktuellen Script verwendet werden, ob deren Subroutinen zurzeit laufen und ob sie den Tastatur- oder Maus-Hook verwenden.
ListLines Zeigt die aktuell ausgeführten Script-Zeilen an.
ListVars Zeigt die Variablen des Scripts an (jeweils Name und aktueller Inhalt).
Log() Gibt den Logarithmus (Basis 10) von Zahl zurück.
Ln() Ermittelt den Logarithmus (Basis e) von Zahl.
Loop Kontinuierliche Wiederholung von mehreren Befehlen: entweder mittels einer definierten Anzahl an Durchläufen oder bis ein Break-Befehl zum Abbruch auffordert.
Loop (Dateien & Ordner) Ermittelt nacheinander die angegebenen Dateien oder Ordner.
Loop (String parsen) Ermittelt nacheinander Teilstrings (Felder) aus einem String.
Loop (Dateiinhalt) Ermittelt nacheinander die Zeilen einer Textdatei (leistungsfähiger als FileReadLine).
Loop (Registrierung) Ermittelt nacheinander den Inhalt des angegebenen Unterschlüssels.
Menu Erstellt, löscht, ändert und zeigt Menüs und Menüpunkte. Ändert das Tray-Icon und dazugehörige ToolTip. Kontrolliert, ob das Hauptfenster eines kompilierten Scripts geöffnet werden kann.
Mod() Modulo. Gibt den Rest zurück, wenn Dividend durch Divisor geteilt wird.
MouseClick Klickt eine Maustaste, hält diese gedrückt oder dreht das Mausrad. HINWEIS: Der Click-Befehl ist in der Regel flexibler und einfacher zu bedienen.
MouseClickDrag Klickt und hält die angegebene Maustaste gedrückt, bewegt die Maus auf die Zielkoordinaten und lässt die Taste wieder los.
MouseGetPos Ermittelt die aktuelle Position des Mauszeigers und optional auch das darunterliegende Fenster und Steuerelement.
MouseMove Bewegt den Mauszeiger.
MsgBox Zeigt den angegebenen Text in einem kleinen Fenster an, das eine oder mehrere Schaltflächen enthält (z. B. Ja und Nein).
OnExit Bestimmt eine Subroutine, die automatisch ausgeführt wird, sobald das Script beendet wird.
OnMessage() Bestimmt eine Funktion, die automatisch aufgerufen wird, sobald das Script die angegebene Nachricht empfängt.
OutputDebug Sendet einen String zum Debugger (falls vorhanden), um ihn anzeigen zu lassen.
Pause Pausiert den aktuellen Thread vom Script.
PixelGetColor Ermittelt die Farbe des Pixels bei den angegebenen Koordinaten.
PixelSearch Sucht auf einen Bereich des Bildschirms ein Pixel mit der angegebenen Farbe.
PostMessage Platziert eine Nachricht in die Warteschlange eines Fensters oder Steuerelements.
Process Führt verschiedene Operationen an einem Prozess aus: Überprüfung seiner Existenz, Änderung seiner Priorität, Schließen, Warten auf Schließen des Prozesses.
Progress Erstellt oder aktualisiert ein Fenster, das ein Fortschrittsbalken oder Bild enthält.
Random Erzeugt eine Pseudo-Zufallszahl.
RegExMatch() Überprüft, ob ein Muster (regulärer Ausdruck) im String vorkommt.
RegExReplace() Ersetzt Vorkommen eines Musters (regulärer Ausdruck) innerhalb eines Strings.
RegDelete Löscht einen Unterschlüssel oder Wert von der Registrierung.
RegRead Liest einen Wert aus der Registrierung.
RegWrite Schreibt einen Wert in die Registrierung.
RegisterCallback() Erstellt eine Maschinencode-Adresse, die beim Aufrufen den Aufruf zu einer Funktion im Script umleitet.
Reload Ersetzt die aktuell laufende Instanz des Scripts mit einer neuen Instanz.
Return Bewirkt die Rückkehr aus einer Subroutine, deren Ausführung vorher mittels Funktionsaufruf, Gosub, Hotkey-Aktivierung, GroupActivate oder Ähnlichem angesprungen worden ist.
Round() Ist N nicht vorhanden oder eine 0, wird Zahl auf den nächsten Integer gerundet. Ist N eine positive Zahl, wird Zahl auf N Dezimalstellen gerundet. Ist N negativ, wird Zahl auf N Stellen nach links gerundet.
Run Führt ein externes Programm aus.
RunAs Bestimmt mehrere Benutzeranmeldeinformationen für nachfolgende RunWait- und Run-Befehle. Benötigt Windows 2000/XP oder höher.
RunWait Führt ein externes Programm aus und wartet, bis es beendet wird.
Send / SendRaw / SendInput / SendPlay Sendet künstliche Tastatureingaben und Mausklicks an das aktive Fenster.
SendLevel Bestimmt, welche künstliche Tastatur- oder Mausereignisse von Hotkeys und Hotstrings ignoriert werden.
SendMessage Sendet eine Nachricht an ein Fenster oder Steuerelement und wartet zusätzlich auf eine Bestätigung.
SendMode Macht Send gleichbedeutend mit SendInput oder SendPlay, anstatt mit SendEvent (Standard). Bewirkt auch, dass die angegebene Methode von Click und MouseMove/Click/Drag verwendet wird.
SetBatchLines Bestimmt, wie schnell ein Script ausgeführt wird (betrifft die CPU-Auslastung).
SetCapslockState Setzt den Status der Feststelltaste. Kann die Taste auch dazu zwingen, an oder aus zubleiben.
SetControlDelay Bestimmt die Verzögerung, die nach jedem Steuerelement-Befehl auftreten wird.
SetDefaultMouseSpeed Bestimmt die zu verwendende Mausgeschwindigkeit, wenn sie noch nicht in Click und MouseMove/Click/Drag definiert wurde.
SetEnv (Var = Wert) Speichert den angegebenen Wert in eine Variable.
SetFormat Bestimmt das Format von Integer- und und Fließkommazahlen, die durch mathematische Operationen erzeugt wurden.
SetKeyDelay Bestimmt die Verzögerung, die nach jeder Tastatureingabe durch Send und ControlSend auftreten wird.
SetMouseDelay Bestimmt die Verzögerung, die nach jeder Mausbewegung auftreten wird.
SetNumlockState Setzt den Status der Num-Taste. Kann die Taste auch dazu zwingen, an oder aus zubleiben.
SetScrollLockState Setzt den Status der Rollen-Taste. Kann die Taste auch dazu zwingen, an oder aus zubleiben.
SetRegView Bewirkt, dass Registrierungsbefehle in einem 32-Bit-Script auf die 64-Bit-Registrierungsansicht zugreifen können und umgekehrt.
SetStoreCapslockMode
 Bestimmt, ob der Zustand der Feststelltaste nach einem Send-Befehl wiederhergestellt werden soll.
SetTimer Bewirkt, dass eine Subroutine in bestimmten Zeitintervallen automatisch und wiederholend ausgeführt wird.
SetTitleMatchMode Bestimmt das Übereinstimmungsverhalten des Fenstertitel-Parameters in Befehlen wie WinWait.
SetWinDelay Bestimmt die Verzögerung, die nach jedem Fenster-Befehl wie WinActivate auftreten wird.
SetWorkingDir Ändert das Arbeitsverzeichnis des Scripts.
Shutdown Bewirkt, dass das System entweder heruntergefahren, neugestartet oder abgemeldet wird.
Sin() Gibt den trigonometrischen Sinus von Zahl zurück.
Sleep Wartet die angegebene Zeit, bevor die Ausführung fortgesetzt wird.
Sort Ordnet den Inhalt einer Variable in alphabetischer, numerischer oder zufälliger Reihenfolge (entfernt optional Duplikate).
SoundBeep Ermöglicht die Wiedergabe eines Tons vom PC-Lautsprecher.
SoundGet Ermittelt diverse Einstellungen eines Audiogeräts (Stummschalten, Gesamtlautstärke und so weiter).
SoundGetWaveVolume Ermittelt die Wave-Wiedergabelautstärke eines Audio-Geräts.
SoundPlay Abspielen einer Sound-, Video- oder anderen unterstützten Datei.
SoundSet Ändert diverse Einstellungen eines Audiogeräts (Stummschalten, Gesamtlautstärke und so weiter).
SoundSetWaveVolume Ändert die Wave-Wiedergabelautstärke eines Audio-Geräts.
SplashImage Erstellt oder aktualisiert ein Fenster mit einem JPG-, GIF- oder BMP-Bild.
SplashTextOn Erstellt ein anpassbares Popup-Fenster.

StatusBarWait Wartet, bis die Statusleiste eines Fensters den angegebenen String enthält.

StringLeft Ermittelt eine bestimmte Anzahl an Zeichen von der linken Seite eines Strings.

StringSplit Teilt einen String in mehrere Teilstrings auf, basierend auf die angegebenen Trennzeichen.

Tan() Gibt den trigonometrischen Tangens von Zahl zurück.

Transform Führt verschiedene mathematische Funktionen, bitweise Operationen und Aufgaben wie ASCII/Unicode-Konvertierung aus.

Trim Entfernt bestimmte Zeichen vom Anfang und/oder Ende eines Strings.

#CommentFlag Wechselt das Kommentarsymbol des Scripts vom Semikolon zu einigen anderen Strings.
#ErrorStdOut Sendet alle Syntaxfehler, die den Start eines Scripts verhindern, zur Standardausgabe, anstatt ein Dialogfenster anzuzeigen.
#EscapeChar Ändert das Schutzzeichen des Scripts (z. B. Akzent zu Backslash).
#HotkeyInterval Zusammen mit #MaxHotkeysPerInterval kann die Frequenz der Hotkey-Aktivierungen angegeben werden, bei dessen Überschreitung eine Warnmeldung angezeigt wird.
#HotkeyModifierTimeout Beeinflusst das Verhalten von Hotkey-Modifikatoren: CTRL, ALT, WIN und SHIFT.
#Hotstring Ändert Hotstring-Optionen oder Endzeichen.
#If Vergleichbar mit #IfWinActive, aber für beliebige Ausdrücke.
#IfTimeout Bestimmt die Zeit, die maximal für die Auswertung eines einzelnen #If-Ausdrucks aufgebracht werden kann.
#IfWinActive / #IfWinExist Erstellt kontextabhängige Hotkeys und Hotstrings. Solche Hotkeys führen eine andere Aktion aus (oder gar keine), abhängig vom Typ des aktiven oder vorhandenen Fensters.
#Include Bewirkt, dass der Inhalt der angegebenen Datei exakt an dieser Position ausgeführt wird.
#InputLevel Bestimmt, welche künstliche Tastatur- oder Mausereignisse von Hotkeys und Hotstrings ignoriert werden.
#InstallKeybdHook Erzwingt die bedingungslose Installation des Tastatur-Hooks.
#InstallMouseHook Erzwingt die bedingungslose Installation des Maus-Hooks.
#KeyHistory Bestimmt die maximale Anzahl an Tastatur- und Mausereignisse, die im KeyHistory-Fenster angezeigt werden sollen. Mit 0 kann der Tastenverlauf deaktiviert werden.
#MaxHotkeysPerInterval Zusammen mit #HotkeyInterval kann die Frequenz der Hotkey-Aktivierungen angegeben werden, bei dessen Überschreitung eine Warnmeldung angezeigt wird.
#MaxMem Setzt die maximale Kapazität jeder Variable auf die angegebene Anzahl an Megabytes.
#MaxThreads Bestimmt die maximale Anzahl an Threads, die gleichzeitig vorhanden sein können.
#MaxThreadsBuffer Bringt einige oder alle Hotkeys dazu, Tastatureingaben zwischenzuspeichern anstatt zu ignorieren, wenn ihr Grenzwert von #MaxThreadsPerHotkey erreicht wurde.
#MaxThreadsPerHotkey Bestimmt die maximale Anzahl an Threads, die pro Hotkey oder Hotstring gleichzeitig vorhanden sein können.
#MenuMaskKey Ändert die Taste, die dazu verwendet wird, Win- oder Alt-Up-Ereignisse zu verbergen.
#NoEnv Verhindert die Überprüfung leerer Variablen, ob sie Umgebungsvariablen sind (für alle neuen Scripts empfohlen).
#NoTrayIcon Deaktiviert die Darstellung des Tray-Icons.
#Persistent Hält ein Script dauerhaft aufrecht (bis der Benutzer das Script schließt oder ein ExitApp vorkommt).
#SingleInstance Bestimmt, ob ein bereits laufendes Script nochmals ausgeführt werden darf.
#UseHook Erzwingt die Verwendung des Hooks, um alle oder einige Tastatur-Hotkeys zu implementieren.
#Warn Aktiviert oder deaktiviert Warnungen bei bestimmten Zuständen, die auf ein Fehler hinweisen, wie Schreibfehler oder fehlende "globale" Deklarationen.
#WinActivateForce Überspringt die sanfte Aktivierungsmethode eines Fensters und erzwingt die gewaltsame Methode.
