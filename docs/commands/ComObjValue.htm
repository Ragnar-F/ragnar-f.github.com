ComObjValue() [v1.0.91+]

Ermittelt den Wert oder Pointer, der in einem COM-Wrapper-Objekt enthalten ist.

Wert := ComObjValue(ComObject)

Parameter

Wert Ein vorzeichenbehafteter 64-Bit-Integer.
ComObject Ein Wrapper-Objekt, das ein COM-Objekt oder typisierten Wert enthält.

Allgemeine Bemerkungen

Diese Funktion ist nicht für den allgemeinen Gebrauch bestimmt.

Der Aufruf von ComObjValue ist das Äquivalent zu variant.llVal, wobei ComObject als eine VARIANT-Struktur behandelt wird. Jedes Script, das diese Funktion verwendet, muss darauf achten, welchen Wertetyp das Wrapper-Objekt enthält und wie es behandelt werden soll. Wenn zum Beispiel ein Interface-Pointer zurückgegeben wird, sollte Release nicht aufgerufen werden, sondern könnte AddRef benötigen, abhängig davon, was das Script mit dem Pointer macht.

Verwandte Befehle

ComObjType, ComObjCreate, ComObjGet, ComObjActive
