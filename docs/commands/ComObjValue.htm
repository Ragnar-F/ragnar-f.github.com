ComObjValue() [v1.0.91+]

Ermittelt den Wert oder Pointer, der in einem COM-Wrapper-Objekt gespeichert wird.

Wert := ComObjValue(ComObject)

Parameter

Wert Ein 64-Bit-Integer.
ComObject Ein Wrapper-Objekt, das ein COM-Objekt enthält, oder ein typisierter Wert.

Allgemeine Bemerkungen

Diese Funktion ist nicht für den allgemeinen Gebrauch bestimmt.

Das Aufrufen von ComObjValue ist mit variant.llVal vergleichbar, wobei ComObject als eine VARIANT-Struktur behandelt wird. Jedes Script, das diese Funktion verwendet, muss sich bewusst sein, welchen Werttyp das Wrapper-Objekt enthält und wie es behandelt werden soll. Wenn zum Beispiel ein Interface-Pointer zurückgegeben wird, dann sollte Release nicht aufgerufen werden. Möglicherweise ist jedoch AddRef erforderlich, abhängig davon, was das Script mit dem Pointer macht.

Verwandte Befehle

ComObjType, ComObjCreate, ComObjGet, ComObjActive
