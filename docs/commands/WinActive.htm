IfWinActive / IfWinNotActive / WinActive

Überprüft, ob das angegebene Fenster existiert und zurzeit aktiv ist. Ist das der Fall, wird WinActive() dessen Eindeutige ID (HWND) zurückgeben.

IfWinActive [, Fenstertitel, Fenstertext, Titelausnahme, Textausnahme]
IfWinNotActive [, Fenstertitel, Fenstertext, Titelausnahme, Textausnahme]
EindeutigeID := WinActive("Fenstertitel", "Fenstertext", "Titelausnahme", "Textausnahme")

Parameter

Fenstertitel

Ein Fenstertitel oder sonstige Kriterien, die das Zielfenster ermitteln sollen. Siehe Fenstertitel.

Fenstertext

In diesem Parameter kann ein Teilstring angegeben werden, der in einem Textelement des Zielfensters vorkommt (abrufbar mit Windows Spy). Versteckte Textelemente werden auch erkannt, sofern DetectHiddenText aktiv ist.

Titelausnahme

Ignoriert Fenster, deren Titel diesen Wert enthalten. Hinweis: Aufgrund der Abwärtskompatibilität interpretieren IfWinActive und IfWinNotActive diesen Parameter als Befehl, falls er exakt mit dem Namen eines Befehls übereinstimmt. Mit der WinActive()-Funktion dieses Problem umgangen werden.

Textausnahme

Ignoriert Fenster, deren Textelemente diesen Wert enthalten.

Bemerkungen

Sobald alle Parameter weggelassen werden, wird das zuletzt gefundene Fenster verwendet.

Wenn einer der Befehle das aktive Fenster als passende Übereinstimmung ansieht, wird das zuletzt gefundene Fenster auf das aktive Fenster aktualisiert. Das heißt, sobald IfWinActive als "wahr" oder IfWinNotActive als "falsch" gewertet wird, wird das zuletzt gefundene Fenster aktualisiert.

Die Funktion WinActive() gibt die eindeutige ID (HWND) des aktiven Fensters zurück, wenn es die angegebenen Kriterien erfüllt. Ist das nicht der Fall, gibt die Funktion eine 0 zurück. Da alle Zahlen außer 0 als "Wahr" angesehen werden, ist die Anweisung if WinActive("Fenstertitel") immer wahr, sobald der Fenstertitel aktiv ist.

Mit ActiveHwnd := WinExist("A") kann auf einfache Weise die eindeutige ID des aktiven Fensters abgerufen werden.

SetWinDelay hat keinen Einfluss auf IfWinExist/IfWinActive.

Fenstertitel und -texte unterscheiden zwischen Groß- und Kleinschreibung. Solange DetectHiddenWindows deaktiviert ist, werden versteckte Fenster nicht wahrgenommen.

Beispiel

IfWinActive, Unbenannt - Editor
{
    WinMaximize  ; Maximiert das oben gefundene Fenster.
    Send, Dies ist ein Text.{Enter}
    return
}

if WinActive("ahk_class Notepad") or WinActive("ahk_class" . Klassenname)  ; Nach "ahk_class" muss ein Leerzeichen sein.
    WinClose  ; Verwendet das zuletzt gefundene Fenster.
