EnvSet

Schreibt einen Wert in eine Umgebungsvariable.

EnvSet, Umgebungsvariable, Wert

Parameter

Umgebungsvariable

Name der Umgebungsvariable, z. B. "COMSPEC" oder "PATH".

Wert

Ein neuer Wert für die Umgebungsvariable.

ErrorLevel

[v1.1.04+] Dieser Befehl ist in der Lage, bei Misserfolg eine Ausnahme auszulösen. Für mehr Informationen, siehe Laufzeitfehler.

ErrorLevel wird bei Misserfolg auf 1 und bei Erfolg auf 0 gesetzt.

Bemerkungen

Eine Umgebungsvariable ist vom Betriebssystem auf 32 KB begrenzt.

Eine mit diesem Befehl erstellte oder geänderte Umgebungsvariable ist nur für Programme zugänglich, die in einem Script per Run oder RunWait ausgeführt werden. Siehe Umgebungsvariablen für mehr Details.

Dieser Befehl existiert, weil normale Script-Variablen nicht in die Systemumgebung gespeichert werden. Das liegt zum einen daran, dass die Performance darunter leiden würde, und zum anderen, dass Umgebungsvariablen auf 32 KB begrenzt sind.

Verwandte Befehle

EnvGet, Umgebungsvariablen, EnvUpdate, Run, RunWait

Beispiel

EnvSet, AutGUI, Beliebiger Text für die Variable.
