EnvSet

Schreibt einen Wert in eine Umgebungsvariable.

EnvSet, Umgebungsvariable, Wert

Parameter

Umgebungsvariable Name der Umgebungsvariable, z. B. "COMSPEC" oder "PATH".
Wert Ein neuer Wert für die Umgebungsvariable.

ErrorLevel

Bei Problemen wird ErrorLevel auf 1 gesetzt, ansonsten enthält ErrorLevel eine 0.

Bemerkungen

Eine Umgebungsvariable ist vom Betriebssystem auf 32 KB begrenzt.

Eine mit diesem Befehl erstellte oder geänderte Umgebungsvariable ist nur für Programme zugänglich, die in einem Script per Run oder RunWait ausgeführt werden. Siehe Umgebungsvariablen für mehr Details.

Dieser Befehl existiert getrennt von SetEnv, weil normale Scriptvariablen nicht in die Systemumgebung gespeichert werden. Denn die Performance wird schlechter und das Betriebssystem begrenzt die Größe jeder Umgebungsvariable auf 32 KB.

Verwandte Befehle

EnvGet, #NoEnv, Umgebungsvariablen, EnvUpdate, SetEnv, Run, RunWait

Beispiel

EnvSet, AutGUI, Beliebiger Text für die Variable.
