StrReplace

Ersetzt den angegebenen Teilstring mit einem neuen String.

StrReplace, Ausgabevariable, Heuhaufen, Suchtext [, Ersatztext, AusgabeAnzahl, Limit = -1]

Parameter

Ausgabevariable

Der Name der Variable, in der das Ergebnis des Ersatzprozesses gespeichert wird.

Heuhaufen

Der String, dessen Inhalt durchsucht und ersetzt wird.

Suchtext

Der String, der gesucht wird. Die Übereinstimmung ist erst von der Groß- bzw. Kleinschreibung abhängig, wenn StringCaseSense aktiv ist.

Ersatztext

Der Suchtext wird mit diesem Text ersetzt. Sobald dieser Parameter weggelassen wird, wird der Suchtext mit einem leeren String ersetzt. Das heißt, dass der Text in der Ausgabevariable weggelassen wird.

AusgabeAnzahl

Der Name einer Variable ohne Anführungszeichen, in der die Anzahl gespeichert wird, wie oft der Heuhaufen ersetzt wurde (0, wenn niemals).

Limit

Sollte Limit weggelassen werden, wird standardmäßig -1 verwendet, um alle Vorkommen des gefundenen Musters im Heuhaufen zu ersetzen. Ansonsten kann bestimmt werden, wie oft der Heuhaufen ersetzt werden darf. Der Teil des Heuhaufens rechts neben dem zuletzt ersetzten Vorkommen bleibt unverändert.

Bemerkungen

Die internen Variablen A_Space und A_Tab enthalten jeweils ein einzelnes Leerzeichen und ein einzelnes Tabulator-Zeichen. Sie sind nützlich, um Leerzeichen und Tabulatoren einzeln oder am Anfang oder Ende des Suchstrings zu suchen.

Verwandte Befehle

RegExReplace(), StringCaseSense, InStr, SubStr, StrLen, StrLower, StrUpper

Beispiele

; Entfernt alle CR+LF-Paare in der Zwischenablage:
StrReplace, Clipboard, Clipboard, `r`n, , All

; Ersetzt alle Leerzeichen mit Pluszeichen:
StrReplace, NeuerStr, AlterStr, %A_SPACE%, +, All

; Entfernt alle leeren Zeilen im Text einer Variable:
Loop
{
    StrReplace, EigenerStr, EigenerStr, `r`n`r`n, `r`n, UseErrorLevel
    If ErrorLevel = 0  ; Ersetzen nicht mehr notwendig.
        break
}
