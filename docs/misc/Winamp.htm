Winamp automatisieren

In diesem Abschnitt wird gezeigt, wie Winamp mithilfe von Hotkeys gesteuert werden kann, selbst wenn es minimiert oder inaktiv ist. Diese Information wurde mit Winamp 2.78c getestet, sollte aber auch mit allen anderen Versionen funktionieren. Bitte schreibt Änderungen und Verbesserungen ins Forum oder an den Autor.

Sobald im folgenden Beispiel der Hotkey STRG+ALT+P gedrückt wird, wird die Pause/Wiedergabe-Schaltfläche betätigt:

^!p::
IfWinNotExist ahk_class Winamp v1.x
    Return
; Ansonsten bestimmt der obere Befehl das "zuletzt gefundene" Fenster für uns:
ControlSend, ahk_parent, c  ; Pause/Wiedergabe
Return

Hier einige verfügbare Tastenkombinationen in Winamp 2.x (könnte auch in anderen Versionen funktionieren). Das obere Beispiel kann mit den folgenden Tasten ersetzt werden:

Taste zum Senden Auswirkung
c Pause/Wiedergabe
x Abspielen/Wiedergabe neu starten
v Stopp
+v Stopp mit Ausblenden
^v Nach jetzigem Titel stoppen
b Nächster Titel
z Vorheriger Titel
{left} 5 Sekunden zurückspulen
{right} 5 Sekunden vorspulen
{up} Lautstärke erhöhen
{down} Lautstärke verringern

 

; Das nächste Beispiel fragt Winamp, welche Track-Nummer zurzeit aktiv ist:
SendMessage, 1024, 0, 120, ahk_class Winamp v1.x
If ErrorLevel <> FAIL
{
    ErrorLevel += 1  ; Die Zählung von Winamp beginnt bei 0, also um 1 erhöhen.
    MsgBox, Track #%ErrorLevel% ist aktiv oder wird gerade abgespielt.
}
