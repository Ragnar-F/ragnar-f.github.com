Erstellung eines Tastatur- oder Maus-Makros

Ein Makro besteht aus mehreren programmierten Aktionen, die auf Anfrage ausgeführt werden. Makros werden hauptsächlich dazu verwendet, künstliche Tastatureingaben und Mausklicks an beliebig vielen Fenstern zu senden. Solche Fenster reagieren auf Tastatureingaben und Mausklicks, als hätte der Benutzer sie manuell durchgeführt, wodurch wiederholende Aufgaben schnell und zuverlässig automatisiert werden können.

Obwohl Makros manuell geschrieben werden können, ist es wahrscheinlich einfacher, umfassende Makros mithilfe des Makro-Rekorders zu erstellen, wie z. B. mit dem Rekorder-Script von Titan/polyethene. Das Script überwacht die Tastatur- und Mauseingabe des Benutzers sowie die Fenster, die aktiv sind. Es überträgt diese Aktionen in einen funktionierenden Makro, dass später schnell ausgeführt werden kann.

Ein Makro sollte einem Hotkey oder Hotstring zugewiesen werden, um ihn auf einfache Weise auszuführen. Zum Beispiel würde der folgende Hotkey eine leere E-Mail-Nachricht erstellen und sie für einen bestimmten Empfänger vorbereiten, so dass man sie vor dem Senden anpassen kann:

^!s::  ; Hotkey STRG+ALT+S.
IfWinNotExist Posteingang - Microsoft Outlook
    return  ; Das gewünschte Fenster von Outlook ist nicht offen, daher ignorieren.
WinActivate  ; Aktiviert das zuletzt gefundene Fenster.
Send ^n  ; Erstellt eine neue/leere E-Mail mit STRG+N.
WinWaitActive Unbenannt - Nachricht
Send {Tab 2}Produktrückruf für ACME Rocket Skates  ; Bestimmt die Betreffszeile.
Send {Tab}Sehr geehrte Damen und Herren,{Enter 2}Wir haben vor Kurzem einen kleinen Fehler entdeckt ...  ; etc.
return  ; Mit dieser Zeile wird der Hotkey beendet.

Hotkey-Makros sind besonders für Aufgaben nützlich, die mehrmals am Tag durchgeführt werden. Weniger verwendete Makros können hingegen in einzelne Scripts aufbewahrt werden, die mithilfe einer Verknüpfung im Startmenü oder auf dem Desktop zugänglich sind.

Für die Erstellung eigener Makros und Hotkeys, siehe Schnellstart-Tutorial.
