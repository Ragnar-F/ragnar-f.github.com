Sprachsyntax

Wann werden im Zusammenhang mit Befehlen und Parametern Anführungszeichen verwendet?

Anführungszeichen (") haben nur innerhalb von Ausdrücken eine besondere Bedeutung. Ansonsten werden sie wie ganz normale direkte Zeichen behandelt. Wenn ein Script allerdings ein Programm oder Dokument ausführt, dann erwartet das Betriebssystem für gewöhnlich Anführungszeichen bei Befehlszeilenparameter, die Leerzeichen enthalten. Zum Beispiel: Run, Notepad.exe "C:\Eigene Dateien\Adressenliste.txt"

Wann genau werden Variablen mit Prozentzeichen umschlossen (%)?

Variablennamen werden immer mit Prozentzeichen umschlossen, bis auf die unten fett dargestellten Ausnahmen:

  1. Bei Parametern, die Eingabe- oder Ausgabevariablen sind: StringLen, Ausgabevariable, Eingabevariable
  2. Auf der linken Seite einer Zuweisung: Var = 123abc
  3. Auf der linken Seite einer traditionellen If-Anweisung (kein Ausdruck): If Var1 < %Var2%
  4. Überall in Ausdrücken. Zum Beispiel: 
    If (Var1 <> Var2)
    Var1 := Var2 + 100

Wann sollten Prozentzeichen und Kommas geschützt werden?

Direkte Prozentzeichen müssen mit einem Akzent geschützt werden. Zum Beispiel: MsgBox Der aktuelle Prozentwert liegt bei 25`%. Direkte Kommas müssen auch geschützt werden (`,), es sei denn, sie werden in einer MsgBox oder im letzten Parameter eines Befehls verwendet (in diesem Fall ist der Akzent zulässig, aber nicht notwendig).

Wenn Kommas oder Prozentzeichen innerhalb eines Ausdrucks von Anführungszeichen umschlossen werden, dann ist der Akzent zulässig, aber nicht notwendig. Zum Beispiel: Var := "15%"

Allgemeine Aufgaben

Warum werden einige Zeilen in meinem Script nie ausgeführt?

Jede Zeile, die unmittelbar beim Script-Start ausgeführt werden soll, sollte am Anfang des Scripts und vor dem ersten Hotkey, Hotstring oder Return angegeben werden. Siehe automatischer Ausführungsbereich für weitere Details.

Außerdem muss ein Hotkey, der mehr als eine Zeile ausführt, seine erste Zeile unter dem Hotkey auflisten, aber nicht auf derselben Zeile. Zum Beispiel:

#space::  ; Win+Leerzeichen
Run Notepad
WinWaitActive Unbenannt - Editor
WinMaximize
Return

Warum ist der Run-Befehl nicht in der Lage, mein Spiel oder Programm auszuführen?

Einige Programme müssen in ihren eigenen Verzeichnissen gestartet werden (im Zweifelsfall der sicherste Weg). Zum Beispiel:

Run, %ProgramFiles%\Beliebige Anwendung\Anwendung.exe, %ProgramFiles%\Beliebige Anwendung

Wie kann die Ausgabe einer Befehlszeilenoperation abgefragt werden?

Tests zeigen, dass eine temporäre Datei aufgrund von Datei-Caching sehr schnell bei kleinen Ausgaben sein kann. Tatsächlich wird sie gar nicht erst auf das Laufwerk geschrieben, wenn die Datei unmittelbar nach ihrer Verwendung gelöscht wird. Zum Beispiel:

RunWait %comspec% /c dir > C:\Eigene Temp-Datei.txt
FileRead, Ausgabevariable, C:\Eigene Temp-Datei.txt
FileDelete, C:\Eigene Temp-Datei.txt

Zur Vermeidung von temporären Dateien (besonders wenn die Ausgabe groß ist) kann CmdRet oder StdoutToVar verwendet werden.

Wie kann ein Script andere Scripts schließen, pausieren oder sperren?

Zunächst ein Beispiel, das ein anderes Script schließt:

DetectHiddenWindows On  ; Damit kann das versteckte Hauptfenster eines Scripts erkannt werden.
SetTitleMatchMode 2  ; Verhindert, dass unten der vollständige Pfad der Datei angegeben werden muss.
WinClose Script-Dateiname.ahk - AutoHotkey  ; Das hier auf den gewünschten Dateinamen aktualisieren (von der Groß- und Kleinschreibung abhängig).

Um ein anderes Script zu sperren oder zu pausieren, ersetzt die obere letzte Zeile mit einer der folgenden Zeilen:

PostMessage, 0x111, 65305,,, Script-Dateiname.ahk - AutoHotkey  ; Suspend.
PostMessage, 0x111, 65306,,, Script-Dateiname.ahk - AutoHotkey  ; Pause.

Wie kann eine sich wiederholende Aktion gestoppt werden, ohne das Script beenden zu müssen?

Um das ganze Script beim Drücken einer Taste zu pausieren oder fortzusetzen, kann dem Pause-Befehl ein Hotkey wie im folgenden Beispiel zugewiesen werden:

^!p::Pause  ; Drückt Strg+Alt+P, um das Script zu pausieren. Drückt ihn nochmal, um es fortzusetzen.

Um eine Aktion zu stoppen, die sich innerhalb einer Schleife wiederholt, verwendet den folgenden Hotkey, der seine eigene wiederholende Aktion sowohl startet als auch beendet. Anders gesagt: Sobald der Hotkey gedrückt wird, beginnt die Schleife. Beim nochmaligen Drücken des Hotkeys wird die Schleife wieder angehalten.

#MaxThreadsPerHotkey 3
#z::  ; Hotkey: Win+Z (passt diesen Hotkey je nach Bedarf an).
#MaxThreadsPerHotkey 1
If WinZLaufenLassen  ; Bedeutet, dass ein darunterliegender Thread bereits die untere Schleife ausführt.
{
    WinZLaufenLassen := false  ; Weist den Thread darauf hin, die Schleife zu stoppen.
    Return  ; Beendet diesen Thread, damit der darunterliegende Thread fortgesetzt und die Veränderung von der oberen Zeile bemerkt wird.
}
; Ansonsten:
WinZLaufenLassen := true
Loop
{
    ; Die nächsten vier Zeilen sind die gewünschten Aktionen, die wiederholt werden sollen (passt sie je nach Bedarf an):
    ToolTip, Drückt nochmals Win+Z, um das hier zu stoppen.
    Sleep 1000
    ToolTip
    Sleep 1000
    ; Lasst den Rest aber unverändert.
    If not WinZLaufenLassen ; Der Benutzer hat die Schleife durch erneutes Drücken von Win+Z gestoppt.
        break  ; Schleife unterbrechen.
}
WinZLaufenLassen := false  ; Zurücksetzen, damit der Hotkey wieder verwendet werden kann.
Return

Wie kann die Performance für Spiele oder zu einem anderen Zeitpunkt verbessert werden, wenn die CPU stark belastet wird?

Wenn Hotkeys, Clicks oder Sends im Script bei starker CPU-Belastung spürbar langsamer als normal sind, dann ist es vielleicht hilfreich, die Priorität des Script-Prozesses zu erhöhen. Fügt dafür die folgende Zeile am Anfang des Scripts hinzu:

Process, Priority, , High

Wie kann man feststellen, dass eine Webseite fertig geladen ist?

Für den Internet Explorer ist das Script-Beispiel von www.autohotkey.com/forum/topic19256.html (in Verbindung mit DllCall und COM) wahrscheinlich die zuverlässigste Methode. Außerdem ist es auch möglich, den Inhalt der Adress- und Statusleiste abzurufen, siehe dazu www.autohotkey.com/forum/topic19255.html.

Ältere, weniger zuverlässige Methode: Das folgende Beispiel funktioniert fast mit allen Webseiten vom MS Internet Explorer. Diese Methode funktioniert möglicherweise auch mit anderen Browsern, wenn sie entsprechend angepasst wird:

Run, www.yahoo.com
MouseMove, 0, 0  ; Hindert die Statusleiste daran, den Link unter dem Mauszeiger anstelle von "Fertig" anzuzeigen.
WinWait, Yahoo! -
WinActivate
StatusBarWait, Done, 30
If ErrorLevel
   MsgBox Die Wartezeit ist abgelaufen oder das Fenster wurde geschlossen.
Else
   MsgBox Die Webseite wurde geladen.

Wie können Datums- oder Uhrzeitwerte verglichen oder geändert werden?

Der EnvAdd-Befehl kann eine bestimmte Menge an Tagen, Stunden, Minuten oder Sekunden im YYYYMMDDHH24MISS-Format addieren oder subtrahieren. Im folgenden Beispiel werden 7 Tage von der angegebenen Zeit abgezogen:

EnvAdd, VariableMitZeitstempel, -7, days

Ein Beispiel zur Ermittlung des Zeitraums zwischen zwei Daten oder Zeiten findet man im Beispielbereich von EnvSub. Zudem enthält die interne Variable A_Now die aktuelle Ortszeit. Es gibt noch mehr interne Datum/Zeit-Variablen, sowie auch der FormatTime-Befehl, der einen benutzerdefinierten Datum/Zeit-String erstellen kann.

Warum funktionieren Hotstrings, Sends und Clicks in einigen Spielen nicht?

Einige Spiele verwenden ausschließlich DirectInput. Das führt möglicherweise dazu, dass alle künstlichen Tastatureingaben und Mausklicks ignoriert werden. Probiert eine der folgenden Möglichkeiten (oder in Kombination), um dieses Problem zu umgehen:

  • Verwendet SendPlay mithilfe des SendPlay-Befehls, SendMode Play und/oder der Hotstring-Option SP.
  • Erhöht SetKeyDelay. Zum Beispiel:
    SetKeyDelay, 0, 50
    SetKeyDelay, 0, 50, Play
  • Probiert es mit ControlSend.

Hotkeys, Hotstrings und Neubelegung

Wie können meine Hotkeys und Hotstrings automatisch bei jedem PC-Start ausgeführt werden?

Mit dem Ordner "Autostart" im Startmenü. Fügt in diesem Ordner eine Verknüpfung des Scripts ein, damit das Script jedesmal automatisch ausgeführt wird, sobald Windows hochfährt. Eine Verknüpfung kann beispielsweise wie folgt erstellt werden:

  1. Findet die Script-Datei, wählt sie aus und drückt Strg+C.
  2. Klickt mit der rechten Maustaste auf die Start-Schaltfläche bzw. auf "Alle Programme" (normalerweise in der linken unteren Ecke des Bildschirms) und wählt "Öffnen - Alle Benutzer" aus.
  3. Navigiert zum Autostart-Ordner innerhalb von "Programme".
  4. Wählt in der Menüleiste Bearbeiten -> Verknüpfung einfügen aus. Die Verknüpfung des Scripts sollte sich nun im Autostart-Ordner befinden.

Ich habe Schwierigkeiten, dass meine Maustasten wie Hotkeys funktionieren. Irgendwelche Hinweise?

Beachtet, dass Maus-Hotkeys zurzeit nicht in Windows 95/98/Me unterstützt werden. Bei anderen Betriebssystemen sollte die linke und rechte Maustaste ganz normal zuweisbar sein (zum Beispiel ist #LButton:: der Hotkey Win+Linksklick). Das gilt auch für die mittlere Maustaste und das Mausrad, außer bei Mäusen, deren Tasten direkt vom Treiber gesteuert werden.

Die vierte (XButton1) und fünfte Maustaste (XButton2) sind möglicherweise auch zuweisbar, wenn der Maustreiber die Klicks für das System erkennbar macht. Falls sie nicht erkennbar sind (oder die Maus mehr als 5 Tasten hat, die verwendet werden sollen), kann die dazugehörige Software der Maus notfalls auch so konfiguriert werden (meistens über der Systemsteuerung oder im Startmenü), dass eine Tastatureingabe gesendet wird, sobald eine dieser Tasten gedrückt wird. Dadurch ist es möglich, so eine Tastatureingabe als Hotkey zu definieren. Wenn die vierte Maustaste zum Beispiel so eingestellt wird, dass sie Strg+F1 sendet, dann ist es mithilfe von ^F1:: im Script möglich, diese Taste indirekt als Hotkey zu verwenden.

Falls die vierte und fünfte Taste bei einer Maus mit 5 Tasten nicht erkannt werden können, benutzt den Standardtreiber des Betriebssystems anstelle des aktuellen Maustreibers. Das setzt voraus, dass so ein Treiber für die entsprechende Maus vorhanden ist und dass der Benutzer ohne den mitgelieferten Funktionen der Software leben kann.

Wie können Tabulatoren und Leerzeichen als Hotkeys definiert werden?

Verwendet die Namen der Tasten (Tab und Space), anstatt deren Zeichen. Zum Beispiel ist #Space die Tastenkombination Win+Leertaste und ^!Tab die Tastenkombination Strg+Alt+Tab.

Wie kann ein Hotkey oder Hotstring exklusiv für ein bestimmtes Programm bereitgestellt werden? Mit anderen Worten möchte ich eine bestimmte Taste dazu bringen, dass sie sich anders als normal verhält, falls ein bestimmtes Fenster aktiv ist.

Die bevorzugte Methode ist #IfWinActive. Zum Beispiel:

#IfWinActive, ahk_class Notepad
^a::MsgBox Steuerung+A wurde gedrückt, während der Editor aktiv ist.

Windows 95/98/Me: Die oben genannte Methode funktioniert nur im Editor. Bei anderen Fenstern hat Strg+A keine Wirkung (selbst die ursprüngliche Funktion nicht). Das Problem kann wie folgt umgangen werden:

$^a::Send ^a  ; Dieser Hotkey muss zuerst in Windows 9x aufgelistet werden. Das $-Präfix erlaubt, dass der Hotkey "sich selbst senden" kann.
#IfWinActive, ahk_class Notepad
^a::MsgBox Steuerung+A wurde gedrückt, während der Editor aktiv ist.

Wie kann bei einer Präfix-Taste erreicht werden, dass sie ihre ursprüngliche Funktion durchführt, anstatt nichts zu tun?

Das folgende Beispiel wandelt Numpad0 in eine Präfix-Taste:

Numpad0 & Numpad1::MsgBox, Numpad1 wurde gedrückt, während Numpad0 gedrückt gehalten wird.

Damit Numpad0 nun einen echte Numpad0-Tastatureingabe sendet, falls sie nicht für ein Hotkey wie oben gebraucht wird, fügt den folgenden Hotkey hinzu:

 $Numpad0::Send, {Numpad0}

Der $-Präfix ist notwendig, um eine Warnmeldung im Falle einer Endlosschleife zu unterdrücken (da der Hotkey "sich selbst sendet"). Darüber hinaus wird die obere Aktion erst ausgelöst, wenn die Taste losgelassen wird.

Meine Tastatur hat eine besondere 000-Taste. Kann ich damit einen Hotkey erzeugen?

Es ist möglich, aber nur, wenn das Betriebssystem Windows NT, 2000, XP oder höher ist. Dieses Beispiel-Script wandelt die "000"-Taste in eine Gleichheitstaste um. Die Aktion kann je nach Bedarf geändert werden, indem die Zeile "Send, =" mit einer anderen Zeile ersetzt wird.
