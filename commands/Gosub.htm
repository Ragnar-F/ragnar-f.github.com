Gosub

Springt zu einem bestimmten Label und setzt die Ausführung bis zu einem Return fort.

Gosub, Label

Parameter

Label Name des Labels (auch Hotkey- oder Hotstring-Label), das ausgeführt werden soll, bis ein Return oder Exit erreicht wird. Mit Return springt das Script zum ersten Befehl nach dem Gosub, um die Ausführung fortzusetzen. Mit Exit wird das aktuelle Script beendet.

Bemerkungen

Wie bei den meisten Parametern von Befehlen kann Label eine Variablenreferenz enthalten (z. B. %EigenesLabel%). In diesem Fall wird der Name in der Variable als Ziel benutzt. Allerdings wird dadurch die Performance ein wenig beeinträchtigt, weil das Ziel-Label jedes Mal neu eingelesen werden muss, anstatt nur einmal, wenn das Script geladen wird.

Beim Verwenden eines dynamischen Labels wie %EigenesLabel% erscheint eine Fehlermeldung, wenn das Label nicht existiert. Zur Vermeidung dieses Problems kann IsLabel() verwendet werden. Zum Beispiel:

If IsLabel(VarMitLabelname)
    Gosub %VarMitLabelname%

Obwohl Gosub für einfache Subroutinen nützlich ist, sollten Funktionen für komplexere Anliegen in Betracht gezogen werden.

Verwandte Befehle

Return, Funktionen, IsLabel(), Block, Loop, Goto

Beispiel

Gosub, Label1
MsgBox, Die Label1-Subroutine hat den Return-Befehl erreicht (und wurde somit ausgeführt).
Return

Label1:
MsgBox, Die Label1-Subroutine läuft gerade.
Return
