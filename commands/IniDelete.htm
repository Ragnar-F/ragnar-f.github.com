IniDelete

Entfernt einen Wert von einer INI-Datei im Standardformat. 

IniDelete, Dateiname, Sektion [, Schlüssel]

Parameter

Dateiname Der Name der Datei, die sich im A_WorkingDir-Verzeichnis befindet, solange kein absoluter Pfad angegeben ist.
Sektion Name der Sektion in der INI-Datei. Solche Sektionen werden mit eckigen Klammern dargestellt (die Klammern dürfen nicht in diesem Parameter stehen).
Schlüssel Name des Schlüssels in der INI-Datei. Wenn weggelassen, dann wird die ganze Sektion gelöscht.

ErrorLevel

Bei Problemen wird ErrorLevel auf 1 gesetzt, ansonsten enthält ErrorLevel eine 0. Wenn das Script allerdings vom Typ .aut (AutoIt v2) ist, dann wird ErrorLevel nicht geändert (aus Gründen der Kompatibilität).

Bemerkungen

Eine typische INI-Datei sieht wie folgt aus:

[Sektionsname]
Schlüssel=Wert

Verwandte Befehle

IniRead, IniWrite, RegDelete

Beispiel

IniDelete, C:\Temp\EigeneDatei.ini, Sektion2, Schlüssel
