#Persistent

Hält ein Script dauerhaft aufrecht (bis der Benutzer das Script schließt oder ein ExitApp vorkommt).

#Persistent

Wenn diese Anweisung irgendwo im Script vorkommt, dann bleibt das Script aufrecht, nachdem der automatische Ausführungsbereich (oberster Bereich des Scripts) durchgeführt wurde. Besonders in Fällen nützlich, wo ein Script Timer und/oder benutzerdefinierte Menüpunkte enthält, aber nicht Hotkeys, Hotstrings, OnMessage() oder Gui.

Wenn diese Anweisung zu einem vorhandenen Script hinzugefügt wird, dann sollten einige oder alle Vorkommen von Exit mit ExitApp ersetzt werden. Denn Exit wird ein persistentes Script nicht beenden, nur den aktuellen Thread.

Ab v1.0.16 macht diese Anweisung ein Script zur Einzelinstanz. Um dieses Verhalten zu überschreiben, verwendet #SingleInstance.

Verwandte Befehle

#SingleInstance, SetTimer, Menu, Exit, ExitApp
