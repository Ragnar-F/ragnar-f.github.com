If Variable [not] in Wert1,Wert2,...
If Variable [not] contains Wert1,Wert2,...

Überprüft, ob der Inhalt der Variable mit einem Element aus der Liste übereinstimmt.

If Variable in Übereinstimmungsliste
If Variable not in Übereinstimmungsliste

If Variable contains Übereinstimmungsliste
If Variable not contains Übereinstimmungsliste

Parameter

Variable Name der Variable, dessen Inhalt überprüft wird. Bei dem Operator "in" ist eine exakte Übereinstimmung mit einem Element aus der Liste notwendig. Bei dem Operator "contains" verhält sich die Übereinstimmung toleranter: sobald die Variable ein Element aus der Liste als Teilstring enthält.
Übereinstimmungsliste

Eine kommagetrennte Liste aus Strings, die mit dem Inhalt der Variable verglichen werden. Alle Leer- und Tabulatorzeichen um Trennkommas sind bedeutend, d. h. sie sind ein Teil des übereinstimmenden Strings. Zum Beispiel: Wenn die Übereinstimmungsliste ABC , XYZ enthält, dann muss die Variable für eine Übereinstimmung entweder ABC mit nachfolgendem Leerzeichen oder XYZ mit führendem Leerzeichen enthalten.

Zwei aufeinanderfolgende Kommas bilden ein direkt vorhandenes Komma. Das folgende Beispiel erstellt ein direkt vorhandenes Komma nach dem Wort string1: If Variable In string1,,,string2. Ebenso enthält die folgende Liste nur ein einzelnes Element mit einem direkt vorhandenen Komma dazwischen: If Variable In single,,item. Für das Hinzufügen eines leeren Elements muss das erste Zeichen ein Komma sein: If Variable In ,string1,string2 (wenn der Operator "contains" verwendet wird, dann ist die Übereinstimmung mit einem leeren Element immer erfolgreich, da ein leerer String immer in einem String gefunden wird).

Da die Elemente in der Übereinstimmungsliste nicht als einzelne Parameter behandelt werden, kann sich die komplette Liste in einer Variable befinden. In der Tat muss sich alles oder ein Teil davon in der Variable befinden, wenn eine Länge von 16383 überschritten wird (die maximale Länge einer Script-Zeile). Zum Beispiel könnte die Übereinstimmungsliste aus %List1%,%List2%,%List3% bestehen, jede Liste enthält eine lange Liste mit Elementen.

Jedes einzelne Element in einer Liste, das länger als 16384 Zeichen ist, behandelt die zusätzlichen Zeichen als ein neues Element. Daher ist es das Beste, solche Elemente zu vermeiden.

Bemerkungen

Der Vergleich erfolgt immer alphabetisch, nicht numerisch. Zum Beispiel stimmt der String "11" nicht mit dem Listenelement "11.0" übereinstimmen.

Der Operator "contains" ist das gleiche wie IfInString/IfNotInString, aber mit der Ausnahme, dass die Mehrfachsuche von Strings unterstützt wird (einer stimmt überein).

In diesem Fall kann StringCaseSense On verwendet werden, um den Vergleich abhängig von der Groß- bzw. Kleinschreibung zu machen.

Wenn Übereinstimmungsliste lang ist, dann kann die Liste in mehreren kurzen Zeilen aufgeteilt werden (Fortsetzungsbereich), die die Lesbarkeit und Wartbarkeit verbessern könnten.

Die Operatoren "between", "is", "in" und "contains" werden in Ausdrücken nicht unterstützt.

Verwandte Befehle

If Variable between, IfEqual/Greater/Less, IfInString, StringCaseSense, Blocks, Else

Beispiele

If Variable in exe,bat,com
    MsgBox Die Datei ist ausführbar.

If Variable in 1,2,3,5,7,11 ; Leerzeichen vermeiden.
    MsgBox %Variable% ist eine kleine Primzahl.

If Variable contains 1,3  ; Die Werte werden als Strings verglichen, nicht als Zahlen.
    MsgBox Variable enthält die Zahl 1 oder 3 (Variable kann eine 1, 3, 10, 21, 23, etc. sein)

If Variable in %Übereinstimmungsliste%
    MsgBox %Variable% ist in der Liste.
