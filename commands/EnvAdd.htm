EnvAdd

Legt eine Variable an, die sich selbst mit einem vorgegebenen Wert addiert (kann auch die Uhrzeit in Datum-Zeit-Werte addieren oder subtrahieren). Gleichbedeutend mit: Variable += Wert

EnvAdd, Variable, Wert [, Zeiteinheit]
Variable += Wert [, Zeiteinheit]
Variable++

Parameter

Variable Der Name der Variable, die bearbeitet wird.
Wert Beliebiger Integer-Wert, Fließkommazahl oder Ausdruck.
Zeiteinheit

Der Wert wird mit der Variable addiert, wobei die Variable als ein Datum-Zeit-Stempel im YYYYMMDDHH24MISS-Format und der Wert als Integer oder Fließkommazahl behandelt wird (mit einer negativen Zahl wird eine Subtraktion erreicht). Die Zeiteinheit enthält entweder Seconds, Minutes, Hours oder Days (oder jeweils nur den ersten Buchstaben).

Wenn Variable leer ist, dann wird die aktuelle Uhrzeit verwendet. Wenn die Variable einen ungültigen Zeitstempel oder ein Jahr vor 1601 enthält, oder der Wert keine gültige Zahl ist, dann wird die Variable leer gemacht werden, um auf ein Problem hinzuweisen.

Die interne Variable A_Now enthält die aktuelle Ortszeit im YYYYMMDDHH24MISS-Format.

Zur Berechnung des Abstands zwischen zwei Zeitstempeln kann EnvSub verwendet werden.

Bemerkungen

Dieser Befehl ist gleichbedeutend mit der Kurzfassung: Variable += Wert

Der Wert einer Variable kann um 1 erhöht oder verringert werden, wenn Variable++, Variable--, ++Variable oder --Variable verwendet wird.

Wenn die Variable oder der Wert leer ist oder nicht mit einer Zahl beginnt, dann wird eine 0 als Berechnungsgrundlage verwendet (außer bei interner Verwendung in Ausdrücken oder bei Verwendung des Zeiteinheit-Parameters).

Wenn die Variable oder der Wert einen Dezimalpunkt enthält, dann ist das Endergebnis eine Fließkommazahl, abhängig vom eingestellten Format (SetFormat).

Verwandte Befehle

EnvSub, EnvMult, EnvDiv, SetFormat, Ausdrücke, If Variable is [not] Typ, SetEnv, FileGetTime

Beispiel

EnvAdd, EigeneZählung, 2
EigeneZählung += 2  ; das Gleiche wie oben

Variable1 =  ; Variable leer machen, damit die aktuelle Uhrzeit verwendet wird.
Variable1 += 31, days
MsgBox, %Variable1%  ; Das Ergebnis ist das Datum nach genau 31 Tagen.
