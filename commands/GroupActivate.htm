GroupActivate

Aktiviert das nächste Fenster in einer Fenstergruppe, die mit GroupAdd erstellt wurde. 

GroupActivate, Gruppenname [, R]

Parameter

Gruppenname Der Name der zu aktivierenden Gruppe, ursprünglich von GroupAdd definiert.
R Damit wird festgelegt, ob das älteste oder das neueste Fenster aktiviert wird, sobald kein Fenster der Gruppe aktiv ist. Wenn nicht vorhanden, dann wird immer das älteste Fenster aktiviert. Mit dem Buchstaben R wird das neueste (zuletzt aktive) Fenster aktiviert, aber mit der Bedingung, dass kein Fenster der Gruppe aktiv ist, wenn der Befehl ausgeführt wird. "R" ist beispielsweise nützlich, falls der Benutzer sich vorübergehend mit einer anderen Aufgabe beschäftigt. Kehrt er dann mit GroupActivate, GroupDeactivate oder GroupClose zur Gruppe zurück, wird das am aktuellsten genutzte Fenster aktiviert.

Bemerkungen

Dieser Befehl aktiviert das erste Fenster, das mit den Spezifikationen der Fenstergruppe übereinstimmt. Beim nochmaligen Verwenden wird das nächste Fenster in der Reihe aktiviert und so weiter. Normalerweise wird dieser Befehl einem Hotkey zugewiesen, damit dieses Verhalten beim Drücken einer Taste automatisiert wird.

Wenn ein Fenster aktiviert wird, nachdem ein anderes Fenster aktiviert wurde, dann fangen die Schaltflächen der Taskleiste an zu blinken (abhängig vom Betriebssystem und von den Einstellungen). Mit #WinActivateForce kann das verhindert werden.

Siehe GroupAdd für mehr Details über Fenstergruppen.

ErrorLevel [AHK_L 54+]

ErrorLevel wird auf 1 gesetzt, wenn kein Fenster gefunden wurde, das aktiviert werden kann, ansonsten enthält ErrorLevel immer eine 0.

Verwandte Befehle

GroupAdd, GroupDeactivate, GroupClose, #WinActivateForce

Beispiel

GroupActivate, EigeneGruppe, R
