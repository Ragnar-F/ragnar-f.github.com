Control

Ermöglicht eine Vielzahl von Änderungen an einem Steuerelement.

Control, Befehl [, Wert, Steuerelement, Fenstertitel, Fenstertext, Titelausnahme, Textausnahme]

Parameter

Befehl, Wert Siehe untere Liste.
Steuerelement

Möglich ist entweder ClassNN (Klassenname und Instanznummer des Steuerelements) oder Name/Text des Steuerelements, die beide mit Windows Spy ermittelt werden können. Wenn Name/Text verwendet wird, wird das Übereinstimmungsverhalten mit SetTitleMatchMode bestimmt. Wenn dieser Parameter leer ist, wird das oberste Steuerelement des Fensters benutzt.

Für das HWND eines Steuerelements (Windows Handle) wird der Steuerelement-Parameter leer gelassen und ahk_id %SteuerelementHwnd% im Fenstertitel-Parameter angegeben (funktioniert auch bei versteckten Steuerelementen, selbst wenn DetectHiddenWindows deaktiviert ist). Das HWND eines Steuerelements wird üblicherweise mittels ControlGet Hwnd, MouseGetPos oder DllCall abgerufen.
Fenstertitel Der Titel oder Teiltitel des Fensters (das Übereinstimmungsverhalten wird mit SetTitleMatchMode bestimmt). Wenn dieser und die nächsten 3 Parameter nicht vorhanden sind, dann wird das zuletzt gefundene Fenster verwendet. Wenn dieser den Buchstaben A enthält und die nächsten 3 Parameter nicht vorhanden sind, dann wird das aktive Fenster verwendet. Bei Verwendung einer Fensterklasse muss ahk_class ExakterKlassenname angegeben werden (wird von Windows Spy angezeigt). Bei Verwendung einer Prozess-Kennung (PID) muss ahk_pid %VariableMitPID% angegeben werden. Bei Verwendung einer Fenstergruppe muss ahk_group Gruppenname angegeben werden. Bei Verwendung einer eindeutigen ID-Nummer eines Fensters muss ahk_id %VariableMitID% angegeben werden. Die Suche kann durch Angabe mehrerer Kriterien eingeschränkt werden. Zum Beispiel: Eigene Datei.txt ahk_class Notepad
Fenstertext Wenn vorhanden, dann muss dieser Parameter einen Teil der einzelnen Textelemente des Fensters enthalten (wird von Windows Spy angezeigt). Versteckte Textelemente werden erfasst, wenn DetectHiddenText aktiv ist.
Titelausnahme Fenstertitel mit diesem Wert werden nicht berücksichtigt.
Textausnahme Fenstertexte mit diesem Wert werden nicht berücksichtigt.

Befehl, Wert

Die Parameter Befehl und Wert sind voneinander abhängig und können wie folgt verwendet werden.

Check: Markiert ein Optionsfeld oder Kontrollkästchen.

Uncheck: Entfernt die Markierung von einem Optionsfeld oder Kontrollkästchen.

Enable: Aktiviert ein Steuerelement, wenn es zuvor deaktiviert war.

Disable: Deaktiviert oder "ergraut" ein Steuerelement.

Show: Steuerelement einblenden, wenn es zuvor ausgeblendet wurde.

Hide: Steuerelement ausblenden. Außerdem ist es möglich, die Tastenkombination des Steuerelements (unterstrichener Buchstabe) mit "Control Disable" zu unterbinden.

Style, N oder ExStyle, N: Ändert jeweils den Style oder Extended Style eines Steuerelements. Wenn das erste Zeichen in N ein Plus- oder Minuszeichen ist, dann wird der Style in N jeweils hinzugefügt bzw. entfernt. Wenn das erste Zeichen in N ein Zirkumflex (^) ist, dann wird der Style jeweils auf den entgegengesetzten Zustand gesetzt. Wenn das erste Zeichen in N eine Ziffer ist, dann wird der Style des Steuerelements komplett überschrieben. ErrorLevel wird auf 1 gesetzt, wenn das Fenster/Steuerelement nicht gefunden wird oder wenn der Style nicht benutzt werden darf (was häufig in Windows 9x auftritt).

Bestimmte Style-Änderungen benötigen eine grafische Aktualisierung des Fensters mit Hilfe von WinSet Redraw. Außerdem bietet die Style-Tabelle noch weitere Style-Nummern. Zum Beispiel:

Control, Style, ^0x800000, Edit1, Fenstertitel  ; Setzt den WS_BORDER-Style auf den entgegengesetzten Zustand.

ShowDropDown: Öffnet die ComboBox, damit die Auswahlmöglichkeiten sichtbar werden.

HideDropDown: Kehrt das Obengenannte um.

TabLeft [, Count] und TabRight [, Count]: Verschiebt den Tab in einem SysTabControl32 beliebig oft nach links oder rechts. Count ist automatisch 1, wenn diese Variable nicht vorhanden oder leer ist. Um stattdessen einen Tab direkt mit einer Nummer auszuwählen, ersetzt die untere Nummer 5 mit der Positionsnummer des ausgewählten Tabs. Beachtet, dass der erste Tab mit der Positionsnummer 0 beginnt.

SendMessage, 0x1330, 5,, SysTabControl321, Fenstertitel  ; 0x1330 ist TCM_SETCURFOCUS.
Sleep 0  ; Diese und die nächste Zeile sind nur für einige Tab-Steuerelemente notwendig.
SendMessage, 0x130C, 5,, SysTabControl321, Fenstertitel  ; 0x130C ist TCM_SETCURSEL.

Add, String: String wird als neuer Eintrag am Ende einer ListBox oder ComboBox hinzugefügt (funktioniert vielleicht auch bei anderen Steuerelementen).

Delete, N: Entfernt den N-ten Eintrag in einer ListBox oder ComboBox. N ist für den ersten Eintrag die Nummer 1, für den zweiten Eintrag die Nummer 2, usw.

Choose, N: Setzt die Auswahl auf den N-ten Eintrag in einer ListBox oder ComboBox. N Für den ersten Eintrag die Nummer 1, für den zweiten Eintrag die Nummer 2, usw. Um die Auswahl in einer ListBox mit Mehrfachauswahl auf mehrere Einträge gleichzeitig zu setzen oder zu entfernen, siehe Beispiel:

PostMessage, 0x185, 1, -1, ListBox1, Fenstertitel  ; Alle ListBox-Einträge auswählen. 0x185 ist LB_SETSEL.

ChooseString, String: Setzt die Auswahl auf den ersten Eintrag in einer ListBox oder ComboBox, dessen führender Teil mit String übereinstimmt. Die Suche ist nicht von der Groß-/Kleinschreibung abhängig. Wenn eine ListBox/ComboBox beispielsweise den Eintrag "UNIX-Text" enthält, dann genügt das Wort unix (kleingeschrieben), um es auszuwählen.

EditPaste, String: String wird in einem Edit-Steuerelement nach dem Text-Cursor eingefügt (Der Inhalt der Zwischenablage ist davon nicht betroffen).

ErrorLevel

Bei Problemen wird ErrorLevel auf 1 gesetzt, ansonsten enthält ErrorLevel eine 0.

Bemerkungen

Um die Zuverlässigkeit des Befehls zu verbessern, erfolgt automatisch eine Verzögerung nach jeder Verwendung (außer bei Style und ExStyle). Diese Verzögerung kann mit SetControlDelay geändert werden.

Zur Erfassung des Steuerelementnamens unter dem Mauszeiger wird MouseGetPos verwendet.

Fenstertitel und -texte sind von der Groß- und Kleinschreibung abhängig. Solange DetectHiddenWindows deaktiviert ist, werden versteckte Fenster nicht wahrgenommen.

Verwandte Befehle

SetControlDelay, ControlGet, GuiControl, ControlGetText, ControlSetText, ControlMove, ControlGetPos, ControlClick, ControlFocus, ControlSend, WinSet
