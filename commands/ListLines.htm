ListLines

Zeigt die aktuell ausgeführten Script-Zeilen an.

ListLines [, On|Off]

Parameter

On|Off

Wenn leer oder nicht vorhanden, dann wird der Verlauf der aktuell ausgeführten Zeilen angezeigt. Seit v1.0.48.01 ist es möglich, wahlweise On oder Off im ersten Parameter anzugeben. Dies wirkt sich wie folgt nur auf das Verhalten des aktuellen Threads aus:

On: Fügt danach ausgeführte Zeilen im Verlauf ein. Das ist der Standardwert für alle Scripts.

Off: Lässt danach ausgeführte Zeilen vom Verlauf weg.

Bemerkungen

ListLines (ohne Parameter) entspricht der Auswahl des Menüelements "View->Lines most recently executed" im Hauptfenster. Es kann helfen, ein Script zu debuggen.

"ListLines Off/On" kann verwendet werden, um bestimmte Zeilen vom Verlauf wegzulassen, damit der Verlauf sich nicht zu schnell füllt (z. B. bei einer Schleife mit schnellen Iterationen). "ListLines, Off" kann außerdem die Performance um einige Prozente verbessern.

Jeder neu gestartete Thread (z. B. ein Hotkey, ein benutzerdefinierter Menüpunkt oder eine zeitlich festgelegte Subroutine) beginnt mit der Voreinstellung für diesen Befehl. Die Voreinstellung lässt sich ändern, wenn dieser Befehl im automatischen Ausführungsbereich (am Anfang des Scripts) ausgeführt wird.

Obwohl keine interne Variable "A_ListLines" vorhanden ist, kann eine ähnliche Funktionalität erreicht werden, indem Folgendes im Script eingefügt wird:

ListLines(PassTrueToTurnOnOrFalseToTurnOff)  ; Gibt die vorherige Einstellung von ListLines zurück (vor diesem Aufruf).
{
    static sListLines := true  ; Die Standardeinstellung für alle Scripts ist "ListLines On".
    ListLines % PassTrueToTurnOnOrFalseToTurnOff ? "On" : "Off"  ; Führt ListLines bedingungslos aus, um nachfolgend ausgeführte Zeilen im log wegzulassen.
    ListLines_prev := sListLines
    sListLines := PassTrueToTurnOnOrFalseToTurnOff
    Return ListLines_prev
}

; Um die obere Funktion anzuwenden:
prev_ListLines := ListLines(false)  ; ListLines wird vorübergehend deaktiviert.
; ...
ListLines(prev_ListLines)  ; Die vorherige Einstellung von ListLines wird wiederhergestellt.

Zusätzlich beinhalten die internen Variablen A_LineNumber und A_LineFile die aktuell ausgeführte Zeilennummer und den dazugehörigen Dateinamen.

Verwandte Befehle

KeyHistory, ListHotkeys, ListVars

Beispiel

ListLines
ListLines Off
